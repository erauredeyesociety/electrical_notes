      Butterworth Filter Design - Comprehensive Worked Examples

      This document contains detailed worked examples covering all major problem types for Butterworth filter design, based on typical homework and exam problems.

      EXAMPLE 1: Basic Low-Pass Filter Design (Given All Specs)

      Problem Statement

      Design a low-pass Butterworth filter with the following specifications:

      • Passband: DC to 2 kHz with maximum gain HMAX=0H_{MAX} = 0HMAX​=0 dB (gain = 1)
      • Stopband: Above 8 kHz with minimum attenuation HMIN=−40H_{MIN} = -40HMIN​=−40 dB (gain = 0.01)
      • Implement using Sallen-Key topology

      Solution

      Step 1: Convert specifications to rad/s

      fC=2 kHz→ωC=2π×2000=12566 rad/sf_C = 2 \text{ kHz} \rightarrow \omega_C = 2\pi \times 2000 = 12566 \text{ rad/s}fC​=2 kHz→ωC​=2π×2000=12566 rad/s

      fMIN=8 kHz→ωMIN=2π×8000=50265 rad/sf_{MIN} = 8 \text{ kHz} \rightarrow \omega_{MIN} = 2\pi \times 8000 = 50265 \text{ rad/s}fMIN​=8 kHz→ωMIN​=2π×8000=50265 rad/s

      Step 2: Calculate minimum order

      n≥ln⁡[(1/0.01)2−1]2ln⁡(50265/12566)=ln⁡(9999)2ln⁡(4)=9.2102.773=3.32n \geq \frac{\ln[(1/0.01)^2-1]}{2\ln(50265/12566)} = \frac{\ln(9999)}{2\ln(4)} = \frac{9.210}{2.773} = 3.32n≥2ln(50265/12566)ln[(1/0.01)2−1]​=2ln(4)ln(9999)​=2.7739.210​=3.32

      Round up: n = 4 (4th order filter)

      Step 3: Determine section breakdown

      • n = 4 (even) → 2 second-order sections
      • No first-order sections needed

      Step 4: Calculate pole angles for n = 4

      θk=π2+(2k−1)π8\theta_k = \frac{\pi}{2} + \frac{(2k-1)\pi}{8}θk​=2π​+8(2k−1)π​

      For k = 1, 2, 3, 4:

      • θ1=π2+π8=5π8=112.5°\theta_1 = \frac{\pi}{2} + \frac{\pi}{8} = \frac{5\pi}{8} = 112.5°θ1​=2π​+8π​=85π​=112.5°
      • θ2=π2+3π8=7π8=157.5°\theta_2 = \frac{\pi}{2} + \frac{3\pi}{8} = \frac{7\pi}{8} = 157.5°θ2​=2π​+83π​=87π​=157.5°
      • θ3=π2+5π8=9π8=202.5°\theta_3 = \frac{\pi}{2} + \frac{5\pi}{8} = \frac{9\pi}{8} = 202.5°θ3​=2π​+85π​=89π​=202.5°
      • θ4=π2+7π8=11π8=247.5°\theta_4 = \frac{\pi}{2} + \frac{7\pi}{8} = \frac{11\pi}{8} = 247.5°θ4​=2π​+87π​=811π​=247.5°

      Step 5: Calculate pole locations

      • p1=cos⁡(112.5°)+jsin⁡(112.5°)=−0.383+j0.924p_1 = \cos(112.5°) + j\sin(112.5°) = -0.383 + j0.924p1​=cos(112.5°)+jsin(112.5°)=−0.383+j0.924
      • p2=cos⁡(157.5°)+jsin⁡(157.5°)=−0.924+j0.383p_2 = \cos(157.5°) + j\sin(157.5°) = -0.924 + j0.383p2​=cos(157.5°)+jsin(157.5°)=−0.924+j0.383
      • p3=−0.924−j0.383p_3 = -0.924 - j0.383p3​=−0.924−j0.383 (conjugate of p2p_2p2​)
      • p4=−0.383−j0.924p_4 = -0.383 - j0.924p4​=−0.383−j0.924 (conjugate of p1p_1p1​)

      All in left half-plane ✓

      Step 6: Form second-order sections

      Section 1 (poles p1,p4p_1, p_4p1​,p4​):

      (s−p1)(s−p4)=s2−2(−0.383)s+[(−0.383)2+(0.924)2](s - p_1)(s - p_4) = s^2 - 2(-0.383)s + [(-0.383)^2 + (0.924)^2](s−p1​)(s−p4​)=s2−2(−0.383)s+[(−0.383)2+(0.924)2]

      =s2+0.765s+1= s^2 + 0.765s + 1=s2+0.765s+1

      Compare to standard form s2+2ζω0s+ω02s^2 + 2\zeta\omega_0 s + \omega_0^2s2+2ζω0​s+ω02​:

      • ω0=1\omega_0 = 1ω0​=1 (normalized)
      • 2ζ=0.7652\zeta = 0.7652ζ=0.765
      • ζ=0.383\zeta = 0.383ζ=0.383
      • Q=12(0.383)=1.307Q = \frac{1}{2(0.383)} = 1.307Q=2(0.383)1​=1.307

      Section 2 (poles p2,p3p_2, p_3p2​,p3​):

      s2+1.848s+1s^2 + 1.848s + 1s2+1.848s+1
      • ζ=0.924\zeta = 0.924ζ=0.924
      • Q=0.541Q = 0.541Q=0.541

      Step 7: Design Method Selection

      Both sections have 0<ζ<10 < \zeta < 10<ζ<1, so we can use either method. Choose Unity Gain Method for both sections (always works).

      Step 8: Design Section 1 (ζ=0.383\zeta = 0.383ζ=0.383, ω0=12566\omega_0 = 12566ω0​=12566 rad/s)

      Choose C1=0.1C_1 = 0.1C1​=0.1 μ\mu μF

      C2=4(0.383)2(0.1 μF)=0.0586 μF≈0.056 μF (standard)C_2 = 4(0.383)^2(0.1\text{ $\mu $F}) = 0.0586 \text{ $\mu $F} \approx 0.056 \text{ $\mu $F (standard)}C2​=4(0.383)2(0.1 μF)=0.0586 μF≈0.056 μF (standard)R=10.383×12566×0.1×10−6=2078 ohms≈2.2 k ohms (standard)R = \frac{1}{0.383 \times 12566 \times 0.1 \times 10^{-6}} = 2078 \text{ ohms} \approx 2.2 \text{ k ohms (standard)}R=0.383×12566×0.1×10−61​=2078 ohms≈2.2 k ohms (standard)

      Step 9: Design Section 2 (ζ=0.924\zeta = 0.924ζ=0.924, ω0=12566\omega_0 = 12566ω0​=12566 rad/s)

      Choose C1=0.1C_1 = 0.1C1​=0.1 μ\mu μF

      C2=4(0.924)2(0.1 μF)=0.341 μF≈0.33 μF (standard)C_2 = 4(0.924)^2(0.1\text{ $\mu $F}) = 0.341 \text{ $\mu $F} \approx 0.33 \text{ $\mu $F (standard)}C2​=4(0.924)2(0.1 μF)=0.341 μF≈0.33 μF (standard)R=10.924×12566×0.1×10−6=861 ohms≈820 ohms (standard)R = \frac{1}{0.924 \times 12566 \times 0.1 \times 10^{-6}} = 861 \text{ ohms} \approx 820 \text{ ohms (standard)}R=0.924×12566×0.1×10−61​=861 ohms≈820 ohms (standard)

      Step 10: Cascade Order

      Section 2 (Q = 0.541, higher ζ\zetaζ) → Section 1 (Q = 1.307, lower ζ\zetaζ)

      Final Design:

      • Stage 1: R1=R2=820R_1 = R_2 = 820R1​=R2​=820 ohms, C1=0.1C_1 = 0.1C1​=0.1 μ\mu μF, C2=0.33C_2 = 0.33C2​=0.33 μ\mu μF, μ=1\mu = 1μ=1
      • Stage 2: R1=R2=2.2R_1 = R_2 = 2.2R1​=R2​=2.2 k ohms, C1=0.1C_1 = 0.1C1​=0.1 μ\mu μF, C2=0.056C_2 = 0.056C2​=0.056 μ\mu μF, μ=1\mu = 1μ=1

      EXAMPLE 2: High-Pass Filter with Gain Requirement

      Problem Statement

      Design a 3rd-order high-pass Butterworth filter with:

      • Cutoff frequency: fC=500f_C = 500fC​=500 Hz
      • Stopband frequency: fMIN=100f_{MIN} = 100fMIN​=100 Hz
      • Stopband attenuation: HMIN=−50H_{MIN} = -50HMIN​=−50 dB
      • Overall passband gain: K = 10

      Solution

      Step 1: Verify order requirement

      Convert to linear: HMIN=10−50/20=0.00316H_{MIN} = 10^{-50/20} = 0.00316HMIN​=10−50/20=0.00316

      n≥ln⁡[(1/0.00316)2−1]2ln⁡(3142/628)=ln⁡(99999)2ln⁡(5)=11.513.22=3.58n \geq \frac{\ln[(1/0.00316)^2-1]}{2\ln(3142/628)} = \frac{\ln(99999)}{2\ln(5)} = \frac{11.51}{3.22} = 3.58n≥2ln(3142/628)ln[(1/0.00316)2−1]​=2ln(5)ln(99999)​=3.2211.51​=3.58

      Given n = 3, check if sufficient: 3<3.583 < 3.583<3.58 → Need n = 4 minimum!

      However, if problem states “design 3rd order,” proceed with n = 3 (may not meet all specs).

      Step 2: Convert to rad/s

      ωC=2π×500=3142 rad/s\omega_C = 2\pi \times 500 = 3142 \text{ rad/s}ωC​=2π×500=3142 rad/s

      Step 3: Pole angles for n = 3

      θk=π2+(2k−1)π6\theta_k = \frac{\pi}{2} + \frac{(2k-1)\pi}{6}θk​=2π​+6(2k−1)π​
      • θ1=2π3=120°\theta_1 = \frac{2\pi}{3} = 120°θ1​=32π​=120° → p1=−0.5+j0.866p_1 = -0.5 + j0.866p1​=−0.5+j0.866
      • θ2=π=180°\theta_2 = \pi = 180°θ2​=π=180° → p2=−1p_2 = -1p2​=−1 (real pole)
      • θ3=4π3=240°\theta_3 = \frac{4\pi}{3} = 240°θ3​=34π​=240° → p3=−0.5−j0.866p_3 = -0.5 - j0.866p3​=−0.5−j0.866

      Step 4: Form sections

      Second-order section (p1,p3p_1, p_3p1​,p3​):

      s2+s+1s^2 + s + 1s2+s+1
      • ζ=0.5\zeta = 0.5ζ=0.5
      • Q=1.0Q = 1.0Q=1.0

      First-order section (p2p_2p2​):

      s+1s + 1s+1

      Step 5: Design high-pass sections

      For high-pass Sallen-Key, we swap R and C from low-pass design.

      Second-order HP (Unity Gain, ζ=0.5\zeta = 0.5ζ=0.5, ω0=3142\omega_0 = 3142ω0​=3142 rad/s):

      Choose R2=10R_2 = 10R2​=10 k ohms

      R1=4(0.5)2(10 k ohms)=10 k ohmsR_1 = 4(0.5)^2(10\text{ k ohms}) = 10 \text{ k ohms}R1​=4(0.5)2(10 k ohms)=10 k ohmsC=10.5×3142×10000=63.7 nF≈68 nFC = \frac{1}{0.5 \times 3142 \times 10000} = 63.7 \text{ nF} \approx 68 \text{ nF}C=0.5×3142×100001​=63.7 nF≈68 nF

      So: R1=R2=10R_1 = R_2 = 10R1​=R2​=10 k ohms, C1=C2=68C_1 = C_2 = 68C1​=C2​=68 nF

      First-order HP (ωC=3142\omega_C = 3142ωC​=3142 rad/s):

      H(s)=sRC1+sRCH(s) = \frac{sRC}{1 + sRC}H(s)=1+sRCsRC​

      Choose C=68C = 68C=68 nF:

      R=13142×68×10−9=4.68 k ohms≈4.7 k ohmsR = \frac{1}{3142 \times 68 \times 10^{-9}} = 4.68 \text{ k ohms} \approx 4.7 \text{ k ohms}R=3142×68×10−91​=4.68 k ohms≈4.7 k ohms

      Step 6: Add gain of K = 10

      Distribute gain equally: K1=K2=10≈3.16K_1 = K_2 = \sqrt{10} \approx 3.16K1​=K2​=10​≈3.16 per section

      For 2nd-order section, use Equal Elements Method:

      • Need μ=3.16\mu = 3.16μ=3.16
      • But Equal Elements requires μ=3−2(0.5)=2\mu = 3 - 2(0.5) = 2μ=3−2(0.5)=2 ✗

      Alternative: Add gain stage at output:

      μfinal=1+RfRg=10\mu_{final} = 1 + \frac{R_f}{R_g} = 10μfinal​=1+Rg​Rf​​=10

      Choose Rg=1R_g = 1Rg​=1 k ohms → Rf=9R_f = 9Rf​=9 k ohms

      Final Cascade: 1st-order HP → 2nd-order HP → Gain stage (non-inverting amp with gain 10)

      EXAMPLE 3: Design from Transfer Function H(s)

      Problem Statement

      Given the transfer function:

      H(s)=10000(s+50)(s2+100s+10000)H(s) = \frac{10000}{(s+50)(s^2 + 100s + 10000)}H(s)=(s+50)(s2+100s+10000)10000​

      Implement this filter using Sallen-Key topology.

      Solution

      Step 1: Identify structure

      • Numerator: constant → Low-pass filter
      • Denominator: one real pole + one complex conjugate pair
      • Order: n = 3

      Step 2: Extract section parameters

      First-order section: s+50s + 50s+50

      • Pole at s=−50s = -50s=−50
      • ωC=50\omega_C = 50ωC​=50 rad/s

      Second-order section: s2+100s+10000s^2 + 100s + 10000s2+100s+10000

      • Compare to s2+2ζω0s+ω02s^2 + 2\zeta\omega_0 s + \omega_0^2s2+2ζω0​s+ω02​
      • ω02=10000\omega_0^2 = 10000ω02​=10000 → ω0=100\omega_0 = 100ω0​=100 rad/s
      • 2ζω0=1002\zeta\omega_0 = 1002ζω0​=100 → 2ζ(100)=1002\zeta(100) = 1002ζ(100)=100 → ζ=0.5\zeta = 0.5ζ=0.5
      • Q=1.0Q = 1.0Q=1.0

      Step 3: Design first-order section

      ωC=50 rad/s\omega_C = 50 \text{ rad/s}ωC​=50 rad/s

      Choose C=1C = 1C=1 μ\mu μF:

      R=150×1×10−6=20 k ohmsR = \frac{1}{50 \times 1 \times 10^{-6}} = 20 \text{ k ohms}R=50×1×10−61​=20 k ohms

      Step 4: Design second-order section (Unity Gain)

      ζ=0.5\zeta = 0.5ζ=0.5, ω0=100\omega_0 = 100ω0​=100 rad/s

      Choose C1=1C_1 = 1C1​=1 μ\mu μF:

      C2=4(0.5)2(1 μF)=1 μFC_2 = 4(0.5)^2(1\text{ $\mu $F}) = 1 \text{ $\mu $F}C2​=4(0.5)2(1 μF)=1 μF

      R=10.5×100×1×10−6=20 k ohmsR = \frac{1}{0.5 \times 100 \times 1 \times 10^{-6}} = 20 \text{ k ohms}R=0.5×100×1×10−61​=20 k ohms

      Step 5: Verify DC gain

      Given H(0)=1000050×10000=0.02H(0) = \frac{10000}{50 \times 10000} = 0.02H(0)=50×1000010000​=0.02

      Designed sections both have unity gain → Need gain stage:

      K=0.02 (or can be distributed)K = 0.02 \text{ (or can be distributed)}K=0.02 (or can be distributed)

      Final Implementation:

      • 1st-order: R=20R = 20R=20 k ohms, C=1C = 1C=1 μ\mu μF
      • 2nd-order: R1=R2=20R_1 = R_2 = 20R1​=R2​=20 k ohms, C1=C2=1C_1 = C_2 = 1C1​=C2​=1 μ\mu μF, μ=1\mu = 1μ=1
      • Attenuator at input or output to achieve DC gain of 0.02

      EXAMPLE 4: Given Bode Plot, Find H(s)

      Problem Statement

      A Bode magnitude plot shows:

      • DC gain: 20 dB
      • -3 dB frequency: 1 kHz
      • Asymptotic slope: -40 dB/decade starting at 1 kHz
      • Phase at 1 kHz: -90°

      Determine H(s) and identify the filter type.

      Solution

      Step 1: Extract information

      • DC gain = 20 dB → Linear: K=1020/20=10K = 10^{20/20} = 10K=1020/20=10
      • ωC=2π×1000=6283\omega_C = 2\pi \times 1000 = 6283ωC​=2π×1000=6283 rad/s
      • Slope = -40 dB/dec → 2nd order
      • Phase = -90° at ωC\omega_CωC​ → Butterworth 2nd order

      Step 2: Identify filter type

      • Constant at DC, rolls off at high frequency → Low-pass
      • 2nd order Butterworth → ζ=0.707\zeta = 0.707ζ=0.707

      Step 3: Write H(s)

      Standard 2nd-order Butterworth LP:

      H(s)=KωC2s2+2ωCs+ωC2H(s) = \frac{K\omega_C^2}{s^2 + \sqrt{2}\omega_C s + \omega_C^2}H(s)=s2+2​ωC​s+ωC2​KωC2​​H(s)=10×(6283)2s2+2(6283)s+(6283)2H(s) = \frac{10 \times (6283)^2}{s^2 + \sqrt{2}(6283)s + (6283)^2}H(s)=s2+2​(6283)s+(6283)210×(6283)2​H(s)=3.95×108s2+8886s+3.95×107H(s) = \frac{3.95 \times 10^8}{s^2 + 8886s + 3.95 \times 10^7}H(s)=s2+8886s+3.95×1073.95×108​

      Verification:

      • At DC (s=0s = 0s=0): H(0)=3.95×1083.95×107=10H(0) = \frac{3.95 \times 10^8}{3.95 \times 10^7} = 10H(0)=3.95×1073.95×108​=10 ✓
      • At ωC\omega_CωC​ (s=j6283s = j6283s=j6283): ∣H(j6283)∣=102=7.07|H(j6283)| = \frac{10}{\sqrt{2}} = 7.07∣H(j6283)∣=2​10​=7.07 → 20log(7.07) = 17 dB (20 - 3 = 17) ✓

      EXAMPLE 5: Frequency and Impedance Scaling

      Problem Statement

      A normalized 2nd-order low-pass Butterworth filter has:

      • R=1R = 1R=1 ohms, C=1C = 1C=1 F, ωC=1\omega_C = 1ωC​=1 rad/s

      Scale this design to:

      • New cutoff: fC=10f_C = 10fC​=10 kHz
      • Practical component values: RRR around 10 k ohms

      Solution

      Step 1: Determine scaling factors

      Frequency scaling:

      kf=ωnewωold=2π×100001=62832k_f = \frac{\omega_{new}}{\omega_{old}} = \frac{2\pi \times 10000}{1} = 62832kf​=ωold​ωnew​​=12π×10000​=62832

      Impedance scaling:

      kz=RnewRold=100001=10000k_z = \frac{R_{new}}{R_{old}} = \frac{10000}{1} = 10000kz​=Rold​Rnew​​=110000​=10000

      Step 2: Apply scaling

      Resistors:

      Rnew=kz×Rold=10000×1=10 k ohmsR_{new} = k_z \times R_{old} = 10000 \times 1 = 10 \text{ k ohms}Rnew​=kz​×Rold​=10000×1=10 k ohms

      Capacitors:

      Cnew=Coldkf×kz=162832×10000=1.59×10−9 F=1.59 nFC_{new} = \frac{C_{old}}{k_f \times k_z} = \frac{1}{62832 \times 10000} = 1.59 \times 10^{-9} \text{ F} = 1.59 \text{ nF}Cnew​=kf​×kz​Cold​​=62832×100001​=1.59×10−9 F=1.59 nF

      Use standard value: C=1.5C = 1.5C=1.5 nF

      Step 3: Verify new cutoff

      ωC,new=1RnewCnew=110000×1.5×10−9=66667 rad/s\omega_{C,new} = \frac{1}{R_{new}C_{new}} = \frac{1}{10000 \times 1.5 \times 10^{-9}} = 66667 \text{ rad/s}ωC,new​=Rnew​Cnew​1​=10000×1.5×10−91​=66667 rad/sfC,new=666672π=10.6 kHzf_{C,new} = \frac{66667}{2\pi} = 10.6 \text{ kHz}fC,new​=2π66667​=10.6 kHz

      (close to 10 kHz target)

      EXAMPLE 6: Equal Elements Method Design

      Problem Statement

      Design a 2nd-order low-pass Butterworth filter using the Equal Elements Method:

      • Cutoff frequency: fC=5f_C = 5fC​=5 kHz

      EXAMPLE 10: Troubleshooting - Why Unity Gain Required

      Problem Statement

      Attempt to design a 2nd-order section with ζ=1.2\zeta = 1.2ζ=1.2 using Equal Elements Method. What goes wrong?

      Solution

      Step 1: Check Equal Elements validity

      μ=3−2ζ=3−2(1.2)=−0.4\mu = 3 - 2\zeta = 3 - 2(1.2) = -0.4μ=3−2ζ=3−2(1.2)=−0.4

      Problem: μ=−0.4<1\mu = -0.4 < 1μ=−0.4<1

      This is impossible! A non-inverting amplifier must have μ≥1\mu \geq 1μ≥1.

      Step 2: Why this happens

      Equal Elements Method constraint:

      μ=3−2ζ≥1\mu = 3 - 2\zeta \geq 1μ=3−2ζ≥1

      3−1≥2ζ3 - 1 \geq 2\zeta3−1≥2ζ

      ζ≤1\zeta \leq 1ζ≤1

      For ζ=1.2>1\zeta = 1.2 > 1ζ=1.2>1, the method cannot work.

      Step 3: Correct approach - Unity Gain Method

      With ζ=1.2\zeta = 1.2ζ=1.2, use Unity Gain:

      Choose C1=0.1C_1 = 0.1C1​=0.1 μ\mu μF:

      C2=4(1.2)2(0.1)=0.576 μFC_2 = 4(1.2)^2(0.1) = 0.576 \text{ $\mu $F}C2​=4(1.2)2(0.1)=0.576 μF

      R=11.2×ω0×0.1×10−6R = \frac{1}{1.2 \times \omega_0 \times 0.1 \times 10^{-6}}R=1.2×ω0​×0.1×10−61​

      Lesson: Always check ζ<1\zeta < 1ζ<1 before using Equal Elements. For overdamped systems (ζ≥1\zeta \geq 1ζ≥1), must use Unity Gain Method.

      SUMMARY OF KEY TAKEAWAYS

      1. Always round order UP - Never round down or you won’t meet specs
      2. Check ζ<1\zeta < 1ζ<1 before using Equal Elements Method
      3. Unity Gain works for everything - It’s the safe choice
      4. Order sections by Q - Low Q (high ζ\zetaζ) first
      5. Use standard component values - Nearest E12/E24 series
      6. Verify DC gain - Make sure overall gain matches requirements
      7. Frequency scaling - Affects only C (not R)
      8. Impedance scaling - Affects both R and C oppositely
      9. Bandpass = Cascade - Multiply transfer functions
      10. Bandstop = Parallel Sum - Add transfer functions
