      SECTION 1: TRANSFER FUNCTIONS (NETWORK FUNCTIONS)

      Definition

      • ZEROS: Roots of numerator N(s)=0N(s) = 0N(s)=0 - Frequencies that are blocked

      Finding H(s) from Circuit - Quick Steps

      1. Transform all elements to s-domain
      2. ZERO all initial conditions
      3. Apply circuit analysis (voltage divider, nodal, mesh)
      4. Form H(s)=OutputInputH(s) = \frac{\text{Output}}{\text{Input}}H(s)=InputOutput​
      5. Simplify to rational function

      SECTION 3: OP-AMP FUNDAMENTALS

      Ideal Op-Amp Rules

      1. No current flows into either input: i+=i−=0i_+ = i_- = 0i+​=i−​=0
      2. Voltage difference is zero (negative feedback): V+=V−V_+ = V_-V+​=V−​

      Common Configurations

      Inverting Amplifier:

      H(s)=−Zf(s)Zin(s)H(s) = -\frac{Z_f(s)}{Z_{in}(s)}H(s)=−Zin​(s)Zf​(s)​

      Non-Inverting Amplifier:

      H(s)=1+Zf(s)Zg(s)=μH(s) = 1 + \frac{Z_f(s)}{Z_g(s)} = \muH(s)=1+Zg​(s)Zf​(s)​=μ

      μ=1+RfRg\mu = 1 + \frac{R_f}{R_g}μ=1+Rg​Rf​​

      Voltage Follower (Buffer):

      H(s)=1(μ=1)H(s) = 1 \quad (\mu = 1)H(s)=1(μ=1)
      • High input impedance, low output impedance
      • Prevents loading between stages

      Summing Amplifier:

      Vout=−Rf(V1R1+V2R2)V_{out} = -R_f\left(\frac{V_1}{R_1} + \frac{V_2}{R_2}\right)Vout​=−Rf​(R1​V1​​+R2​V2​​)

      Why Op-Amps in Filter Design?

      • Enable cascading without loading: Htotal(s)=H1(s)×H2(s)×⋯H_{total}(s) = H_1(s) \times H_2(s) \times \cdotsHtotal​(s)=H1​(s)×H2​(s)×⋯
      • Provide gain (μ>1\mu > 1μ>1)
      • Buffer stages (high input Z, low output Z)

      PHASE 1: Calculate Required Filter Order

      Given Specifications

      • HMAXH_{MAX}HMAX​: Maximum passband gain (linear or dB)
      • HMINH_{MIN}HMIN​: Minimum stopband gain (linear or dB)
      • ωC\omega_CωC​ or fCf_CfC​: Cutoff frequency
      • ωMIN\omega_{MIN}ωMIN​ or fMINf_{MIN}fMIN​: Stopband frequency

      Convert dB to Linear

      Hlinear=10HdB/20H_{\text{linear}} = 10^{H_{\text{dB}}/20}Hlinear​=10HdB​/20

      Butterworth Order Formulas

      Low-Pass:

      n≥12⋅ln⁡[(HMAX/HMIN)2−1]ln⁡(ωMIN/ωC)n \geq \frac{1}{2} \cdot \frac{\ln[(H_{MAX}/H_{MIN})^2 - 1]}{\ln(\omega_{MIN}/\omega_C)}n≥21​⋅ln(ωMIN​/ωC​)ln[(HMAX​/HMIN​)2−1]​

      High-Pass:

      n≥12⋅ln⁡[(HMAX/HMIN)2−1]ln⁡(ωC/ωMIN)n \geq \frac{1}{2} \cdot \frac{\ln[(H_{MAX}/H_{MIN})^2 - 1]}{\ln(\omega_C/\omega_{MIN})}n≥21​⋅ln(ωC​/ωMIN​)ln[(HMAX​/HMIN​)2−1]​

      ALWAYS ROUND UP to nearest integer

      Section Count

      • Odd n: (n−1)/2(n-1)/2(n−1)/2 second-order + 1 first-order
      • Even n: n/2n/2n/2 second-order sections

      PHASE 4: UNITY GAIN METHOD (RECOMMENDED)

      Why Unity Gain?

      • Works for ALL damping ratios (any positive value)
      • Most stable configuration
      • Op-amp as simple buffer (μ=1\mu = 1μ=1)

      Design Equations - Low-Pass

      Option A: Choose C1C_1C1​ (typical: 0.01 to 1 microF)

      C2=4ζ2C1C_2 = 4\zeta^2 C_1C2​=4ζ2C1​

      R1=R2=R=1ζω0C1R_1 = R_2 = R = \frac{1}{\zeta\omega_0 C_1}R1​=R2​=R=ζω0​C1​1​

      Option B: Choose R2R_2R2​ (typical: 1k to 100k ohms)

      R1=4ζ2R2R_1 = 4\zeta^2 R_2R1​=4ζ2R2​

      C1=C2=C=1ζω0R2C_1 = C_2 = C = \frac{1}{\zeta\omega_0 R_2}C1​=C2​=C=ζω0​R2​1​

      Design Equations - High-Pass

      Swap R and C positions from low-pass design.

      With R2R_2R2​ chosen:

      R1=4ζ2R2R_1 = 4\zeta^2 R_2R1​=4ζ2R2​

      C1=C2=C=1ζω0R2C_1 = C_2 = C = \frac{1}{\zeta\omega_0 R_2}C1​=C2​=C=ζω0​R2​1​

      PHASE 5: EQUAL ELEMENTS METHOD (BACKUP)

      When to Use

      • Only if damping ratio is between 0 and 1 (underdamped, Q > 0.5)
      • Requires op-amp gain greater than 1

      Design Equations - Low-Pass

      Set R1=R2=RR_1 = R_2 = RR1​=R2​=R and C1=C2=CC_1 = C_2 = CC1​=C2​=C

      μ=3−2ζ\mu = 3 - 2\zetaμ=3−2ζ

      Choose C, then:

      R=1ω0CR = \frac{1}{\omega_0 C}R=ω0​C1​

      Op-amp gain circuit:

      μ=1+RfRg\mu = 1 + \frac{R_f}{R_g}μ=1+Rg​Rf​​

      Rf=(μ−1)⋅RgR_f = (\mu - 1) \cdot R_gRf​=(μ−1)⋅Rg​

      Validity check: 1≤μ≤31 \leq \mu \leq 31≤μ≤3 if and only if damping ratio is between 0 and 1

      Design Equations - High-Pass

      Same as low-pass: R1=R2=RR_1 = R_2 = RR1​=R2​=R, C1=C2=CC_1 = C_2 = CC1​=C2​=C, μ=3−2ζ\mu = 3 - 2\zetaμ=3−2ζ

      PHASE 6: First-Order Sections

      For Real Pole at s=−ωCs = -\omega_Cs=−ωC​

      Passive RC Low-Pass:

      H(s)=11+sRC,ωC=1RCH(s) = \frac{1}{1 + sRC}, \quad \omega_C = \frac{1}{RC}H(s)=1+sRC1​,ωC​=RC1​

      Choose C, then: R=1ωCCR = \frac{1}{\omega_C C}R=ωC​C1​

      Active Low-Pass (with gain K):

      H(s)=K1+sRCH(s) = \frac{K}{1 + sRC}H(s)=1+sRCK​

      Use non-inverting op-amp: K=1+RfRgK = 1 + \frac{R_f}{R_g}K=1+Rg​Rf​​

      High-Pass (1st order):

      H(s)=sRC1+sRCH(s) = \frac{sRC}{1 + sRC}H(s)=1+sRCsRC​

      Swap R and C positions from low-pass.

      PHASE 7: Cascade Assembly

      Section Ordering (CRITICAL)

      Order by Q: LOW to HIGH

      Place highest damping ratio (lowest Q) first, lowest damping ratio (highest Q) last.

      Why? Prevents early clipping from resonant peaks.

      Overall Transfer Function

      Htotal(s)=H1(s)×H2(s)×⋯×Hn(s)H_{\text{total}}(s) = H_1(s) \times H_2(s) \times \cdots \times H_n(s)Htotal​(s)=H1​(s)×H2​(s)×⋯×Hn​(s)

      Effect: All frequencies multiplied by kfk_fkf​

      Impedance Scaling (multiply by k_z)

      Rnew=kz×Rold,Cnew=ColdkzR_{\text{new}} = k_z \times R_{\text{old}}, \quad C_{\text{new}} = \frac{C_{\text{old}}}{k_z}Rnew​=kz​×Rold​,Cnew​=kz​Cold​​

      Effect: H(s)H(s)H(s) and frequencies unchanged

      SECTION 5: BODE PLOTS

      Definition

      Bode plots show magnitude (in dB) and phase versus frequency (log scale).

      Axes:

      • X: log⁡10(ω)\log_{10}(\omega)log10​(ω)
      • Y: 20log⁡10(∣H(jω)∣)20\log_{10}(|H(j\omega)|)20log10​(∣H(jω)∣) dB

      Magnitude in dB

      ∣H(jω)∣dB=20log⁡10∣H(jω)∣|H(j\omega)|_{dB} = 20\log_{10}|H(j\omega)|∣H(jω)∣dB​=20log10​∣H(jω)∣

      Conversions

      dB to Linear:

      ∣H∣=10(dB/20)|H| = 10^{(\text{dB}/20)}∣H∣=10(dB/20)

      Linear to dB:

      dB=20log⁡10∣H∣\text{dB} = 20\log_{10}|H|dB=20log10​∣H∣

      Frequency Response from H(s)

      1. Substitute s=jωs = j\omegas=jω
      2. Calculate magnitude: ∣H(jω)∣=[Re(H)]2+[Im(H)]2|H(j\omega)| = \sqrt{[\text{Re}(H)]^2 + [\text{Im}(H)]^2}∣H(jω)∣=[Re(H)]2+[Im(H)]2​
      3. Calculate phase: ∠H(jω)=arctan⁡[Im(H)/Re(H)]\angle H(j\omega) = \arctan[\text{Im}(H)/\text{Re}(H)]∠H(jω)=arctan[Im(H)/Re(H)]

      Sinusoidal Steady-State Response

      Input: vin(t)=Acos⁡(ωt+θ)v_{in}(t) = A\cos(\omega t + \theta)vin​(t)=Acos(ωt+θ)

      Output: vout(t)=A⋅∣H(jω)∣cos⁡(ωt+θ+∠H(jω))v_{out}(t) = A \cdot |H(j\omega)| \cos(\omega t + \theta + \angle H(j\omega))vout​(t)=A⋅∣H(jω)∣cos(ωt+θ+∠H(jω))

      Key Bode Plot Features

      Cutoff Frequency (minus 3 dB point)

      ∣H(jωc)∣=∣Hmax∣2|H(j\omega_c)| = \frac{|H_{max}|}{\sqrt{2}}∣H(jωc​)∣=2​∣Hmax​∣​

      Slopes (Asymptotic)

      • 1st order: plus/minus 20 dB/decade per pole/zero
      • 2nd order: plus/minus 40 dB/decade per pole pair
      • nth order: plus/minus (n times 20) dB/decade

      Phase at Cutoff

      • 1st order: plus/minus 45 degrees at cutoff frequency
      • 2nd order (Butterworth): minus 90 degrees at natural frequency

      First-Order Filter Forms

      Low-Pass:

      H(s)=Kωcs+ωcH(s) = \frac{K\omega_c}{s + \omega_c}H(s)=s+ωc​Kωc​​
      • DC gain = K
      • Slope: minus 20 dB/decade above cutoff frequency

      High-Pass:

      H(s)=Kss+ωcH(s) = \frac{Ks}{s + \omega_c}H(s)=s+ωc​Ks​
      • HF gain = K
      • Slope: plus 20 dB/decade below cutoff frequency

      Second-Order Standard Form
