      Explanation of Filter Design Classes (26, 27, & 28)

      The classes you referenced focus on the design and implementation of active filters, particularly second-order filters using the Sallen-Key topology, and the methodology for achieving higher-order lowpass, highpass, and bandpass characteristics.

      Class 26: Sallen and Key Circuit (Second-Order Filters)

      Class 26 introduces the Sallen-Key circuit, which is an electronic filter topology used to implement second-order active filters.

      TopicHigh-Level Theory and Concepts
      Active vs. Passive FiltersActive filters require a power source, unlike passive filters. Active filters are generally preferred because their performance is maintained irrespective of the load connected, and they can apply additional gain to the signal, whereas passive filters cannot.
      Network Function (H(s))The primary goal is to design a circuit that achieves a specific network function H(s)H(s)H(s). High-order filters are often designed by cascading low-order circuits, resulting in the multiplication of individual transfer functions: H(s)=H1(s)⋅H2(s)⋅⋯⋅Hn(s)H(s) = H_1(s) \cdot H_2(s) \cdot \dots \cdot H_n(s)H(s)=H1​(s)⋅H2​(s)⋅⋯⋅Hn​(s).
      Second-Order ResponseThe general second-order filter response is determined by two key parameters: the damping ratio (ζ\zetaζ) and the undamped natural frequency (ω0\omega_0ω0​). The filter exhibits peaky behavior at the resonant frequency (ωr\omega_rωr​) when the circuit is underdamped (0<ζ<10 < \zeta < 10<ζ<1), meaning the poles of sss are closer to the imaginary axis.
      DampingCircuits are classified by damping ratio ζ\zetaζ: Critically Damped (ζ=1\zeta=1ζ=1), Overdamped (ζ>1\zeta>1ζ>1), and Underdamped (0<ζ<10<\zeta<10<ζ<1). Underdamped circuits possess resonant frequencies.

      Key Equations and Design Methods (Sallen-Key Lowpass):

      The Sallen-Key circuit, typically requiring only a single operational amplifier (op-amp), simplifies design. For a second-order lowpass filter, the denominator of H(s)H(s)H(s) contains a positive power polynomial of sss or jωj\omegajω, meaning that as sss or jωj\omegajω increases, the magnitude of H(s)H(s)H(s) decreases.

      1. Equal Elements Method:

        • Requires choosing equal resistors (R1=R2=RR_1 = R_2 = RR1​=R2​=R) and equal capacitors (C1=C2=CC_1 = C_2 = CC1​=C2​=C).
        • This method is only suitable for underdamped systems (0<ζ<10 < \zeta < 10<ζ<1) because the relationship for the op-amp gain is restricted such that μ≥1\mu \ge 1μ≥1, which limits the range of implementable ζ\zetaζ values (0<ζ<10 < \zeta < 10<ζ<1).

      2. Unity Gain Method:

        • Requires setting the op-amp gain to unity (μ=1\mu = 1μ=1).
        • This method can be used to create overdamped systems (ζ>1\zeta > 1ζ>1) and is more flexible than the Equal Elements Method.

      Class 27: Lowpass Filter Design (Butterworth and Chebyshev)

      Class 27 elaborates on designing high-order lowpass filters by selecting a suitable form of the network function, typically using Butterworth or Chebyshev polynomials.

      TopicHigh-Level Theory and Concepts
      Filter SpecificationDesign begins by defining requirements such as the desired maximum magnitude in the passband (HMAXH_{MAX}HMAX​), the minimum allowable magnitude in the stopband (HMINH_{MIN}HMIN​), and the transition frequency (ωMIN\omega_{MIN}ωMIN​) relative to the cutoff frequency (ωC\omega_CωC​).
      Butterworth FiltersButterworth filters are defined by poles that are evenly distributed in a circle in the sss-plane. This results in a magnitude response that is maximally flat in the passband. For an nthn^{th}nth order Butterworth filter, the stopband slope is −n×20-n \times 20−n×20 dB/decade.
      Chebyshev FiltersChebyshev filters provide a steeper transition band compared to Butterworth filters of the same order, meaning a lesser order circuit can achieve a sharper filter. The poles for Chebyshev polynomials are distributed along an ellipse.
      Chebyshev MagnitudeFor odd order Chebyshev filters, the passband magnitude varies from 111 (0 dB) to 1/21/\sqrt{2}1/2​ (−3-3−3 dB). For even order Chebyshev filters, the passband magnitude varies from 111 (0 dB) to 2\sqrt{2}2​ (333 dB).

      Highpass SlopeFor a 1st1^{st}1st order highpass filter, the slope is +20+20+20 dB/decade. For higher orders, the slope magnitude is n×20n \times 20n×20 dB/decade.

      Relevant High-Level Theory and Prerequisite Topics

      To successfully tackle filter design problems, familiarity with the following core concepts is essential:

      Laplace Transform and Network Functions

      • Laplace Transform (LT): Converts differential equations (ODEs) from the time domain into algebraic expressions in the sss-domain (phase domain), which are easier to solve.
      • Network Function H(s): The ratio of the zero-state response (ZSR) output in the sss-domain to the ZSR input in the sss-domain. The network function H(s)H(s)H(s) can be found using impedances of elements and resistive circuit techniques.
      • Zero-State Response (ZSR): The response due only to the input, assuming zero stored energy (initial conditions are zero).
      • Poles and Zeros: Poles of H(s)H(s)H(s) (roots of the denominator) determine the natural frequencies and exponential modes of the circuit, which constitute the natural response. Zeros (roots of the numerator) determine the frequencies that are filtered out or blocked from the output.

      Frequency Response and Damping

      • Frequency Response: Describes how the circuit responds to sinusoidal input across all frequencies, typically measured by gain and phase shift.
      • Bode Plots: Graphical representation of frequency response, plotting the magnitude (in dB) and phase angle versus the logarithm of frequency. Filter cutoff rates are expressed as slopes on the Bode magnitude plot (e.g., 20 dB/decade for a first-order pole).
      • Second-Order Parameters: The standard second-order system characteristic equation includes the damping ratio ζ\zetaζ and the undamped natural frequency ω0\omega_0ω0​. For active filters, the bandwidth is given by B=2ζω0B = 2\zeta\omega_0B=2ζω0​.

      Design Implementation

      • Op Amps: Necessary for creating active filters and implementing high-order systems via cascading low-order sections so that the overall transfer function is the product of the individual sections: H(s)=H1(s)⋅H2(s)⋅⋯⋅Hn(s)H(s) = H_1(s) \cdot H_2(s) \cdot \dots \cdot H_n(s)H(s)=H1​(s)⋅H2​(s)⋅⋯⋅Hn​(s).
      • Impedances: Components are represented by their sss-domain impedances: a resistor Z=RZ=RZ=R, and a capacitor Z=1/(sC)Z=1/(sC)Z=1/(sC) (or admittance Y=sCY=sCY=sC).

      Comprehensive Step-by-Step Process for Filter Design

      The general process for designing filters typically involves determining the minimum required performance characteristics and translating those needs into a realizable electronic circuit.

      Phase I: Specification and Transfer Function Determination

      This phase translates verbal requirements into a precise mathematical transfer function H(s)H(s)H(s):

      1. Define Filter Specifications (Requirements):

        • Determine the filter type required: Lowpass, Highpass, Bandpass, or Band Stop.
        • Specify the target cutoff frequency (ωC\omega_CωC​).
        • Define the maximum required magnitude in the passband (HMAXH_{MAX}HMAX​) and the minimum required magnitude in the stopband (HMINH_{MIN}HMIN​), often in relation to the magnitude at a specific transition frequency (ωMIN\omega_{MIN}ωMIN​).
        • Note any phase or transient response requirements (though these are often checked/refined in Phase II).

      2. Determine Filter Order (nnn) and Type (Butterworth/Chebyshev):

        • Select the polynomial type (Butterworth for flat response, Chebyshev for sharp transition/steeper slope).
        • Calculate the minimum necessary order (nnn) using the relevant Lowpass or Highpass formulas corresponding to the chosen polynomial type (e.g., nBn_BnB​ or nCn_CnC​) and requirements. The design must use an integer order equal to or greater than the calculated nnn.

      3. Define the Overall Transfer Function H(s)H(s)H(s):

        • Using the determined order (nnn) and polynomial type (Butterworth or Chebyshev), define the overall transfer function H(s)H(s)H(s) by referencing standard polynomial tables or formulas.
        • The resulting H(s)H(s)H(s) will typically be a ratio of polynomials, often with a constant numerator for lowpass filters (or sns^nsn in the numerator for highpass filters).

      Phase II: Decomposition and Circuit Design (Sallen-Key)

      This phase breaks the high-order transfer function into implementable low-order active filter sections:

      1. Decompose H(s)H(s)H(s) into Low-Order Sections:

        • Factor the denominator of the overall H(s)H(s)H(s) into a sequence of first-order terms and/or second-order quadratic terms.
        • Express the required transfer function as a cascade (multiplication) of simple transfer functions, H(s)=H1(s)⋅H2(s)⋅…H(s) = H_1(s) \cdot H_2(s) \cdot \dotsH(s)=H1​(s)⋅H2​(s)⋅…. These sections will typically be implemented using Op Amps.

      2. Design Individual Second-Order Sections (Sallen-Key):

        • For each second-order section Hk(s)H_k(s)Hk​(s), compare its polynomial factors to the canonical second-order form to determine the required damping ratio (ζ\zetaζ) and undamped natural frequency (ω0\omega_0ω0​) for that specific section.
        • Select the Sallen-Key Design Method:
          • If the section is critically damped (ζ=1\zeta=1ζ=1) or overdamped (ζ>1\zeta>1ζ>1), use the Unity Gain Method (μ=1\mu=1μ=1).
          • If the section is underdamped (0<ζ<10<\zeta<10<ζ<1), either method can be used, but the Equal Elements Method may offer simplicity.
        • Calculate the necessary component values (R and C) for the chosen design method by comparing the terms in the prototype Sallen-Key transfer function equation to the decomposed section Hk(s)H_k(s)Hk​(s).

      3. Handle Specialized Filter Types:

        • Highpass Sections: Implement using the Sallen-Key Highpass topology, which requires swapping the roles of resistors and capacitors relative to the lowpass prototype.
        • Bandpass/Band Stop Filters: Design the constituent lowpass and highpass components separately (steps 1–5), ensuring the appropriate cutoff frequencies are met to create the required pass or stop bands. Implement the overall filter either by cascading (Bandpass) or summing (Band Stop) the resulting sections.

      4. Assemble the Final Circuit:

        • Cascade the designed low-order sections using Op Amps (or buffers) to realize the total high-order H(s)H(s)H(s).
        • Incorporate any first-order sections using simple voltage divider configurations or equivalent active filters.
        • Implement any specific gain requirements by adjusting the input or feedback impedance of the Op Amp stages.
