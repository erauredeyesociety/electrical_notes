      Complex Frequency and the s-Domain

      The complex frequency variable is defined as:

      s=σ+jωs = \sigma + j\omegas=σ+jω

      where:

      • σ\sigmaσ is the real part (damping/growth rate)
      • ω\omegaω is the imaginary part (oscillation frequency)
      • j=−1j = \sqrt{-1}j=−1​

      For frequency response analysis, we evaluate H(s)H(s)H(s) along the imaginary axis by substituting s=jωs = j\omegas=jω:

      H(jω)=∣H(jω)∣∠ϕ(ω)H(j\omega) = |H(j\omega)| \angle \phi(\omega)H(jω)=∣H(jω)∣∠ϕ(ω)

      The magnitude in decibels:

      ∣H(jω)∣dB=20log⁡10∣H(jω)∣|H(j\omega)|_{dB} = 20\log_{10}|H(j\omega)|∣H(jω)∣dB​=20log10​∣H(jω)∣

      Standard Second-Order Transfer Function

      The general form of a second-order transfer function is:

      H(s)=K⋅N(s)s2+2ζω0s+ω02H(s) = \frac{K \cdot N(s)}{s^2 + 2\zeta\omega_0 s + \omega_0^2}H(s)=s2+2ζω0​s+ω02​K⋅N(s)​

      where:

      • KKK = DC gain or high-frequency gain
      • N(s)N(s)N(s) = numerator polynomial (determines filter type)
      • ζ\zetaζ = damping ratio
      • ω0\omega_0ω0​ = undamped natural frequency (rad/s)

      Filter-specific numerators:

      • Lowpass: N(s)=ω02N(s) = \omega_0^2N(s)=ω02​ → H(s)=Kω02s2+2ζω0s+ω02H(s) = \frac{K\omega_0^2}{s^2 + 2\zeta\omega_0 s + \omega_0^2}H(s)=s2+2ζω0​s+ω02​Kω02​​
      • Highpass: N(s)=s2N(s) = s^2N(s)=s2 → H(s)=Ks2s2+2ζω0s+ω02H(s) = \frac{Ks^2}{s^2 + 2\zeta\omega_0 s + \omega_0^2}H(s)=s2+2ζω0​s+ω02​Ks2​
      • Bandpass: N(s)=2ζω0sN(s) = 2\zeta\omega_0 sN(s)=2ζω0​s → H(s)=K⋅2ζω0ss2+2ζω0s+ω02H(s) = \frac{K \cdot 2\zeta\omega_0 s}{s^2 + 2\zeta\omega_0 s + \omega_0^2}H(s)=s2+2ζω0​s+ω02​K⋅2ζω0​s​
      • Notch (Band Stop): N(s)=s2+ω02N(s) = s^2 + \omega_0^2N(s)=s2+ω02​ → H(s)=K(s2+ω02)s2+2ζω0s+ω02H(s) = \frac{K(s^2 + \omega_0^2)}{s^2 + 2\zeta\omega_0 s + \omega_0^2}H(s)=s2+2ζω0​s+ω02​K(s2+ω02​)​

      Pole-Zero Analysis

      Poles are the roots of the denominator:

      s2+2ζω0s+ω02=0s^2 + 2\zeta\omega_0 s + \omega_0^2 = 0s2+2ζω0​s+ω02​=0

      Solution using the quadratic formula:

      s1,2=−ζω0±ω0ζ2−1s_{1,2} = -\zeta\omega_0 \pm \omega_0\sqrt{\zeta^2 - 1}s1,2​=−ζω0​±ω0​ζ2−1​

      Three damping cases:

      1. Underdamped (0<ζ<10 < \zeta < 10<ζ<1): Complex conjugate poles

        s1,2=−ζω0±jω01−ζ2=−σ±jωds_{1,2} = -\zeta\omega_0 \pm j\omega_0\sqrt{1-\zeta^2} = -\sigma \pm j\omega_ds1,2​=−ζω0​±jω0​1−ζ2​=−σ±jωd​

        where ωd=ω01−ζ2\omega_d = \omega_0\sqrt{1-\zeta^2}ωd​=ω0​1−ζ2​ is the damped natural frequency

      2. Critically Damped (ζ=1\zeta = 1ζ=1): Two real, equal poles

        s1,2=−ω0s_{1,2} = -\omega_0s1,2​=−ω0​

      3. Overdamped (ζ>1\zeta > 1ζ>1): Two real, distinct poles

        s1,2=−ζω0±ω0ζ2−1s_{1,2} = -\zeta\omega_0 \pm \omega_0\sqrt{\zeta^2 - 1}s1,2​=−ζω0​±ω0​ζ2−1​

      Quality Factor (Q)

      The quality factor is inversely related to damping:

      Q=12ζQ = \frac{1}{2\zeta}Q=2ζ1​

      or equivalently:

      ζ=12Q\zeta = \frac{1}{2Q}ζ=2Q1​

      Physical interpretations:

      • Q>0.5Q > 0.5Q>0.5 (ζ<1\zeta < 1ζ<1): Underdamped, exhibits resonance
      • Q=0.5Q = 0.5Q=0.5 (ζ=1\zeta = 1ζ=1): Critically damped
      • Q<0.5Q < 0.5Q<0.5 (ζ>1\zeta > 1ζ>1): Overdamped

      Resonant frequency (for underdamped systems):

      ωr=ω01−2ζ2=ω01−12Q2\omega_r = \omega_0\sqrt{1-2\zeta^2} = \omega_0\sqrt{1-\frac{1}{2Q^2}}ωr​=ω0​1−2ζ2​=ω0​1−2Q21​​

      Note: Resonance only exists when ζ<1/2\zeta < 1/\sqrt{2}ζ<1/2​ (i.e., Q>1/2Q > 1/\sqrt{2}Q>1/2​)

      Peak magnitude at resonance:

      ∣H(jωr)∣max=Q1−14Q2≈Q (for Q≫1)|H(j\omega_r)|_{max} = \frac{Q}{\sqrt{1-\frac{1}{4Q^2}}} \approx Q \text{ (for } Q \gg 1\text{)}∣H(jωr​)∣max​=1−4Q21​​Q​≈Q (for Q≫1)

      Butterworth Filter Theory

      Magnitude Response

      The magnitude-squared function for an nthn^{th}nth-order Butterworth lowpass filter:

      ∣H(jω)∣2=11+(ωωC)2n|H(j\omega)|^2 = \frac{1}{1 + \left(\frac{\omega}{\omega_C}\right)^{2n}}∣H(jω)∣2=1+(ωC​ω​)2n1​

      where ωC\omega_CωC​ is the -3dB cutoff frequency (half-power point).

      Key properties:

      • At DC (ω=0\omega = 0ω=0): ∣H(0)∣=1|H(0)| = 1∣H(0)∣=1 (0 dB)
      • At cutoff (ω=ωC\omega = \omega_Cω=ωC​): ∣H(jωC)∣=12|H(j\omega_C)| = \frac{1}{\sqrt{2}}∣H(jωC​)∣=2​1​ (-3 dB)
      • Asymptotic slope: −20n-20n−20n dB/decade for ω≫ωC\omega \gg \omega_Cω≫ωC​

      Maximally flat property:

      The first (2n−1)(2n-1)(2n−1) derivatives of ∣H(jω)∣2|H(j\omega)|^2∣H(jω)∣2 are zero at ω=0\omega = 0ω=0, giving the flattest possible passband response.

      Pole Locations

      Butterworth poles lie on a circle of radius ωC\omega_CωC​ in the left-half s-plane:

      sk=ωC⋅ej(π/2+(2k+n−1)π/(2n))s_k = \omega_C \cdot e^{j(\pi/2 + (2k+n-1)\pi/(2n))}sk​=ωC​⋅ej(π/2+(2k+n−1)π/(2n))

      for k=1,2,...,nk = 1, 2, ..., nk=1,2,...,n

      Simplified angle formula:

      θk=π2+(2k+n−1)π2n=(2k+n−1)π2n\theta_k = \frac{\pi}{2} + \frac{(2k+n-1)\pi}{2n} = \frac{(2k+n-1)\pi}{2n}θk​=2π​+2n(2k+n−1)π​=2n(2k+n−1)π​

      For even nnn: Poles are symmetrically placed, no pole on the real axis

      For odd nnn: One pole at s=−ωCs = -\omega_Cs=−ωC​, remaining poles in conjugate pairs

      The denominator polynomials Bn(s)B_n(s)Bn​(s) for normalized Butterworth filters (ωC=1\omega_C = 1ωC​=1):

      • B5(s)=(s+1)(s2+0.618s+1)(s2+1.618s+1)B_5(s) = (s+1)(s^2 + 0.618s + 1)(s^2 + 1.618s + 1)B5​(s)=(s+1)(s2+0.618s+1)(s2+1.618s+1)
      • B6(s)=(s2+0.518s+1)(s2+1.414s+1)(s2+1.932s+1)B_6(s) = (s^2 + 0.518s + 1)(s^2 + 1.414s + 1)(s^2 + 1.932s + 1)B6​(s)=(s2+0.518s+1)(s2+1.414s+1)(s2+1.932s+1)

      Frequency scaling: Replace sss with s/ωCs/\omega_Cs/ωC​ to denormalize.

      Magnitude-squared function:

      • ωC\omega_CωC​ = passband edge frequency

      Cascade Design Methodology

      Section Ordering

      For optimal noise and dynamic range, order sections by increasing Q:

      1. Lowest Q (most damped) section first
      2. Highest Q (least damped) section last

      This prevents early stages from clipping due to resonant peaks.

      Inter-stage Loading

      Buffer requirement: Each Sallen-Key section should be buffered (high input impedance) to prevent loading of previous stages.

      Op-amp as buffer: The unity-gain follower (μ=1\mu = 1μ=1) naturally provides buffering.

      Overall Gain Distribution

      Option 1: Implement all gain in first stage Option 2: Distribute gain equally across all stages Option 3: Implement gain in final stage

      Trade-offs involve noise, distortion, and signal-to-noise ratio at each stage.
