      PHASE 1: CALCULATE REQUIRED FILTER ORDER

      Given Specifications (Typical)

      • Filter Type: LP, HP, BP, or BS
      • HMAXH_{MAX}HMAX​: Maximum passband gain (linear or dB)
      • HMINH_{MIN}HMIN​: Minimum stopband gain (linear or dB)
      • ωC\omega_CωC​ or fCf_CfC​: Cutoff/corner frequency
      • ωMIN\omega_{MIN}ωMIN​ or fMINf_{MIN}fMIN​: Frequency where HMINH_{MIN}HMIN​ must be met

      Step 1.1: Convert dB to Linear (if needed)

      Hlinear=10HdB/20H_{\text{linear}} = 10^{H_{\text{dB}}/20}Hlinear​=10HdB​/20

      Step 1.2: Calculate Minimum Order n

      Low-Pass Butterworth:

      n≥12⋅ln⁡[(HMAX/HMIN)2−1]ln⁡(ωMIN/ωC)n \geq \frac{1}{2} \cdot \frac{\ln[(H_{MAX}/H_{MIN})^2 - 1]}{\ln(\omega_{MIN}/\omega_C)}n≥21​⋅ln(ωMIN​/ωC​)ln[(HMAX​/HMIN​)2−1]​

      High-Pass Butterworth:

      n≥12⋅ln⁡[(HMAX/HMIN)2−1]ln⁡(ωC/ωMIN)n \geq \frac{1}{2} \cdot \frac{\ln[(H_{MAX}/H_{MIN})^2 - 1]}{\ln(\omega_C/\omega_{MIN})}n≥21​⋅ln(ωC​/ωMIN​)ln[(HMAX​/HMIN​)2−1]​

      ⚠️ CRITICAL: Always round UP to nearest integer

      Step 1.3: Determine Section Count

      Always use as many second order circuits if possible. If an odd number n, then the most second order and then 1 first order.

      General formula:

      • Odd n: (n−1)/2(n-1)/2(n−1)/2 second-order + 1 first-order
      • Even n: n/2n/2n/2 second-order

      PHASE 2: OBTAIN BUTTERWORTH POLES

      Step 2.1: Calculate Pole Angles

      For normalized Butterworth (ωC=1\omega_C = 1ωC​=1):

      θk=π2+(2k−1)π2n,k=1,2,…,n\theta_k = \frac{\pi}{2} + \frac{(2k-1)\pi}{2n}, \quad k = 1, 2, \ldots, nθk​=2π​+2n(2k−1)π​,k=1,2,…,n

      k is simply an index used to enumerate the poles of the normalized Butterworth polynomial. Nothing more. It has no electrical meaning like gain, component value, or stage count.

      Step 2.2: Calculate Pole Locations

      pk=ejθk=cos⁡(θk)+jsin⁡(θk)p_k = e^{j\theta_k} = \cos(\theta_k) + j\sin(\theta_k)pk​=ejθk​=cos(θk​)+jsin(θk​)

      Only use left half-plane poles (Real part < 0)

      PHASE 3: EXTRACT SECTION PARAMETERS

      For 2nd-Order Section from Conjugate Pair

      Step 3.1: Form quadratic

      Poles p=α+jβp = \alpha + j\betap=α+jβ and p∗=α−jβp^* = \alpha - j\betap∗=α−jβ:

      (s−p)(s−p∗)=s2−2αs+(α2+β2)(s - p)(s - p^*) = s^2 - 2\alpha s + (\alpha^2 + \beta^2)(s−p)(s−p∗)=s2−2αs+(α2+β2)

      For normalized: ∣p∣=1|p| = 1∣p∣=1, so:

      s2+as+1s^2 + as + 1s2+as+1

      where a=−2Re(p)=−2cos⁡(θk)a = -2\text{Re}(p) = -2\cos(\theta_k)a=−2Re(p)=−2cos(θk​)

      Step 3.2: Extract damping ratio ζ\zetaζ

      Standard form: s2+2ζω0s+ω02s^2 + 2\zeta\omega_0 s + \omega_0^2s2+2ζω0​s+ω02​

      For normalized (ω0=1\omega_0 = 1ω0​=1): s2+2ζs+1s^2 + 2\zeta s + 1s2+2ζs+1

      ζk=a2=−cos⁡(θk)\zeta_k = \frac{a}{2} = -\cos(\theta_k)ζk​=2a​=−cos(θk​)

      Step 3.3: Calculate Q factor

      Qk=12ζkQ_k = \frac{1}{2\zeta_k}Qk​=2ζk​1​

      PHASE 5A: UNITY GAIN METHOD (RECOMMENDED)

      Circuit Description

      Op-amp as voltage follower (buffer), non-inverting input connects to RC network, output feeds back directly to inverting input. Components: 2 resistors (R1,R2R_1, R_2R1​,R2​) and 2 capacitors (C1,C2C_1, C_2C1​,C2​).

      Design Equations

      Step 1: Set gain

      μ=1\mu = 1μ=1

      Step 2: Choose starting component

      Option A – Choose C1C_1C1​ (typical: 0.01 to 1 µF)

      C2=(2ζ)2⋅C1=4ζ2C1C_2 = (2\zeta)^2 \cdot C_1 = 4\zeta^2 C_1C2​=(2ζ)2⋅C1​=4ζ2C1​R1=R2=R=1ζω0C1R_1 = R_2 = R = \frac{1}{\zeta \omega_0 C_1}R1​=R2​=R=ζω0​C1​1​

      Option B – Choose R2R_2R2​ (typical: 1k to 100k)

      R1=(2ζ)2⋅R2=4ζ2R2R_1 = (2\zeta)^2 \cdot R_2 = 4\zeta^2 R_2R1​=(2ζ)2⋅R2​=4ζ2R2​C1=C2=C=1ζω0R2C_1 = C_2 = C = \frac{1}{\zeta \omega_0 R_2}C1​=C2​=C=ζω0​R2​1​

      PHASE 5B: EQUAL ELEMENTS METHOD

      Circuit Description

      Op-amp with gain > 1 (non-inverting configuration). Feedback resistors RfR_fRf​ and RgR_gRg​ set gain. Components: Equal resistors R1=R2=RR_1 = R_2 = RR1​=R2​=R and equal capacitors C1=C2=CC_1 = C_2 = CC1​=C2​=C in filter network.

      Design Equations

      Step 1: Calculate required gain

      μ=3−2ζ\mu = 3 - 2\zetaμ=3−2ζ

      Validity check: 1≤μ≤3⇔0<ζ<11 \leq \mu \leq 3 \Leftrightarrow 0 < \zeta < 11≤μ≤3⇔0<ζ<1. If outside this range, cannot use this method.

      Step 2: Choose C (typical: 0.01 to 1 µF)

      R=1ω0CR = \frac{1}{\omega_0 C}R=ω0​C1​

      Step 3: Design op-amp gain circuit

      μ=1+RfRg\mu = 1 + \frac{R_f}{R_g}μ=1+Rg​Rf​​Rf=(μ−1)⋅RgR_f = (\mu - 1) \cdot R_gRf​=(μ−1)⋅Rg​

      Typical: Choose Rg=10R_g = 10Rg​=10 kΩ\OmegaΩ, then: Rf=(μ−1)×10R_f = (\mu - 1) \times 10Rf​=(μ−1)×10 kΩ\OmegaΩ

      PHASE 6: FIRST-ORDER SECTIONS

      For Real Pole at s=−ωCs = -\omega_Cs=−ωC​

      Components: 1 resistor (R) and 1 capacitor (C).

      Passive RC Low-Pass:

      H(s)=11+sRC,ωC=1RCH(s) = \frac{1}{1 + sRC}, \quad \omega_C = \frac{1}{RC}H(s)=1+sRC1​,ωC​=RC1​

      Choose C, solve for R: R=1ωCCR = \frac{1}{\omega_C C}R=ωC​C1​

      Active Low-Pass (with gain K):

      H(s)=K1+sRCH(s) = \frac{K}{1 + sRC}H(s)=1+sRCK​

      Use non-inverting op-amp: K=1+RfRgK = 1 + \frac{R_f}{R_g}K=1+Rg​Rf​​

      High-Pass (1st order):

      H(s)=sRC1+sRCH(s) = \frac{sRC}{1 + sRC}H(s)=1+sRCsRC​

      Swap R and C positions from low-pass.

      PHASE 7: CASCADE ASSEMBLY

      Step 7.1: Section Ordering (CRITICAL)

      Order sections by Q: LOW → HIGH

      Place highest ζ\zetaζ (lowest Q) first, lowest ζ\zetaζ (highest Q) last. Why? Prevents early clipping from resonant peaks.

      Step 7.2: Buffering

      Each Sallen-Key stage has high input impedance (op-amp input), so stages naturally don’t load each other. No additional buffers needed between stages.

      Step 7.3: Overall Transfer Function

      Htotal(s)=H1(s)×H2(s)×⋯×Hn(s)H_{\text{total}}(s) = H_1(s) \times H_2(s) \times \cdots \times H_n(s)Htotal​(s)=H1​(s)×H2​(s)×⋯×Hn​(s)

      OP-AMP FUNDAMENTALS

      Why Use Op-Amps?

      • High input impedance (prevents loading previous stages)
      • Low output impedance (drives next stage without degradation)
      • Enable gain without passive component losses
      • Allow cascading of sections without interaction

      Key Configurations

      1. Voltage Follower (Buffer): μ=1\mu = 1μ=1 exactly. Use for Unity Gain Method. Output connects directly to inverting input. Most stable configuration.

      2. Non-Inverting Amplifier: μ=1+RfRg\mu = 1 + \frac{R_f}{R_g}μ=1+Rg​Rf​​. Use for Equal Elements Method when 1<μ<31 < \mu < 31<μ<3. Maintains input signal polarity.

      3. Inverting Amplifier: μ=−RfRin\mu = -\frac{R_f}{R_{in}}μ=−Rin​Rf​​. NOT typically used in Sallen-Key filters. Can be used for summing (bandstop filters).

      4. Summing Amplifier: Vout=−Rf(V1R1+V2R2)V_{out} = -R_f\left(\frac{V_1}{R_1} + \frac{V_2}{R_2}\right)Vout​=−Rf​(R1​V1​​+R2​V2​​). Required for bandstop (notch) filters to combine LP and HP paths.

      Op-Amp Placement

      • Within each Sallen-Key section: Required (one per 2nd-order section)
      • After passive RC sections: Required (use buffer)
      • Between active sections: Usually NOT needed (already buffered)
      • At filter output: Recommended if driving cables/loads

      Critical Op-Amp Specifications

      Gain-Bandwidth Product (GBW): GBW≥100×fc×Q×μGBW \geq 100 \times f_c \times Q \times \muGBW≥100×fc​×Q×μ

      Slew Rate (SR): SR≥2πfmaxVpeakSR \geq 2\pi f_{max} V_{peak}SR≥2πfmax​Vpeak​

      Common choices: TL072/LF412 (audio), LM358/TL082 (general), TL071/OPA2134 (high-speed)

      Power Supply & Decoupling

      Use dual supply (±15V, ±12V, or ±5V typical). Always add 0.1 µF ceramic capacitor at each IC between V+ and GND, and between V− and GND. Prevents oscillation and noise.

      DERIVING MISSING SPECIFICATIONS

      If ωC\omega_CωC​ Given, But ωMIN\omega_{MIN}ωMIN​ Not Given

      Typical assumption: Transition band = 1 decade

      ωMIN=10×ωC (low-pass),ωMIN=0.1×ωC (high-pass)\omega_{MIN} = 10 \times \omega_C \text{ (low-pass)}, \quad \omega_{MIN} = 0.1 \times \omega_C \text{ (high-pass)}ωMIN​=10×ωC​ (low-pass),ωMIN​=0.1×ωC​ (high-pass)

      If HMINH_{MIN}HMIN​ Not Given

      Use standard attenuation: -40 dB (0.01) for audio, -60 dB (0.001) for instrumentation

      If ζ\zetaζ or Q Given for Section

      Q=12ζ,ζ=12QQ = \frac{1}{2\zeta}, \quad \zeta = \frac{1}{2Q}Q=2ζ1​,ζ=2Q1​

      If ω0\omega_0ω0​ and ζ\zetaζ Given, Find Pole Locations

      Complex conjugate poles: p=−ζω0±jω01−ζ2p = -\zeta\omega_0 \pm j\omega_0\sqrt{1-\zeta^2}p=−ζω0​±jω0​1−ζ2​

      Real part: σ=−ζω0\sigma = -\zeta\omega_0σ=−ζω0​, Imaginary part: ωd=ω01−ζ2\omega_d = \omega_0\sqrt{1-\zeta^2}ωd​=ω0​1−ζ2​ (damped frequency)

      Bandwidth Relationships

      For 2nd-order: BW=2ζω0=ω0QBW = 2\zeta\omega_0 = \frac{\omega_0}{Q}BW=2ζω0​=Qω0​​

      Half-power frequencies:

      ω1=ω0(−ζ+ζ2+1),ω2=ω0(ζ+ζ2+1)\omega_1 = \omega_0\left(-\zeta + \sqrt{\zeta^2+1}\right), \quad \omega_2 = \omega_0\left(\zeta + \sqrt{\zeta^2+1}\right)ω1​=ω0​(−ζ+ζ2+1​),ω2​=ω0​(ζ+ζ2+1​)

      BANDPASS AND BANDSTOP FILTER DESIGN

      Bandpass Filter (BP)

      Concept: Pass middle band, attenuate low and high frequencies.

      Design: Cascade lowpass and highpass: HBP(s)=HLP(s)×HHP(s)H_{BP}(s) = H_{LP}(s) \times H_{HP}(s)HBP​(s)=HLP​(s)×HHP​(s)

      Steps:

      1. Define: Lower cutoff ω1\omega_1ω1​ (HP), upper cutoff ω2\omega_2ω2​ (LP), center ω0=ω1ω2\omega_0 = \sqrt{\omega_1 \omega_2}ω0​=ω1​ω2​​, bandwidth BW=ω2−ω1BW = \omega_2 - \omega_1BW=ω2​−ω1​
      2. Design HP section with ωC=ω1\omega_C = \omega_1ωC​=ω1​
      3. Design LP section with ωC=ω2\omega_C = \omega_2ωC​=ω2​
      4. Cascade: HP → LP (or LP → HP)

      Key: Total order = (LP order) + (HP order). Use series connection.

      Bandstop (Notch) Filter (BS)

      Concept: Attenuate middle band, pass low and high frequencies.

      Design: Parallel sum: HBS(s)=HLP(s)+HHP(s)H_{BS}(s) = H_{LP}(s) + H_{HP}(s)HBS​(s)=HLP​(s)+HHP​(s)

      Steps:

      1. Define: Lower passband edge ω1\omega_1ω1​ (LP cutoff), upper passband edge ω2\omega_2ω2​ (HP cutoff), notch center ω0=ω1ω2\omega_0 = \sqrt{\omega_1 \omega_2}ω0​=ω1​ω2​​
      2. Design LP section with ωC=ω1\omega_C = \omega_1ωC​=ω1​ (passes below notch)
      3. Design HP section with ωC=ω2\omega_C = \omega_2ωC​=ω2​ (passes above notch)
      4. Use summing amplifier to combine outputs

      Key: Unlike BP (cascade), BS requires parallel combination via summing amplifier. Both sections must have same DC/HF gain for flat response.

      Comparison: BP vs BS Implementation

      AspectBandpass (BP)Bandstop (BS)
      CombinationCascade (multiply)Parallel (sum)
      CircuitSeries connectionSumming amplifier
      PassesMiddle bandLow & High
      AttenuatesLow & HighMiddle band

      Order Calculation

      HP: n≥ln⁡[(HMAX/HMIN)2−1]2ln⁡(ωC/ωMIN)n \geq \frac{\ln[(H_{MAX}/H_{MIN})^2-1]}{2\ln(\omega_C/\omega_{MIN})}n≥2ln(ωC​/ωMIN​)ln[(HMAX​/HMIN​)2−1]​

      Pole Angles

      θk=π2+(2k−1)π2n\theta_k = \frac{\pi}{2} + \frac{(2k-1)\pi}{2n}θk​=2π​+2n(2k−1)π​

      Damping & Q

      ζ=−cos⁡(θk)=a2,Q=12ζ\zeta = -\cos(\theta_k) = \frac{a}{2}, \quad Q = \frac{1}{2\zeta}ζ=−cos(θk​)=2a​,Q=2ζ1​

      Unity Gain

      C2=4ζ2C1,R1=R2=1ζω0C1C_2 = 4\zeta^2 C_1, \quad R_1 = R_2 = \frac{1}{\zeta\omega_0 C_1}C2​=4ζ2C1​,R1​=R2​=ζω0​C1​1​

      Equal Elements

      μ=3−2ζ,R=1ω0C\mu = 3 - 2\zeta, \quad R = \frac{1}{\omega_0 C}μ=3−2ζ,R=ω0​C1​

      CRITICAL CHECKLIST

      • Converted all dB to linear
      • Rounded n UP to integer
      • Extracted all poles (left half-plane only)
      • Grouped poles into sections
      • Calculated ζ\zetaζ and Q for each section
      • Checked ζ<1\zeta < 1ζ<1 before using Equal Elements
      • Verified 1≤μ≤31 \leq \mu \leq 31≤μ≤3 for Equal Elements
      • Ordered sections LOW Q → HIGH Q
      • Scaled components to practical values
      • Verified Htotal(s)H_{\text{total}}(s)Htotal​(s) = product of sections

      COMMON MISTAKES TO AVOID

      ❌ Rounding n down instead of up

      ❌ Using Equal Elements when ζ≥1\zeta \geq 1ζ≥1

      ❌ Forgetting to convert dB to linear

      ❌ Wrong frequency ratio (LP vs HP)

      ❌ Cascading high-Q sections first

      ❌ Using right half-plane poles

      ❌ Forgetting to denormalize from ωC=1\omega_C = 1ωC​=1

      COMPLETE WORKED EXAMPLE

      Given Requirements

      Low-pass filter: HMAX=0H_{MAX} = 0HMAX​=0 dB (1.0), HMIN=−40H_{MIN} = -40HMIN​=−40 dB (0.01), fC=1f_C = 1fC​=1 kHz, fMIN=5f_{MIN} = 5fMIN​=5 kHz

      Solution

      Calculate order: n≥ln⁡[(1/0.01)2−1]2ln⁡(5)=2.86n \geq \frac{\ln[(1/0.01)^2-1]}{2\ln(5)} = 2.86n≥2ln(5)ln[(1/0.01)2−1]​=2.86 → Round up: n = 3
