      Filter Design Cheatsheet

      DESIGN PROCESS OVERVIEW

      PHASE 1: IDENTIFY SPECIFICATIONS

      • Transfer function H(s)H(s)H(s) given → Phase 3
      • Frequency specs (ωc\omega_cωc​, HMAXH_{MAX}HMAX​, HMINH_{MIN}HMIN​, ωMIN\omega_{MIN}ωMIN​) → Phase 2
      • Time response → Extract ζ\zetaζ, ω0\omega_0ω0​ → Phase 2
      • Bode plot → Extract poles/zeros → Phase 3

      Filter Type Identification:

      • Lowpass: H(s)=K/(s2+bs+c)H(s) = K/(s^2 + bs + c)H(s)=K/(s2+bs+c) or K/(s+α)K/(s + \alpha)K/(s+α)
      • Highpass: H(s)=Ks2/(s2+bs+c)H(s) = Ks^2/(s^2 + bs + c)H(s)=Ks2/(s2+bs+c) or Ks/(s+α)Ks/(s + \alpha)Ks/(s+α)
      • Bandpass: H(s)=Ks/(s2+bs+c)H(s) = Ks/(s^2 + bs + c)H(s)=Ks/(s2+bs+c)
      • Band-Stop: H(s)=K(s2+ω02)/(s2+bs+c)H(s) = K(s^2 + \omega_0^2)/(s^2 + bs + c)H(s)=K(s2+ω02​)/(s2+bs+c)

      PHASE 2: CALCULATE FILTER ORDER

      Butterworth (Maximally Flat):

      • Lowpass: nB=⌈12log⁡[(HMAX/HMIN)2−1]log⁡(ωMIN/ωc)⌉n_B = \lceil \frac{1}{2} \frac{\log[(H_{MAX}/H_{MIN})^2 - 1]}{\log(\omega_{MIN}/\omega_c)} \rceilnB​=⌈21​log(ωMIN​/ωc​)log[(HMAX​/HMIN​)2−1]​⌉
      • Highpass: nB=⌈12log⁡[(HMAX/HMIN)2−1]log⁡(ωc/ωMIN)⌉n_B = \lceil \frac{1}{2} \frac{\log[(H_{MAX}/H_{MIN})^2 - 1]}{\log(\omega_c/\omega_{MIN})} \rceilnB​=⌈21​log(ωc​/ωMIN​)log[(HMAX​/HMIN​)2−1]​⌉

      Chebyshev (Steeper Roll-off):

      • Lowpass: nC=⌈cosh⁡−1((HMAX/HMIN)2−1)cosh⁡−1(ωMIN/ωc)⌉n_C = \lceil \frac{\cosh^{-1}(\sqrt{(H_{MAX}/H_{MIN})^2 - 1})}{\cosh^{-1}(\omega_{MIN}/\omega_c)} \rceilnC​=⌈cosh−1(ωMIN​/ωc​)cosh−1((HMAX​/HMIN​)2−1​)​⌉
      • Highpass: nC=⌈cosh⁡−1((HMAX/HMIN)2−1)cosh⁡−1(ωc/ωMIN)⌉n_C = \lceil \frac{\cosh^{-1}(\sqrt{(H_{MAX}/H_{MIN})^2 - 1})}{\cosh^{-1}(\omega_c/\omega_{MIN})} \rceilnC​=⌈cosh−1(ωc​/ωMIN​)cosh−1((HMAX​/HMIN​)2−1​)​⌉

      Round up to nearest integer!

      PHASE 3: FACTOR H(s) INTO SECTIONS

      1. Get poles from polynomial tables
      2. Factor: H(s)=K/[(s+α1)(s2+b2s+c2)(s2+b3s+c3)]H(s) = K / [(s + \alpha_1)(s^2 + b_2s + c_2)(s^2 + b_3s + c_3)]H(s)=K/[(s+α1​)(s2+b2​s+c2​)(s2+b3​s+c3​)]
      3. Real pole → 1st order section
      4. Complex pair → 2nd order section

      Extract 2nd order parameters:

      • ω0=c\omega_0 = \sqrt{c}ω0​=c​
      • ζ=b2c\zeta = \frac{b}{2\sqrt{c}}ζ=2c​b​
      • Q=cb=12ζQ = \frac{\sqrt{c}}{b} = \frac{1}{2\zeta}Q=bc​​=2ζ1​

      PHASE 4: DESIGN 2ND ORDER SECTIONS (SALLEN-KEY)

      ALWAYS TRY UNITY GAIN FIRST!

      Unity Gain Method (μ=1\mu = 1μ=1) - PREFERRED:

      • Works for ALL ζ\zetaζ values
      • Most stable configuration

      Lowpass:

      1. Set μ=1\mu = 1μ=1 (buffer)
      2. Choose C1C_1C1​
      3. C2=4ζ2C1C_2 = 4\zeta^2 C_1C2​=4ζ2C1​
      4. R1=R2=1ζω0C1R_1 = R_2 = \frac{1}{\zeta\omega_0 C_1}R1​=R2​=ζω0​C1​1​

      Highpass:

      1. Set μ=1\mu = 1μ=1
      2. Choose R2R_2R2​
      3. R1=4ζ2R2R_1 = 4\zeta^2 R_2R1​=4ζ2R2​
      4. C1=C2=1ζω0R2C_1 = C_2 = \frac{1}{\zeta\omega_0 R_2}C1​=C2​=ζω0​R2​1​

      Equal Elements Method (μ=3−2ζ\mu = 3 - 2\zetaμ=3−2ζ):

      • ONLY for 0<ζ<10 < \zeta < 10<ζ<1
      • Set R1=R2=RR_1 = R_2 = RR1​=R2​=R, C1=C2=CC_1 = C_2 = CC1​=C2​=C
      • R=1ω0CR = \frac{1}{\omega_0 C}R=ω0​C1​
      • μ=3−2ζ\mu = 3 - 2\zetaμ=3−2ζ
      • μ=1+RfRg\mu = 1 + \frac{R_f}{R_g}μ=1+Rg​Rf​​

      Topology Swap: Highpass = swap all R↔CR \leftrightarrow CR↔C from lowpass

      PHASE 5: DESIGN 1ST ORDER SECTIONS

      Passive RC:

      • Lowpass: H(s)=11+sRCH(s) = \frac{1}{1 + sRC}H(s)=1+sRC1​, ωc=1RC\omega_c = \frac{1}{RC}ωc​=RC1​
      • Highpass: H(s)=sRC1+sRCH(s) = \frac{sRC}{1 + sRC}H(s)=1+sRCsRC​, ωc=1RC\omega_c = \frac{1}{RC}ωc​=RC1​

      Active (Non-Inverting):

      • H(s)=K1+sRCH(s) = \frac{K}{1 + sRC}H(s)=1+sRCK​
      • K=1+RfRgK = 1 + \frac{R_f}{R_g}K=1+Rg​Rf​​
      • ωc=1RC\omega_c = \frac{1}{RC}ωc​=RC1​

      PHASE 6: CASCADE & ORDER SECTIONS

      Section Ordering (Critical!):

      1. FIRST: Lowest QQQ (Highest ζ\zetaζ)
      2. MIDDLE: Medium QQQ
      3. LAST: Highest QQQ (Lowest ζ\zetaζ)

      Prevents early clipping from resonant peaks

      Gain Distribution:

      • All in first stage, OR
      • Equal distribution: K1/nK^{1/n}K1/n per stage, OR
      • All in last stage

      Verify: Htotal(s)=H1(s)×H2(s)×⋯×Hn(s)H_{total}(s) = H_1(s) \times H_2(s) \times \cdots \times H_n(s)Htotal​(s)=H1​(s)×H2​(s)×⋯×Hn​(s)

      PHASE 7: BANDPASS & BAND-STOP

      Bandpass (Cascade):

      • HBP(s)=HLP(s)×HHP(s)H_{BP}(s) = H_{LP}(s) \times H_{HP}(s)HBP​(s)=HLP​(s)×HHP​(s)
      • LP with ωc2\omega_{c2}ωc2​ (upper) \times HP with ωc1\omega_{c1}ωc1​ (lower)
      • ω0=ωc1ωc2\omega_0 = \sqrt{\omega_{c1} \omega_{c2}}ω0​=ωc1​ωc2​​
      • BW=ωc2−ωc1BW = \omega_{c2} - \omega_{c1}BW=ωc2​−ωc1​

      Band-Stop (Parallel):

      • HBS(s)=HLP(s)+HHP(s)H_{BS}(s) = H_{LP}(s) + H_{HP}(s)HBS​(s)=HLP​(s)+HHP​(s)
      • LP with ωc1\omega_{c1}ωc1​ + HP with ωc2\omega_{c2}ωc2​
      • Use summing amplifier

      PHASE 8: COMPONENT SCALING

      Frequency Scaling (×kf\times k_f×kf​):

      • Rnew=RoldR_{new} = R_{old}Rnew​=Rold​ (unchanged)
      • Cnew=ColdkfC_{new} = \frac{C_{old}}{k_f}Cnew​=kf​Cold​​
      • ωnew=kfωold\omega_{new} = k_f \omega_{old}ωnew​=kf​ωold​

      Impedance Scaling (×kz\times k_z×kz​):

      • Rnew=kzRoldR_{new} = k_z R_{old}Rnew​=kz​Rold​
      • Cnew=ColdkzC_{new} = \frac{C_{old}}{k_z}Cnew​=kz​Cold​​
      • H(s)H(s)H(s), ω\omegaω unchanged

      PHASE 9: VERIFICATION

      • Cutoff frequency correct?
      • Gain at DC/∞ correct?
      • Order matches calculated nnn?
      • All ζ\zetaζ achievable with chosen method?
      • Components: R (1kΩ-1MΩ), C (100pF-10μF)
      • Op-amp: GBW≥100fcQKGBW \geq 100 f_c Q KGBW≥100fc​QK

      KEY FORMULAS

      Standard 2nd Order:

      • Lowpass: Kω02s2+2ζω0s+ω02\frac{K\omega_0^2}{s^2 + 2\zeta\omega_0 s + \omega_0^2}s2+2ζω0​s+ω02​Kω02​​
      • Highpass: Ks2s2+2ζω0s+ω02\frac{Ks^2}{s^2 + 2\zeta\omega_0 s + \omega_0^2}s2+2ζω0​s+ω02​Ks2​
      • Bandpass: K(2ζω0)ss2+2ζω0s+ω02\frac{K(2\zeta\omega_0)s}{s^2 + 2\zeta\omega_0 s + \omega_0^2}s2+2ζω0​s+ω02​K(2ζω0​)s​

      Relationships:

      • Q=12ζQ = \frac{1}{2\zeta}Q=2ζ1​, ζ=12Q\zeta = \frac{1}{2Q}ζ=2Q1​
      • BW=2ζω0=ω0QBW = 2\zeta\omega_0 = \frac{\omega_0}{Q}BW=2ζω0​=Qω0​​
      • ω0=1RC\omega_0 = \frac{1}{RC}ω0​=RC1​ (equal elements)

      Damping:

      • ζ>1\zeta > 1ζ>1: Overdamped, Unity Gain ONLY
      • ζ=1\zeta = 1ζ=1: Critically Damped, Unity Gain ONLY
      • 0<ζ<10 < \zeta < 10<ζ<1: Underdamped, Either method

      BODE PLOT ESSENTIALS

      Axes:

      • X: log⁡10(ω)\log_{10}(\omega)log10​(ω)
      • Y: 20log⁡10(∣H(jω)∣)20\log_{10}(|H(j\omega)|)20log10​(∣H(jω)∣) dB

      Slopes:

      • 1st order LP: -20 dB/decade
      • 1st order HP: +20 dB/decade
      • 2nd order LP: -40 dB/decade
      • 2nd order HP: +40 dB/decade
      • nth order: ±20n\pm 20n±20n dB/decade

      Corner Frequency (ωc\omega_cωc​):

      • Actual: -3 dB from asymptote
      • Approximation: lines intersect

      Decibel Conversions:

      • 0 dB = 1.0
      • -3 dB = 0.707
      • -6 dB = 0.5
      • -20 dB = 0.1
      • 20 dB = 10

      Plotting Complex H(s)H(s)H(s):

      1. Find all poles/zeros
      2. Order by frequency
      3. Start with DC/HF asymptote
      4. At each break: change slope by \pm20 or \pm40 dB/dec
      5. Poles: decrease slope
      6. Zeros: increase slope

      CRITICAL DESIGN TIPS

      DO:

      • Try Unity Gain first
      • Check ζ\zetaζ before using Equal Elements
      • Order sections by Q (low → high)
      • Use buffers between stages
      • Verify practical component values

      DON’T:

      • Use Equal Elements for ζ≥1\zeta \geq 1ζ≥1
      • Forget to round up nnn
      • Mix up LP/HP frequency ratios
      • Cascade high-Q sections first
      • Use extreme component values

      Quick Checks:

      • Lowpass: H(0)H(0)H(0) = max
      • Highpass: H(∞)H(\infty)H(∞) = max
      • Butterworth at ωc\omega_cωc​: ∣H∣=K/2|H| = K/\sqrt{2}∣H∣=K/2​
      • Unity gain: direct connection
      • Equal elements: need gain resistors (μ>1\mu > 1μ>1)
