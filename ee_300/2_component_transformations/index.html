      Component Transformations: Time Domain to s-Domain

      Introduction

      This document covers the fundamental transformation rules for converting circuit components from the time domain to the s-domain (Laplace domain). These transformations are essential for both PATH A and PATH B.

      The Basic Principle

      In the s-domain, we replace time-domain relationships with frequency-domain (s-domain) impedances.

        graph LR
    A[Time Domain<br/>v_t and i_t relationships] --> B[Laplace Transform]
    B --> C[s-Domain<br/>V_s and I_s with impedances]

      iR(t)=vR(t)Ri_R(t) = \frac{v_R(t)}{R}iR​(t)=RvR​(t)​

      s-Domain Impedance

      ZR(s)=RZ_R(s) = RZR​(s)=R

      • Same value, same units (Ohms)
      • Simplest component to transform

      Capacitor Transformation

      Time Domain Relationships

      iC(t)=CdvC(t)dti_C(t) = C \frac{dv_C(t)}{dt}iC​(t)=CdtdvC​(t)​

      vC(t)=1C∫0tiC(τ)dτ+vC(0)v_C(t) = \frac{1}{C}\int_{0}^{t} i_C(\tau)d\tau + v_C(0)vC​(t)=C1​∫0t​iC​(τ)dτ+vC​(0)

      s-Domain Impedance

      ZC(s)=1sCZ_C(s) = \frac{1}{sC}ZC​(s)=sC1​

      Transformation 

        graph LR
    A[Capacitor C farads] -->|Transform| B[Impedance 1/sC Ω]
    
    style A fill:#ffebee
    style B fill:#e8f5e9

      Key Points:

      • Impedance is inversely proportional to both s and C
      • At low frequencies (s → 0): impedance → ∞ (open circuit)
      • At high frequencies (s → ∞): impedance → 0 (short circuit)
      • Unit: Ohms (Ω)

      s-Domain Ohm’s Law

      VC(s)=1sC⋅IC(s)V_C(s) = \frac{1}{sC} \cdot I_C(s)VC​(s)=sC1​⋅IC​(s)

      IC(s)=sC⋅VC(s)I_C(s) = sC \cdot V_C(s)IC​(s)=sC⋅VC​(s)

      Why 1/sC?

      Taking Laplace transform of the capacitor current equation:

      iC(t)=CdvC(t)dti_C(t) = C \frac{dv_C(t)}{dt}iC​(t)=CdtdvC​(t)​IC(s)=C[sVC(s)−vC(0)]I_C(s) = C[sV_C(s) - v_C(0)]IC​(s)=C[sVC​(s)−vC​(0)]

      Without initial condition (or with it as a separate source):

      IC(s)=sC⋅VC(s)I_C(s) = sC \cdot V_C(s)IC​(s)=sC⋅VC​(s)

      Therefore:

      VC(s)=1sC⋅IC(s)V_C(s) = \frac{1}{sC} \cdot I_C(s)VC​(s)=sC1​⋅IC​(s)

      This is Ohm’s law with impedance ZC=1sCZ_C = \frac{1}{sC}ZC​=sC1​

      Inductor Transformation

      Time Domain Relationships

      vL(t)=LdiL(t)dtv_L(t) = L \frac{di_L(t)}{dt}vL​(t)=LdtdiL​(t)​

      iL(t)=1L∫0tvL(τ)dτ+iL(0)i_L(t) = \frac{1}{L}\int_{0}^{t} v_L(\tau)d\tau + i_L(0)iL​(t)=L1​∫0t​vL​(τ)dτ+iL​(0)

      s-Domain Impedance

      ZL(s)=sLZ_L(s) = sLZL​(s)=sL

      Transformation 

        graph LR
    A[Inductor L henries] -->|Transform| B[Impedance sL Ω]
    
    style A fill:#ffebee
    style B fill:#e8f5e9

      Key Points:

      • Impedance is proportional to both s and L
      • At low frequencies (s → 0): impedance → 0 (short circuit)
      • At high frequencies (s → ∞): impedance → ∞ (open circuit)
      • Unit: Ohms (Ω)

      s-Domain Ohm’s Law

      VL(s)=sL⋅IL(s)V_L(s) = sL \cdot I_L(s)VL​(s)=sL⋅IL​(s)

      IL(s)=VL(s)sLI_L(s) = \frac{V_L(s)}{sL}IL​(s)=sLVL​(s)​

      Why sL?

      Taking Laplace transform of the inductor voltage equation:

      vL(t)=LdiL(t)dtv_L(t) = L \frac{di_L(t)}{dt}vL​(t)=LdtdiL​(t)​VL(s)=L[sIL(s)−iL(0)]V_L(s) = L[sI_L(s) - i_L(0)]VL​(s)=L[sIL​(s)−iL​(0)]

      Without initial condition (or with it as a separate source):

      VL(s)=sL⋅IL(s)V_L(s) = sL \cdot I_L(s)VL​(s)=sL⋅IL​(s)

      This is Ohm’s law with impedance ZL=sLZ_L = sLZL​=sL

      Series and Parallel Impedances

      Series Impedances

      Just like resistors:

      Ztotal=Z1+Z2+Z3+⋯Z_{total} = Z_1 + Z_2 + Z_3 + \cdotsZtotal​=Z1​+Z2​+Z3​+⋯

      Example - Series RLC:

      Ztotal=R+sL+1sCZ_{total} = R + sL + \frac{1}{sC}Ztotal​=R+sL+sC1​

      Parallel Impedances

      Just like resistors:

      1Ztotal=1Z1+1Z2+1Z3+⋯\frac{1}{Z_{total}} = \frac{1}{Z_1} + \frac{1}{Z_2} + \frac{1}{Z_3} + \cdotsZtotal​1​=Z1​1​+Z2​1​+Z3​1​+⋯

      For two impedances:

      Ztotal=Z1Z2Z1+Z2Z_{total} = \frac{Z_1 Z_2}{Z_1 + Z_2}Ztotal​=Z1​+Z2​Z1​Z2​​

      Example - Parallel RLC:

      1Ztotal=1R+1sL+sC\frac{1}{Z_{total}} = \frac{1}{R} + \frac{1}{sL} + sCZtotal​1​=R1​+sL1​+sC

      Transformation Examples

      Example 1: Series RC Circuit

      Time Domain:

      v_s(t) = 5u(t) V ---[R = 10Ω]---[C = 0.1F]--- ground

      s-Domain:

      V_s(s) = 5/s V ---[10Ω]---[10/s Ω]--- ground

      Calculations:

      • Source: vs(t)=5u(t)→Vs(s)=5sv_s(t) = 5u(t) \rightarrow V_s(s) = \frac{5}{s}vs​(t)=5u(t)→Vs​(s)=s5​
      • Resistor: R=10ΩR = 10ΩR=10Ω (unchanged)
      • Capacitor: ZC=1sC=1s⋅0.1=10sZ_C = \frac{1}{sC} = \frac{1}{s \cdot 0.1} = \frac{10}{s}ZC​=sC1​=s⋅0.11​=s10​

      Total impedance:

      Ztotal=R+ZC=10+10s=10s+10sZ_{total} = R + Z_C = 10 + \frac{10}{s} = \frac{10s + 10}{s}Ztotal​=R+ZC​=10+s10​=s10s+10​

      Example 3: Series RLC Circuit

      Time Domain:

      v_s(t) = 10\cos(5t)u(t) V ---[R=2Ω]---[L=0.4H]---[C=0.05F]--- ground

      s-Domain:

      V_s(s) = 10s/(s²+25) V ---[2Ω]---[0.4s Ω]---[20/s Ω]--- ground

      Calculations:

      • Source: vs(t)=10cos⁡(5t)u(t)→Vs(s)=10ss2+25v_s(t) = 10\cos(5t)u(t) \rightarrow V_s(s) = \frac{10s}{s^2+25}vs​(t)=10cos(5t)u(t)→Vs​(s)=s2+2510s​
      • Resistor: R=2ΩR = 2ΩR=2Ω (unchanged)
      • Inductor: ZL=sL=0.4sZ_L = sL = 0.4sZL​=sL=0.4s
      • Capacitor: ZC=1sC=10.05s=20sZ_C = \frac{1}{sC} = \frac{1}{0.05s} = \frac{20}{s}ZC​=sC1​=0.05s1​=s20​

      Total impedance:

      Ztotal=2+0.4s+20s=2s+0.4s2+20s=0.4s2+2s+20sZ_{total} = 2 + 0.4s + \frac{20}{s} = \frac{2s + 0.4s^2 + 20}{s} = \frac{0.4s^2 + 2s + 20}{s}Ztotal​=2+0.4s+s20​=s2s+0.4s2+20​=s0.4s2+2s+20​

      Unit Verification

      All impedances must have units of Ohms (Ω).

      Capacitor Impedance Units

      [ZC]=1[s][C]=1s−1⋅F=1s−1⋅sΩ=Ω[Z_C] = \frac{1}{[s][C]} = \frac{1}{s^{-1} \cdot F} = \frac{1}{s^{-1} \cdot \frac{s}{\Omega}} = \Omega[ZC​]=[s][C]1​=s−1⋅F1​=s−1⋅Ωs​1​=Ω

      [CHECK]

      Inductor Impedance Units

      [ZL]=[s][L]=s−1⋅H=s−1⋅(Ω⋅s)=Ω[Z_L] = [s][L] = s^{-1} \cdot H = s^{-1} \cdot (\Omega \cdot s) = \Omega[ZL​]=[s][L]=s−1⋅H=s−1⋅(Ω⋅s)=Ω

      [CHECK]

      Common Mistakes to Avoid

      [X] WRONG: Capacitor Transformation 

      C = 0.1F → Z = 0.1sC  (NO!)
C = 0.1F → Z = sC     (NO!)
C = 0.1F → Z = C      (NO!)

      [CHECK] CORRECT: Capacitor Transformation 

      C = 0.1F → Z = 1/(sC) = 1/(0.1s) = 10/s

      [X] WRONG: Inductor Transformation 

      L = 2H → Z = L/s    (NO!)
L = 2H → Z = 1/(sL) (NO!)
L = 2H → Z = L      (NO!)

      [CHECK] CORRECT: Inductor Transformation 

      L = 2H → Z = sL = 2s

      [X] WRONG: Forgetting the s Variable 

      Z_total = R + L + 1/C  (NO! Missing s)

      [CHECK] CORRECT: Including s 

      Z_total = R + sL + 1/(sC)
