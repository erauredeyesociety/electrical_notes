      Definitions

      Zero-Input Response (ZIR)

      Response due to initial conditions only, with input set to zero.

      • Caused by: Initial capacitor voltage vC(0−)v_C(0^-)vC​(0−) or inductor current iL(0−)i_L(0^-)iL​(0−)
      • Input sources: Set to zero (voltage → short, current → open)
      • Nature: Pure natural response (exponential decay)
      • In S-domain: Generated by initial condition sources vC(0−)s\frac{v_C(0^-)}{s}svC​(0−)​ or LiL(0−)Li_L(0^-)LiL​(0−)

      Zero-State Response (ZSR)

      Response due to input source only, with initial conditions set to zero.

      • Caused by: External input vs(t)v_s(t)vs​(t) or is(t)i_s(t)is​(t)
      • Initial conditions: Set to zero (vC(0−)=0v_C(0^-) = 0vC​(0−)=0, iL(0−)=0i_L(0^-) = 0iL​(0−)=0)
      • Nature: Natural response + forced response
      • In S-domain: Generated by input sources Vs(s)V_s(s)Vs​(s) or Is(s)I_s(s)Is​(s)

      Step-by-Step Solution Process

      Finding Initial Conditions (for t<0t < 0t<0)

      1. Replace capacitors with open circuits (no current)
      2. Replace inductors with short circuits (no voltage)
      3. Solve DC circuit to find vC(0−)v_C(0^-)vC​(0−) and iL(0−)i_L(0^-)iL​(0−)
      4. Apply continuity: vC(0+)=vC(0−)v_C(0^+) = v_C(0^-)vC​(0+)=vC​(0−) and iL(0+)=iL(0−)i_L(0^+) = i_L(0^-)iL​(0+)=iL​(0−)

      Calculating ZSR (Zero-State Response)

      Given: Circuit with input source and initial conditions

      Steps:

      1. Transform to S-domain: Replace all components with S-domain equivalents
      2. Zero initial conditions: Remove (short/open) all initial condition sources
        • Capacitor: Remove vC(0−)s\frac{v_C(0^-)}{s}svC​(0−)​ voltage source (or CvC(0−)Cv_C(0^-)CvC​(0−) current source)
        • Inductor: Remove LiL(0−)Li_L(0^-)LiL​(0−) voltage source (or iL(0−)s\frac{i_L(0^-)}{s}siL​(0−)​ current source)
      3. Keep input active: Leave Vs(s)V_s(s)Vs​(s) or Is(s)I_s(s)Is​(s) in circuit
      4. Analyze circuit: Use KVL/KCL/nodal/mesh to find VZSR(s)V_{\text{ZSR}}(s)VZSR​(s) or IZSR(s)I_{\text{ZSR}}(s)IZSR​(s)
      5. Apply PFE: Factor denominator and find residues
      6. Inverse transform: Get vZSR(t)v_{\text{ZSR}}(t)vZSR​(t) or iZSR(t)i_{\text{ZSR}}(t)iZSR​(t)

      Calculating ZIR (Zero-Input Response)

      Given: Circuit with input source and initial conditions

      Steps:

      1. Transform to S-domain: Replace all components with S-domain equivalents
      2. Zero input sources: Short voltage sources, open current sources
      3. Keep initial conditions: Leave initial condition sources active
        • Capacitor: Keep vC(0−)s\frac{v_C(0^-)}{s}svC​(0−)​ voltage source (or CvC(0−)Cv_C(0^-)CvC​(0−) current source)
        • Inductor: Keep LiL(0−)Li_L(0^-)LiL​(0−) voltage source (or iL(0−)s\frac{i_L(0^-)}{s}siL​(0−)​ current source)
      4. Analyze circuit: Use KVL/KCL/nodal/mesh to find VZIR(s)V_{\text{ZIR}}(s)VZIR​(s) or IZIR(s)I_{\text{ZIR}}(s)IZIR​(s)
      5. Apply PFE: Factor denominator and find residues
      6. Inverse transform: Get vZIR(t)v_{\text{ZIR}}(t)vZIR​(t) or iZIR(t)i_{\text{ZIR}}(t)iZIR​(t)

      Calculating Total Response

      Steps:

      2. Verify initial value: Check vtotal(0+)v_{\text{total}}(0^+)vtotal​(0+) matches initial condition

      Common Mistakes

      1. ❌ Forgetting to zero initial conditions when calculating ZSR
      2. ❌ Forgetting to zero input when calculating ZIR
      3. ❌ Wrong polarity on inductor initial voltage source (should be −LiL(0−)-Li_L(0^-)−LiL​(0−))
      4. ❌ Not verifying total response matches initial and final values
      5. ❌ Confusing natural response with ZIR (ZSR also has natural component!)
