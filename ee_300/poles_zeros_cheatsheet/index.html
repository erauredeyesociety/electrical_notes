      Poles & Zeros Cheat Sheet

      1. Poles of circuit are rates of exponentials that are “always” in the output
      2. Poles determined by output side of differential equation
      3. Zeros of circuit are rates of exponentials that are “never” in the output
      4. Zeros determined by input side of differential equation

      1. Circuit Response Components

      Total Response

      Total Response=ZIR+ZSR\text{Total Response} = \text{ZIR} + \text{ZSR}Total Response=ZIR+ZSR

      Zero-Input Response (ZIR)

      • Response due to initial conditions only (stored energy)
      • No external sources applied
      • Composed entirely of natural response

      Zero-State Response (ZSR)

      • Response due to external inputs only
      • All initial conditions = 0
      • Composed of natural response + forced response

      Element Impedances

      Initial Condition Sources

      • Capacitor: Voltage source Vc(0)s\frac{V_c(0)}{s}sVc​(0)​ in series with 1sC\frac{1}{sC}sC1​
      • Inductor: Current source IL(0)s\frac{I_L(0)}{s}sIL​(0)​ in parallel with sLsLsL

      For Network Functions: Set all initial conditions to ZERO

      3. Network Functions (Transfer Functions)

      Definition

      H(s)=Output(s)Input(s)[with all ICs = 0]H(s) = \frac{\text{Output}(s)}{\text{Input}(s)} \quad \text{[with all ICs = 0]}H(s)=Input(s)Output(s)​[with all ICs = 0]

      Types

      1. Transfer Function: H(s)=Vout(s)Vin(s)H(s) = \frac{V_{out}(s)}{V_{in}(s)}H(s)=Vin​(s)Vout​(s)​ (different ports)
      2. Driving-Point Impedance: Z(s)=V(s)I(s)Z(s) = \frac{V(s)}{I(s)}Z(s)=I(s)V(s)​ (same port)
      3. Driving-Point Admittance: Y(s)=I(s)V(s)=1Z(s)Y(s) = \frac{I(s)}{V(s)} = \frac{1}{Z(s)}Y(s)=V(s)I(s)​=Z(s)1​ (same port)

      Finding H(s) from Circuit - 6 Steps

      1. Transform all elements to s-domain (RRR, 1sC\frac{1}{sC}sC1​, sLsLsL)
      2. Zero all initial conditions (voltage sources → short, current sources → open)
      3. Define input and output variables
      4. Apply circuit analysis (voltage/current divider, nodal, mesh, KCL/KVL)
      5. Form H(s)=Output(s)Input(s)H(s) = \frac{\text{Output}(s)}{\text{Input}(s)}H(s)=Input(s)Output(s)​
      6. Simplify to rational function: H(s)=N(s)D(s)H(s) = \frac{N(s)}{D(s)}H(s)=D(s)N(s)​

      4. Poles and Zeros

      Rational Function Form

      H(s)=N(s)D(s)=amsm+am−1sm−1+⋯+a0bnsn+bn−1sn−1+⋯+b0H(s) = \frac{N(s)}{D(s)} = \frac{a_m s^m + a_{m-1}s^{m-1} + \cdots + a_0}{b_n s^n + b_{n-1}s^{n-1} + \cdots + b_0}H(s)=D(s)N(s)​=bn​sn+bn−1​sn−1+⋯+b0​am​sm+am−1​sm−1+⋯+a0​​

      Definitions

      ZEROS: Roots of numerator

      N(s)=0  ⟹  Zeros: s=z1,z2,…,zmN(s) = 0 \implies \text{Zeros: } s = z_1, z_2, \ldots, z_mN(s)=0⟹Zeros: s=z1​,z2​,…,zm​

      POLES: Roots of denominator (characteristic equation)

      D(s)=0  ⟹  Poles: s=p1,p2,…,pnD(s) = 0 \implies \text{Poles: } s = p_1, p_2, \ldots, p_nD(s)=0⟹Poles: s=p1​,p2​,…,pn​

      Finding Poles and Zeros - 3 Steps

      1. Express H(s)=N(s)D(s)H(s) = \frac{N(s)}{D(s)}H(s)=D(s)N(s)​ in polynomial form
      2. Find Zeros: Set N(s)=0N(s) = 0N(s)=0 and solve for sss
      3. Find Poles: Set D(s)=0D(s) = 0D(s)=0 and solve for sss

      Solving Polynomials

      • 1st order (as+bas + bas+b): s=−bas = -\frac{b}{a}s=−ab​
      • 2nd order (as2+bs+cas^2 + bs + cas2+bs+c): s=−b±b2−4ac2as = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}s=2a−b±b2−4ac​​
      • Higher order: Factor or use numerical methods

      6. Natural vs Forced Response

      From Output Expression

      Vout(s)=H(s)⋅Vin(s)V_{out}(s) = H(s) \cdot V_{in}(s)Vout​(s)=H(s)⋅Vin​(s)

      After factoring denominator, all poles come from:

      1. Poles of H(s)H(s)H(s) → Natural Response
      2. Poles of Vin(s)V_{in}(s)Vin​(s) → Forced Response

      Natural Response

      • Comes from network function poles (circuit’s own modes)
      • Independent of input type
      • Determined by R, L, C values
      • Same modes appear in ZIR

      Forced Response

      • Comes from input signal poles
      • Has same “shape” as input
      • Amplitude scaled by H(s)H(s)H(s) evaluated at input frequency
      • If input is Kes0tu(t)Ke^{s_0 t}u(t)Kes0​tu(t), forced term is K⋅H(s0)⋅es0tK \cdot H(s_0) \cdot e^{s_0 t}K⋅H(s0​)⋅es0​t

      Special Case: Resonance

      When input frequency = pole of H(s)H(s)H(s):

      Output term becomes: t⋅es0t (grows with time)\text{Output term becomes: } t \cdot e^{s_0 t} \text{ (grows with time)}Output term becomes: t⋅es0​t (grows with time)

      7. Time Response via Inverse Laplace Transform

      Complete Process - 6 Steps

      1. Find Output: Vout(s)=H(s)⋅Vin(s)V_{out}(s) = H(s) \cdot V_{in}(s)Vout​(s)=H(s)⋅Vin​(s)

      2. Factor Denominator: Find ALL poles (from H(s)H(s)H(s) and Vin(s)V_{in}(s)Vin​(s))

      3. Partial Fraction Expansion (PFE):

        Vout(s)=k0s−p0+k1s−p1+k2s−p2+⋯V_{out}(s) = \frac{k_0}{s - p_0} + \frac{k_1}{s - p_1} + \frac{k_2}{s - p_2} + \cdotsVout​(s)=s−p0​k0​​+s−p1​k1​​+s−p2​k2​​+⋯

      6. Combine and Identify:

        • Terms from H(s)H(s)H(s) poles = Natural response
        • Terms from Vin(s)V_{in}(s)Vin​(s) poles = Forced response

      9. Differential Equations

      From H(s) to ODE - 4 Steps

      1. Start with: H(s)=Vout(s)Vin(s)=N(s)D(s)H(s) = \frac{V_{out}(s)}{V_{in}(s)} = \frac{N(s)}{D(s)}H(s)=Vin​(s)Vout​(s)​=D(s)N(s)​

      2. Cross-multiply: Vout(s)⋅D(s)=Vin(s)⋅N(s)V_{out}(s) \cdot D(s) = V_{in}(s) \cdot N(s)Vout​(s)⋅D(s)=Vin​(s)⋅N(s)

      3. Replace sss with ddt\frac{d}{dt}dtd​:

        • s→ddts \to \frac{d}{dt}s→dtd​
        • s2→d2dt2s^2 \to \frac{d^2}{dt^2}s2→dt2d2​
        • sn→dndtns^n \to \frac{d^n}{dt^n}sn→dtndn​

      4. Write ODE:

        (bndndtn+bn−1dn−1dtn−1+⋯+b0)vout(t)=(amdmdtm+⋯+a0)vin(t)\left(b_n \frac{d^n}{dt^n} + b_{n-1}\frac{d^{n-1}}{dt^{n-1}} + \cdots + b_0\right) v_{out}(t) = \left(a_m \frac{d^m}{dt^m} + \cdots + a_0\right) v_{in}(t)(bn​dtndn​+bn−1​dtn−1dn−1​+⋯+b0​)vout​(t)=(am​dtmdm​+⋯+a0​)vin​(t)

      Key Insight

      • Denominator coefficients (poles) → Output side (left)
      • Numerator coefficients (zeros) → Input side (right)

      Example

      H(s)=2s+3s2+5s+6H(s) = \frac{2s + 3}{s^2 + 5s + 6}H(s)=s2+5s+62s+3​

      Cross-multiply: Vout(s)(s2+5s+6)=Vin(s)(2s+3)V_{out}(s)(s^2 + 5s + 6) = V_{in}(s)(2s + 3)Vout​(s)(s2+5s+6)=Vin​(s)(2s+3)

      ODE:

      d2voutdt2+5dvoutdt+6vout=2dvindt+3vin\frac{d^2v_{out}}{dt^2} + 5\frac{dv_{out}}{dt} + 6v_{out} = 2\frac{dv_{in}}{dt} + 3v_{in}dt2d2vout​​+5dtdvout​​+6vout​=2dtdvin​​+3vin​

      10. Quick Problem-Solving Guide

      Given Circuit → Find H(s)

      1. Transform to s-domain: RRR, 1sC\frac{1}{sC}sC1​, sLsLsL
      2. Zero all ICs (critical!)
      3. Use voltage/current divider, nodal, or mesh analysis
      4. Form H(s)=OutputInputH(s) = \frac{\text{Output}}{\text{Input}}H(s)=InputOutput​

      Given H(s) → Find Poles/Zeros

      1. Express as H(s)=N(s)D(s)H(s) = \frac{N(s)}{D(s)}H(s)=D(s)N(s)​
      2. Zeros: N(s)=0N(s) = 0N(s)=0
      3. Poles: D(s)=0D(s) = 0D(s)=0

      Given H(s) → Find Differential Equation

      1. Cross-multiply: Output(s)⋅D(s)=Input(s)⋅N(s)\text{Output}(s) \cdot D(s) = \text{Input}(s) \cdot N(s)Output(s)⋅D(s)=Input(s)⋅N(s)
      2. Replace s→ddts \to \frac{d}{dt}s→dtd​
      3. Write ODE

      Given Circuit + Input → Find Time Response

      1. Find H(s)H(s)H(s) from circuit
      2. Calculate Vout(s)=H(s)⋅Vin(s)V_{out}(s) = H(s) \cdot V_{in}(s)Vout​(s)=H(s)⋅Vin​(s)
      3. Do partial fraction expansion
      4. Use cover-up method for residues
      5. Inverse Laplace transform each term
      6. Add u(t)u(t)u(t) to each term

      11. Critical Reminders

      Always Remember

      ✓ Zero all initial conditions when finding H(s)H(s)H(s) ✓ H(s)H(s)H(s) only describes ZSR (not ZIR) ✓ Capacitor: 1sC\frac{1}{sC}sC1​ NOT sCsCsC ✓ Inductor: sLsLsL NOT 1sL\frac{1}{sL}sL1​ ✓ Include u(t)u(t)u(t) in time-domain responses ✓ Complex poles come in conjugate pairs

      Common Mistakes

      ✗ Forgetting to zero ICs for H(s)H(s)H(s) ✗ Confusing poles (denominator) with zeros (numerator) ✗ Wrong impedance transformations ✗ Not factoring denominator completely ✗ Forgetting u(t)u(t)u(t) in inverse Laplace ✗ Mixing up natural response (from H(s)H(s)H(s) poles) vs forced response (from input poles)

      13. Example: RC Low-Pass Filter

      Given: Vin→R→C→VoutV_{in} \to R \to C \to V_{out}Vin​→R→C→Vout​ (across capacitor)

      Find H(s)

      H(s)=ZCZR+ZC=1/(sC)R+1/(sC)=11+sRC=1/RCs+1/RCH(s) = \frac{Z_C}{Z_R + Z_C} = \frac{1/(sC)}{R + 1/(sC)} = \frac{1}{1 + sRC} = \frac{1/RC}{s + 1/RC}H(s)=ZR​+ZC​ZC​​=R+1/(sC)1/(sC)​=1+sRC1​=s+1/RC1/RC​

      Poles and Zeros

      • Pole: s+1RC=0  ⟹  s=−1RCs + \frac{1}{RC} = 0 \implies s = -\frac{1}{RC}s+RC1​=0⟹s=−RC1​
      • Zeros: None (numerator is constant)

      Differential Equation

      Cross-multiply: Vout(s)(s+1RC)=Vin(s)⋅1RCV_{out}(s)(s + \frac{1}{RC}) = V_{in}(s) \cdot \frac{1}{RC}Vout​(s)(s+RC1​)=Vin​(s)⋅RC1​

      dvoutdt+voutRC=vinRC\frac{dv_{out}}{dt} + \frac{v_{out}}{RC} = \frac{v_{in}}{RC}dtdvout​​+RCvout​​=RCvin​​

      Time Response to Step Input (vin=u(t)v_{in} = u(t)vin​=u(t))

      Vin(s)=1s,Vout(s)=1/RC(s+1/RC)⋅sV_{in}(s) = \frac{1}{s}, \quad V_{out}(s) = \frac{1/RC}{(s + 1/RC) \cdot s}Vin​(s)=s1​,Vout​(s)=(s+1/RC)⋅s1/RC​

      PFE: Vout(s)=1s−1s+1/RCV_{out}(s) = \frac{1}{s} - \frac{1}{s + 1/RC}Vout​(s)=s1​−s+1/RC1​

      vout(t)=[1−e−t/RC]u(t)v_{out}(t) = \left[1 - e^{-t/RC}\right]u(t)vout​(t)=[1−e−t/RC]u(t)
      • Forced response: 111 (from input pole at s=0s = 0s=0)
      • Natural response: −e−t/RC-e^{-t/RC}−e−t/RC (from H(s)H(s)H(s) pole at s=−1/RCs = -1/RCs=−1/RC)
