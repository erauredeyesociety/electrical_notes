      Initial Conditions in s-Domain Analysis

      Introduction

      Initial conditions represent the stored energy in capacitors and inductors at time t=0t = 0t=0. This document explains how to properly include initial conditions in s-domain circuit analysis.

      Why Initial Conditions Matter

      Energy storage elements (capacitors and inductors) have “memory”:

      • Capacitors store energy in electric fields → voltage across capacitor at t=0t=0t=0
      • Inductors store energy in magnetic fields → current through inductor at t=0t=0t=0

      These initial states affect the circuit response and must be included for accurate analysis.

      Capacitor Initial Conditions

      Time Domain Representation

      At t=0−t = 0^-t=0−, the capacitor has voltage vC(0)v_C(0)vC​(0) stored.

      s-Domain Representation

      Add a voltage source vC(0)s\frac{v_C(0)}{s}svC​(0)​ in series with the capacitor impedance 1sC\frac{1}{sC}sC1​.

        graph LR
    A[Capacitor C<br/>with v_C0] -->|Transform| B[1/sC in series with<br/>voltage source v_C0/s]
    
    style A fill:#ffebee
    style B fill:#e8f5e9

      Circuit Representation

      Without initial condition:

      ---[ 1/sC ]---

      With initial condition vC(0)v_C(0)vC​(0):

      ---[ 1/sC ]---[+]---
            [v_C(0)/s]
              [-]

      The polarity: positive terminal is where the capacitor was positively charged.

      Derivation

      From the capacitor current equation:

      iC(t)=CdvC(t)dti_C(t) = C \frac{dv_C(t)}{dt}iC​(t)=CdtdvC​(t)​

      Taking Laplace transform:

      IC(s)=C[sVC(s)−vC(0)]I_C(s) = C[sV_C(s) - v_C(0)]IC​(s)=C[sVC​(s)−vC​(0)]

      Rearranging:

      IC(s)=sC⋅VC(s)−C⋅vC(0)I_C(s) = sC \cdot V_C(s) - C \cdot v_C(0)IC​(s)=sC⋅VC​(s)−C⋅vC​(0)

      Using Ohm’s law form:

      VC(s)=IC(s)sC+vC(0)sV_C(s) = \frac{I_C(s)}{sC} + \frac{v_C(0)}{s}VC​(s)=sCIC​(s)​+svC​(0)​

      This shows: impedance 1sC\frac{1}{sC}sC1​ with voltage source vC(0)s\frac{v_C(0)}{s}svC​(0)​ in series.

      Example: Capacitor with Initial Voltage

      Given: C=0.1FC = 0.1FC=0.1F, vC(0)=5Vv_C(0) = 5VvC​(0)=5V

      s-Domain representation:

      ---[ 10/s Ω ]---[+]---
              [5/s V]
              [-]

      Where:

      • ZC=1sC=10.1s=10sZ_C = \frac{1}{sC} = \frac{1}{0.1s} = \frac{10}{s}ZC​=sC1​=0.1s1​=s10​
      • Initial condition source = vC(0)s=5s\frac{v_C(0)}{s} = \frac{5}{s}svC​(0)​=s5​

      Inductor Initial Conditions

      Time Domain Representation

      At t=0−t = 0^-t=0−, the inductor has current iL(0)i_L(0)iL​(0) flowing through it.

      s-Domain Representation (Two Options)

      Option A - Series Voltage Source (for KVL/Mesh): Add voltage source LiL(0)Li_L(0)LiL​(0) in series with impedance sLsLsL

      Option B - Parallel Current Source (for KCL/Node): Add current source iL(0)s\frac{i_L(0)}{s}siL​(0)​ in parallel with impedance sLsLsL

      Circuit Representation - Series Form

      Without initial condition:

      ---[ sL ]---

      With initial condition iL(0)i_L(0)iL​(0) (series form):

      ---[ sL ]---[+]---
          [Li_L(0)]
            [-]

      Polarity: opposes the change in current (back-EMF)

      Circuit Representation - Parallel Form

      With initial condition iL(0)i_L(0)iL​(0) (parallel form):

            [ sL ]
---+----||----+---
   |          |
   [i_L(0)/s]→
   |          |

      The current source flows in the direction of initial current.

      Derivation - Series Form

      From the inductor voltage equation:

      vL(t)=LdiL(t)dtv_L(t) = L \frac{di_L(t)}{dt}vL​(t)=LdtdiL​(t)​

      Taking Laplace transform:

      VL(s)=L[sIL(s)−iL(0)]V_L(s) = L[sI_L(s) - i_L(0)]VL​(s)=L[sIL​(s)−iL​(0)]

      Rearranging:

      VL(s)=sL⋅IL(s)−L⋅iL(0)V_L(s) = sL \cdot I_L(s) - L \cdot i_L(0)VL​(s)=sL⋅IL​(s)−L⋅iL​(0)

      This shows: impedance sLsLsL with voltage source LiL(0)Li_L(0)LiL​(0) in series.

      Derivation - Parallel Form

      Using the integral form:

      iL(t)=1L∫0tvL(τ)dτ+iL(0)i_L(t) = \frac{1}{L}\int_{0}^{t} v_L(\tau)d\tau + i_L(0)iL​(t)=L1​∫0t​vL​(τ)dτ+iL​(0)

      Taking Laplace transform:

      IL(s)=VL(s)sL+iL(0)sI_L(s) = \frac{V_L(s)}{sL} + \frac{i_L(0)}{s}IL​(s)=sLVL​(s)​+siL​(0)​

      This shows: impedance sLsLsL with current source iL(0)s\frac{i_L(0)}{s}siL​(0)​ in parallel.

      Equivalence of Two Forms

      The series and parallel forms are equivalent by source transformation:

      LiL(0) in series with sL≡iL(0)s in parallel with sLLi_L(0) \text{ in series with } sL \equiv \frac{i_L(0)}{s} \text{ in parallel with } sLLiL​(0) in series with sL≡siL​(0)​ in parallel with sL

      Verification:

      LiL(0)sL=iL(0)s\frac{Li_L(0)}{sL} = \frac{i_L(0)}{s}sLLiL​(0)​=siL​(0)​

      Example: Inductor with Initial Current

      Given: L=2HL = 2HL=2H, iL(0)=3Ai_L(0) = 3AiL​(0)=3A

      Series form (for KVL):

      ---[ 2s Ω ]---[+]---
            [6 V]
            [-]

      Where:

      • ZL=sL=2sZ_L = sL = 2sZL​=sL=2s
      • Initial condition source = LiL(0)=2×3=6VLi_L(0) = 2 \times 3 = 6VLiL​(0)=2×3=6V

      Parallel form (for KCL):

          [ 2s Ω ]
---+----||----+---
   |          |
   [3/s A]→
   |          |

      Where:

      • ZL=sL=2sZ_L = sL = 2sZL​=sL=2s
      • Initial condition source = iL(0)s=3s\frac{i_L(0)}{s} = \frac{3}{s}siL​(0)​=s3​

      General Rule:

      • KVL/Mesh: Use series voltage sources for both components
      • KCL/Node: Use series voltage for capacitor, parallel current for inductor

      Polarity Conventions

      Capacitor Initial Condition Polarity

      The voltage source vC(0)s\frac{v_C(0)}{s}svC​(0)​ has polarity matching the stored voltage:

      • If capacitor top plate is positive → voltage source + on top
      • The source represents the voltage the capacitor starts with

      Inductor Initial Condition Polarity

      Series form: The voltage source LiL(0)Li_L(0)LiL​(0) opposes current change

      • Acts like back-EMF
      • Polarity opposes the direction you’d expect from applied voltage

      Parallel form: The current source iL(0)s\frac{i_L(0)}{s}siL​(0)​ flows in the same direction as initial current

      • Represents the current the inductor wants to maintain

      Complete Transformation Examples

      Example 1: Series RC with Initial Conditions

      Given:

      • R=50ΩR = 50ΩR=50Ω, C=20μFC = 20μFC=20μF
      • vC(0)=10Vv_C(0) = 10VvC​(0)=10V
      • vs(t)=100u(t)Vv_s(t) = 100u(t)Vvs​(t)=100u(t)V

      s-Domain circuit:

      V_s(s)=100/s ---[50Ω]---[50000/s Ω]---[+]--- ground
                                    [10/s V]
                                    [-]

      Calculations:

      • ZC=1sC=1s⋅20×10−6=50000sZ_C = \frac{1}{sC} = \frac{1}{s \cdot 20 \times 10^{-6}} = \frac{50000}{s}ZC​=sC1​=s⋅20×10−61​=s50000​
      • Initial condition: vC(0)s=10s\frac{v_C(0)}{s} = \frac{10}{s}svC​(0)​=s10​

      Example 2: Series RL with Initial Conditions (KVL)

      Given:

      • R=10ΩR = 10ΩR=10Ω, L=0.5HL = 0.5HL=0.5H
      • iL(0)=2Ai_L(0) = 2AiL​(0)=2A
      • vs(t)=20e−tu(t)Vv_s(t) = 20e^{-t}u(t)Vvs​(t)=20e−tu(t)V

      s-Domain circuit:

      V_s(s)=20/(s+1) ---[10Ω]---[0.5s Ω]---[+]--- ground
                                    [1 V]
                                    [-]

      Calculations:

      • ZL=sL=0.5sZ_L = sL = 0.5sZL​=sL=0.5s
      • Initial condition: LiL(0)=0.5×2=1VLi_L(0) = 0.5 \times 2 = 1VLiL​(0)=0.5×2=1V

      Example 3: Parallel RLC with Initial Conditions (KCL)

      Given:

      • R=5ΩR = 5ΩR=5Ω, L=1HL = 1HL=1H, C=0.1FC = 0.1FC=0.1F
      • vC(0)=0Vv_C(0) = 0VvC​(0)=0V, iL(0)=3Ai_L(0) = 3AiL​(0)=3A
      • is(t)=4u(t)Ai_s(t) = 4u(t)Ais​(t)=4u(t)A

      s-Domain circuit:

               I_s(s) = 4/s
              |

      Since vC(0)=0v_C(0) = 0vC​(0)=0, the capacitor has no initial condition source.

      The inductor uses parallel form for KCL: iL(0)s=3s\frac{i_L(0)}{s} = \frac{3}{s}siL​(0)​=s3​

      Adding Initial Conditions to Equations

      For KVL (Series Circuits)

      Basic equation without initial conditions:

      Vs(s)=I(s)[Z1+Z2+⋯ ]V_s(s) = I(s)[Z_1 + Z_2 + \cdots]Vs​(s)=I(s)[Z1​+Z2​+⋯]

      With initial conditions:

      Vs(s)=I(s)[Z1+Z2+⋯ ]+Vinitial,C(s)+Vinitial,L(s)V_s(s) = I(s)[Z_1 + Z_2 + \cdots] + V_{initial,C}(s) + V_{initial,L}(s)Vs​(s)=I(s)[Z1​+Z2​+⋯]+Vinitial,C​(s)+Vinitial,L​(s)

      Where:

      • Vinitial,C(s)=vC(0)sV_{initial,C}(s) = \frac{v_C(0)}{s}Vinitial,C​(s)=svC​(0)​
      • Vinitial,L(s)=LiL(0)V_{initial,L}(s) = Li_L(0)Vinitial,L​(s)=LiL​(0)

      For KCL (Node Analysis)

      Basic equation without initial conditions:

      Is(s)=V(s)Z1+V(s)Z2+⋯I_s(s) = \frac{V(s)}{Z_1} + \frac{V(s)}{Z_2} + \cdotsIs​(s)=Z1​V(s)​+Z2​V(s)​+⋯

      With initial conditions (capacitor):

      Is(s)=V(s)−vC(0)sZC+V(s)Z1+⋯I_s(s) = \frac{V(s) - \frac{v_C(0)}{s}}{Z_C} + \frac{V(s)}{Z_1} + \cdotsIs​(s)=ZC​V(s)−svC​(0)​​+Z1​V(s)​+⋯

      With initial conditions (inductor, parallel form):

      Is(s)+Iinitial,L(s)=V(s)ZL+V(s)Z1+⋯I_s(s) + I_{initial,L}(s) = \frac{V(s)}{Z_L} + \frac{V(s)}{Z_1} + \cdotsIs​(s)+Iinitial,L​(s)=ZL​V(s)​+Z1​V(s)​+⋯

      Where Iinitial,L(s)=iL(0)sI_{initial,L}(s) = \frac{i_L(0)}{s}Iinitial,L​(s)=siL​(0)​

      Zero Initial Conditions

      If initial conditions are not specified, you typically assume:

      • vC(0)=0v_C(0) = 0vC​(0)=0 (capacitor uncharged)
      • iL(0)=0i_L(0) = 0iL​(0)=0 (inductor unenergized)

      This means the circuit was de-energized before t=0t=0t=0.

      Exception: If the problem describes steady-state operation before t=0t=0t=0, you must solve for initial conditions from the t<0t < 0t<0 circuit.
