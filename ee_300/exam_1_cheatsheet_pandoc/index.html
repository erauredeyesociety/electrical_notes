      Laplace Transform Fundamentals

      Definition

      The Laplace Transform converts time-domain functions f(t)f(t)f(t) into s-domain functions F(s)F(s)F(s):

      F(s)=L{f(t)}=∫0∞f(t)e−stdtF(s) = \mathcal{L}\{f(t)\} = \int_0^\infty f(t)e^{-st} dtF(s)=L{f(t)}=∫0∞​f(t)e−stdt

      where s=σ+jωs = \sigma + j\omegas=σ+jω is a complex variable.

      Why Use Laplace Transforms?

      • Converts differential equations → algebraic equations
      • Handles initial conditions automatically
      • Simplifies circuit analysis dramatically

      Poles and Zeros

      • Poles: Values of sss where F(s)→∞F(s) \to \inftyF(s)→∞ (denominator = 0)
      • Zeros: Values of sss where F(s)=0F(s) = 0F(s)=0 (numerator = 0)
      • Poles determine the nature of the time response (exponential decay, oscillation, etc.)

      Six Essential Properties

      1. Linearity

      L{af(t)+bg(t)}=aF(s)+bG(s)\mathcal{L}\{af(t) + bg(t)\} = aF(s) + bG(s)L{af(t)+bg(t)}=aF(s)+bG(s)

      Use this to: Break complex functions into simpler parts and transform each separately.

      2. Frequency Shift (Multiply by e−ate^{-at}e−at)

      L{e−atf(t)}=F(s+a)\mathcal{L}\{e^{-at}f(t)\} = F(s+a)L{e−atf(t)}=F(s+a)

      Use this to: Handle exponential envelopes. Replace sss with s+as+as+a in F(s)F(s)F(s).

      Example:

      L{e−2tcos⁡(3t)}=s+2(s+2)2+9\mathcal{L}\{e^{-2t}\cos(3t)\} = \frac{s+2}{(s+2)^2+9}L{e−2tcos(3t)}=(s+2)2+9s+2​

      3. Time Shift (Delay Property)

      L{f(t−T)u(t−T)}=e−sTF(s)\mathcal{L}\{f(t-T)u(t-T)\} = e^{-sT}F(s)L{f(t−T)u(t−T)}=e−sTF(s)

      Use this to: Handle delayed signals. The term e−sTe^{-sT}e−sT indicates a delay of TTT seconds.

      4. Differentiation Property CRITICAL FOR CIRCUITS

      L{dfdt}=sF(s)−f(0−)\mathcal{L}\left\{\frac{df}{dt}\right\} = sF(s) - f(0^-)L{dtdf​}=sF(s)−f(0−)L{d2fdt2}=s2F(s)−sf(0−)−f′(0−)\mathcal{L}\left\{\frac{d^2f}{dt^2}\right\} = s^2F(s) - sf(0^-) - f'(0^-)L{dt2d2f​}=s2F(s)−sf(0−)−f′(0−)

      Use this to: Transform differential equations. Initial conditions appear as source terms!

      5. Integration Property

      L{∫0tf(τ)dτ}=1sF(s)\mathcal{L}\left\{\int_0^t f(\tau)d\tau\right\} = \frac{1}{s}F(s)L{∫0t​f(τ)dτ}=s1​F(s)

      Use this to: Handle integrals in your equations (less common in basic circuit analysis).

      6. Multiply by ttt Property

      L{tf(t)}=−dF(s)ds\mathcal{L}\{tf(t)\} = -\frac{dF(s)}{ds}L{tf(t)}=−dsdF(s)​

      Use this to: Handle terms multiplied by ttt, or deal with multiple-order poles in inverse transforms.

      Inverse Laplace Transform

      Goal

      Convert F(s)F(s)F(s) back to f(t)f(t)f(t) using Partial Fraction Expansion (PFE).

      Terminology

      • PRF (Proper Rational Function): Order of numerator < order of denominator
      • IRF (Improper Rational Function): Order of numerator \geq order of denominator

      General Form After PFE

      F(s)=k1s−p1+k2s−p2+⋯+kns−pnF(s) = \frac{k_1}{s-p_1} + \frac{k_2}{s-p_2} + \cdots + \frac{k_n}{s-p_n}F(s)=s−p1​k1​​+s−p2​k2​​+⋯+s−pn​kn​​

      where:

      • pip_ipi​ = pole locations (determine exponential rates)
      • kik_iki​ = residues (determine weights/amplitudes)

      Case 1: Distinct Real Poles

      PFE Form:

      F(s)=k1s−p1+k2s−p2+⋯+kns−pnF(s) = \frac{k_1}{s-p_1} + \frac{k_2}{s-p_2} + \cdots + \frac{k_n}{s-p_n}F(s)=s−p1​k1​​+s−p2​k2​​+⋯+s−pn​kn​​

      Time Domain:

      f(t)=k1ep1tu(t)+k2ep2tu(t)+⋯+knepntu(t)f(t) = k_1e^{p_1t}u(t) + k_2e^{p_2t}u(t) + \cdots + k_ne^{p_nt}u(t)f(t)=k1​ep1​tu(t)+k2​ep2​tu(t)+⋯+kn​epn​tu(t)

      Finding Residues: Cover-Up Method

      The cover-up method is a quick technique for finding residues in partial fraction expansion.

      Basic Principle: To find residue kik_iki​ for pole at s=pis = p_is=pi​:

      ki=(s−pi)F(s)∣s=pik_i = \left.(s-p_i)F(s)\right|_{s=p_i}ki​=(s−pi​)F(s)∣s=pi​​

      Step-by-Step Procedure:

      1. Multiply both sides of F(s)F(s)F(s) by the factor (s−pi)(s-p_i)(s−pi​) corresponding to the pole you’re solving for
      2. “Cover up” (mentally or physically) the factor (s−pi)(s-p_i)(s−pi​) in the original denominator
      3. Substitute s=pis = p_is=pi​ into what remains
      4. Evaluate the expression to get kik_iki​

      Why It Works: Multiplying by (s−pi)(s-p_i)(s−pi​) cancels that factor in the denominator. When we set s=pis = p_is=pi​, all other fractions in the PFE become zero (they have (s−pi)(s-p_i)(s−pi​) in their numerators), leaving only kik_iki​.

      Detailed Example:

      F(s)=3s+5(s+1)(s+2)F(s) = \frac{3s+5}{(s+1)(s+2)}F(s)=(s+1)(s+2)3s+5​

      Set up PFE:

      3s+5(s+1)(s+2)=k1s+1+k2s+2\frac{3s+5}{(s+1)(s+2)} = \frac{k_1}{s+1} + \frac{k_2}{s+2}(s+1)(s+2)3s+5​=s+1k1​​+s+2k2​​

      Find k1k_1k1​ (pole at s=−1s=-1s=−1):

      1. Multiply by (s+1)(s+1)(s+1):

        k1+k2(s+1)s+2=3s+5s+2k_1 + \frac{k_2(s+1)}{s+2} = \frac{3s+5}{s+2}k1​+s+2k2​(s+1)​=s+23s+5​

      2. Set s=−1s = -1s=−1 (the second term vanishes):

        k1=3(−1)+5(−1)+2=21=2k_1 = \frac{3(-1)+5}{(-1)+2} = \frac{2}{1} = 2k1​=(−1)+23(−1)+5​=12​=2

      Find k2k_2k2​ (pole at s=−2s=-2s=−2):

      1. Multiply by (s+2)(s+2)(s+2):

        k1(s+2)s+1+k2=3s+5s+1\frac{k_1(s+2)}{s+1} + k_2 = \frac{3s+5}{s+1}s+1k1​(s+2)​+k2​=s+13s+5​

      2. Set s=−2s = -2s=−2 (the first term vanishes):

        k2=3(−2)+5(−2)+1=−1−1=1k_2 = \frac{3(-2)+5}{(-2)+1} = \frac{-1}{-1} = 1k2​=(−2)+13(−2)+5​=−1−1​=1

      Result: f(t)=2e−tu(t)+e−2tu(t)f(t) = 2e^{-t}u(t) + e^{-2t}u(t)f(t)=2e−tu(t)+e−2tu(t)

      Practical Shortcut (the actual “cover-up”):

      For F(s)=N(s)(s−p1)(s−p2)...(s−pn)F(s) = \frac{N(s)}{(s-p_1)(s-p_2)...(s-p_n)}F(s)=(s−p1​)(s−p2​)...(s−pn​)N(s)​

      To find kik_iki​:

      • Cover up (s−pi)(s-p_i)(s−pi​) in the denominator
      • Replace all remaining sss with pip_ipi​
      • Calculate the result

      Visual Example:

      F(s)=10(s+2)(s+3)(s+5)F(s) = \frac{10}{(s+2)(s+3)(s+5)}F(s)=(s+2)(s+3)(s+5)10​

      Find k2k_2k2​ at s=−3s = -3s=−3:

      • Cover up (s+3)(s+3)(s+3) in denominator: 10(s+2)(s+5)\frac{10}{(s+2)(s+5)}(s+2)(s+5)10​
      • Substitute s=−3s = -3s=−3: 10(−3+2)(−3+5)=10(−1)(2)=−5\frac{10}{(-3+2)(-3+5)} = \frac{10}{(-1)(2)} = -5(−3+2)(−3+5)10​=(−1)(2)10​=−5

      Three-Pole Example:

      F(s)=s+7(s+1)(s+2)(s+4)F(s) = \frac{s+7}{(s+1)(s+2)(s+4)}F(s)=(s+1)(s+2)(s+4)s+7​

      Find all residues:

      k1k_1k1​ at s=−1s=-1s=−1:

      k1=(−1)+7(−1+2)(−1+4)=6(1)(3)=2k_1 = \frac{(-1)+7}{(-1+2)(-1+4)} = \frac{6}{(1)(3)} = 2k1​=(−1+2)(−1+4)(−1)+7​=(1)(3)6​=2

      k2k_2k2​ at s=−2s=-2s=−2:

      k2=(−2)+7(−2+1)(−2+4)=5(−1)(2)=−52k_2 = \frac{(-2)+7}{(-2+1)(-2+4)} = \frac{5}{(-1)(2)} = -\frac{5}{2}k2​=(−2+1)(−2+4)(−2)+7​=(−1)(2)5​=−25​

      k3k_3k3​ at s=−4s=-4s=−4:

      k3=(−4)+7(−4+1)(−4+2)=3(−3)(−2)=12k_3 = \frac{(-4)+7}{(-4+1)(-4+2)} = \frac{3}{(-3)(-2)} = \frac{1}{2}k3​=(−4+1)(−4+2)(−4)+7​=(−3)(−2)3​=21​

      Answer:

      f(t)=[2e−t−52e−2t+12e−4t]u(t)f(t) = \left[2e^{-t} - \frac{5}{2}e^{-2t} + \frac{1}{2}e^{-4t}\right]u(t)f(t)=[2e−t−25​e−2t+21​e−4t]u(t)

      Important Notes:

      • Cover-up method only works for distinct (simple) poles
      • For repeated poles, use the derivative method (see Case 3)
      • For complex poles, you can use cover-up but will get complex residues (see Case 2)
      • Always verify: multiply out your PFE to check it equals the original F(s)F(s)F(s)

      Case 2: Complex Conjugate Poles

      PFE Form (poles at s=−α±jβs = -\alpha \pm j\betas=−α±jβ):

      F(s)=ks−(−α+jβ)+k∗s−(−α−jβ)F(s) = \frac{k}{s-(-\alpha+j\beta)} + \frac{k^*}{s-(-\alpha-j\beta)}F(s)=s−(−α+jβ)k​+s−(−α−jβ)k∗​

      where kkk and k∗k^*k∗ are complex conjugates.

      Time Domain:

      f(t)=2∣k∣e−αtcos⁡(βt+∠k)u(t)f(t) = 2|k|e^{-\alpha t}\cos(\beta t + \angle k)u(t)f(t)=2∣k∣e−αtcos(βt+∠k)u(t)

      Steps:

      1. Find residue kkk at one pole using cover-up method
      2. Express kkk in polar form: k=∣k∣ej∠kk = |k|e^{j\angle k}k=∣k∣ej∠k
      3. Apply formula above

      Alternative Form (using real coefficients):

      f(t)=e−αt[Acos⁡(βt)+Bsin⁡(βt)]u(t)f(t) = e^{-\alpha t}[A\cos(\beta t) + B\sin(\beta t)]u(t)f(t)=e−αt[Acos(βt)+Bsin(βt)]u(t)

      where:

      • A=2Re{k}A = 2\text{Re}\{k\}A=2Re{k}
      • B=−2Im{k}B = -2\text{Im}\{k\}B=−2Im{k}

      Case 3: Multiple-Order Poles

      PFE Form (pole at s=p1s=p_1s=p1​ with multiplicity mmm):

      F(s)=k1m(s−p1)m+k1(m−1)(s−p1)m−1+⋯+k11s−p1+(other poles)F(s) = \frac{k_{1m}}{(s-p_1)^m} + \frac{k_{1(m-1)}}{(s-p_1)^{m-1}} + \cdots + \frac{k_{11}}{s-p_1} + \text{(other poles)}F(s)=(s−p1​)mk1m​​+(s−p1​)m−1k1(m−1)​​+⋯+s−p1​k11​​+(other poles)

      Time Domain:

      f(t)=[k1mtm−1(m−1)!+k1(m−1)tm−2(m−2)!+⋯+k11]ep1tu(t)+(other terms)f(t) = \left[k_{1m}\frac{t^{m-1}}{(m-1)!} + k_{1(m-1)}\frac{t^{m-2}}{(m-2)!} + \cdots + k_{11}\right]e^{p_1t}u(t) + \text{(other terms)}f(t)=[k1m​(m−1)!tm−1​+k1(m−1)​(m−2)!tm−2​+⋯+k11​]ep1​tu(t)+(other terms)

      Finding Residues - Method 1: Matching Coefficients

      1. Find highest-order residue k1mk_{1m}k1m​ using cover-up method
      2. Multiply both sides by (s−p1)m(s-p_1)^m(s−p1​)m
      3. Expand and match coefficients of powers of sss

      Finding Residues - Method 2: n-to-1-Order-Poles For highest order residue:

      k1m=(s−p1)mF(s)∣s=p1k_{1m} = \left.(s-p_1)^m F(s)\right|_{s=p_1}k1m​=(s−p1​)mF(s)∣s=p1​​

      For next lower order:

      k1(m−1)=dds[(s−p1)mF(s)]∣s=p1k_{1(m-1)} = \left.\frac{d}{ds}\left[(s-p_1)^m F(s)\right]\right|_{s=p_1}k1(m−1)​=dsd​[(s−p1​)mF(s)]​s=p1​​

      And so on with higher derivatives.

      Case 4: Improper Rational Functions (IRF)

      If numerator order \geq denominator order:

      1. Perform long division first
      2. Write as: F(s)=Q(s)+R(s)D(s)F(s) = Q(s) + \frac{R(s)}{D(s)}F(s)=Q(s)+D(s)R(s)​
      3. Apply PFE to the proper fraction R(s)D(s)\frac{R(s)}{D(s)}D(s)R(s)​
      4. Inverse transform polynomial terms:
        • L−1{1}=δ(t)\mathcal{L}^{-1}\{1\} = \delta(t)L−1{1}=δ(t) (impulse)
        • L−1{s}=δ′(t)\mathcal{L}^{-1}\{s\} = \delta'(t)L−1{s}=δ′(t) (derivative of impulse)

      Case 5: Delayed Functions (e−sTe^{-sT}e−sT terms)

      L−1{F(s)e−sT}=f(t−T)u(t−T)\mathcal{L}^{-1}\{F(s)e^{-sT}\} = f(t-T)u(t-T)L−1{F(s)e−sT}=f(t−T)u(t−T)

      The function is delayed by TTT seconds and is zero for t<Tt < Tt<T.

      Kirchhoff’s Laws in S-Domain

      KVL (Kirchhoff’s Voltage Law):

      ∑Vi(s)=0\sum V_i(s) = 0∑Vi​(s)=0

      Sum of voltages around any closed loop equals zero (same as time domain, but with V(s)V(s)V(s)).

      KCL (Kirchhoff’s Current Law):

      ∑Ii(s)=0\sum I_i(s) = 0∑Ii​(s)=0

      Sum of currents entering a node equals sum leaving (same as time domain, but with I(s)I(s)I(s)).

      Initial and Final Value Theorems

      Initial Value Theorem (IVT)

      f(0+)=lim⁡s→∞sF(s)f(0^+) = \lim_{s \to \infty} sF(s)f(0+)=s→∞lim​sF(s)

      Requirements:

      • F(s)F(s)F(s) must be a Proper Rational Function (PRF)
      • If IRF, IVT gives ∞\infty∞ (not useful)

      Use: Quickly find initial value without doing full inverse transform.

      Final Value Theorem (FVT)

      f(∞)=lim⁡s→0sF(s)f(\infty) = \lim_{s \to 0} sF(s)f(∞)=s→0lim​sF(s)

      Requirements:

      • All poles of sF(s)sF(s)sF(s) must have Re(s)<0\text{Re}(s) < 0Re(s)<0 (lie in left half-plane)
      • If poles on imaginary axis or right half-plane, FVT is NOT APPLICABLE

      Use: Quickly find steady-state value without doing full inverse transform.

      Common Mistake: Applying FVT when poles are at s=0s=0s=0 or in right half-plane. Always check pole locations of sF(s)sF(s)sF(s) first!

      Common Mistakes to Avoid

      1. Forgetting initial conditions when using differentiation property
      2. Not checking if FVT is applicable (poles in right half-plane)
      3. Missing the u(t)u(t)u(t) in time-domain answers
      4. Incorrect sign on initial condition voltage/current sources
      5. Not factoring denominator completely before PFE
      6. Using IVT on improper rational functions
      7. Forgetting to perform long division for IRFs before PFE
      8. Not checking your answer with IVT/FVT when possible
