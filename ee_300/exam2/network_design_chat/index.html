      Network Design: Practical Math & Step-by-Step Process

      Part 1: Understanding What You’re Given - H(s)

      What is H(s)?

      H(s) is just a ratio of polynomials that describes how your circuit transforms input to output:

      H(s)=Vout(s)Vin(s)=numerator polynomialdenominator polynomialH(s) = \frac{V_{out}(s)}{V_{in}(s)} = \frac{\text{numerator polynomial}}{\text{denominator polynomial}}H(s)=Vin​(s)Vout​(s)​=denominator polynomialnumerator polynomial​

      Example: H(s)=1000s+1000H(s) = \frac{1000}{s + 1000}H(s)=s+10001000​

      Part 2: The Math You MUST Know

      Component Impedances (s-domain)

      Think of impedance as “resistance in the s-domain”:

      ComponentImpedance Z(s)When you see this in H(s)…
      Resistor RRRRJust a number (like 1000)
      Capacitor C1sC\frac{1}{sC}sC1​Terms with 1s\frac{1}{s}s1​ or numbers\frac{\text{number}}{s}snumber​
      Inductor LsLsLsLTerms with sss multiplied by a number

      Voltage Divider Formula (THE MOST IMPORTANT)

      When you have two impedances in series, the output voltage is:

      H(s)=ZbottomZtop+ZbottomH(s) = \frac{Z_{bottom}}{Z_{top} + Z_{bottom}}H(s)=Ztop​+Zbottom​Zbottom​​

      This is the foundation of most passive designs.

      OpAmp Gain Formulas

      Non-Inverting Amplifier:

      Gain=K=1+RfRg\text{Gain} = K = 1 + \frac{R_f}{R_g}Gain=K=1+Rg​Rf​​

      where RfR_fRf​ = feedback resistor, RgR_gRg​ = ground resistor

      Inverting Amplifier:

      H(s)=−ZfZinH(s) = -\frac{Z_f}{Z_{in}}H(s)=−Zin​Zf​​

      where ZfZ_fZf​ = feedback impedance, ZinZ_{in}Zin​ = input impedance

      Part 2B: Converting Circuit Equations to H(s)

      From KCL/KVL to Network Function

      The Process:

      1. Write KCL or KVL equations in the s-domain (replace components with impedances)
      2. Solve for Vout(s)V_{out}(s)Vout​(s) in terms of Vin(s)V_{in}(s)Vin​(s)
      3. Form the ratio H(s)=Vout(s)Vin(s)H(s) = \frac{V_{out}(s)}{V_{in}(s)}H(s)=Vin​(s)Vout​(s)​
      4. Simplify to get polynomial form

      Example: RC Circuit with KCL

      Given: RC low-pass filter, find H(s)

      Step 1: Draw the circuit mentally

      • Input: VinV_{in}Vin​
      • Series resistor R
      • Capacitor C to ground
      • Output: VoutV_{out}Vout​ (voltage across capacitor)

      Step 2: Write KCL at the output node

      Current through R = Current through C

      Vin(s)−Vout(s)R=sC⋅Vout(s)\frac{V_{in}(s) - V_{out}(s)}{R} = sC \cdot V_{out}(s)RVin​(s)−Vout​(s)​=sC⋅Vout​(s)

      Step 3: Solve for Vout(s)V_{out}(s)Vout​(s)

      Vin(s)−Vout(s)=sRC⋅Vout(s)V_{in}(s) - V_{out}(s) = sRC \cdot V_{out}(s)Vin​(s)−Vout​(s)=sRC⋅Vout​(s)

      Vin(s)=Vout(s)+sRC⋅Vout(s)V_{in}(s) = V_{out}(s) + sRC \cdot V_{out}(s)Vin​(s)=Vout​(s)+sRC⋅Vout​(s)

      Vin(s)=Vout(s)(1+sRC)V_{in}(s) = V_{out}(s)(1 + sRC)Vin​(s)=Vout​(s)(1+sRC)

      Vout(s)=Vin(s)1+sRCV_{out}(s) = \frac{V_{in}(s)}{1 + sRC}Vout​(s)=1+sRCVin​(s)​

      Step 4: Form H(s)

      H(s)=Vout(s)Vin(s)=11+sRCH(s) = \frac{V_{out}(s)}{V_{in}(s)} = \frac{1}{1 + sRC}H(s)=Vin​(s)Vout​(s)​=1+sRC1​

      Step 5: Rearrange to standard form

      Multiply top and bottom by 1RC\frac{1}{RC}RC1​:

      H(s)=1/RCs+1/RCH(s) = \frac{1/RC}{s + 1/RC}H(s)=s+1/RC1/RC​

      Now you have: Numerator = 1RC\frac{1}{RC}RC1​, Denominator = s+1RCs + \frac{1}{RC}s+RC1​

      Example: Using KVL

      Given: Series RLC circuit

      Step 1: Write KVL around the loop

      Vin(s)=VR(s)+VL(s)+VC(s)V_{in}(s) = V_R(s) + V_L(s) + V_C(s)Vin​(s)=VR​(s)+VL​(s)+VC​(s)

      Step 2: Express voltages in terms of current I(s)

      Vin(s)=I(s)⋅R+I(s)⋅sL+I(s)⋅1sCV_{in}(s) = I(s) \cdot R + I(s) \cdot sL + I(s) \cdot \frac{1}{sC}Vin​(s)=I(s)⋅R+I(s)⋅sL+I(s)⋅sC1​

      Step 3: Factor out I(s)

      Vin(s)=I(s)(R+sL+1sC)V_{in}(s) = I(s) \left(R + sL + \frac{1}{sC}\right)Vin​(s)=I(s)(R+sL+sC1​)

      Step 4: If output is voltage across capacitor:

      Vout(s)=I(s)⋅1sCV_{out}(s) = I(s) \cdot \frac{1}{sC}Vout​(s)=I(s)⋅sC1​

      Step 5: Find I(s) and substitute

      I(s)=Vin(s)R+sL+1sCI(s) = \frac{V_{in}(s)}{R + sL + \frac{1}{sC}}I(s)=R+sL+sC1​Vin​(s)​

      Vout(s)=Vin(s)sC(R+sL+1sC)V_{out}(s) = \frac{V_{in}(s)}{sC\left(R + sL + \frac{1}{sC}\right)}Vout​(s)=sC(R+sL+sC1​)Vin​(s)​

      Step 6: Simplify to get H(s)

      Multiply numerator and denominator by sCsCsC:

      H(s)=1s2LC+sRC+1H(s) = \frac{1}{s^2LC + sRC + 1}H(s)=s2LC+sRC+11​

      Quick Reference: Equation to Polynomial

      Given any circuit equation, follow these steps:

      1. Substitute impedances: Replace all R, L, C with their s-domain impedances
      2. Apply circuit laws: Use KCL (currents) or KVL (voltages)
      3. Isolate Vout(s)V_{out}(s)Vout​(s): Get all terms with Vout(s)V_{out}(s)Vout​(s) on one side
      4. Factor out Vout(s)V_{out}(s)Vout​(s): Group all terms multiplying output
      5. Divide both sides by Vin(s)V_{in}(s)Vin​(s): This gives you H(s)
      6. Clear fractions: Multiply top and bottom to eliminate complex fractions
      7. Arrange in polynomial form: H(s)=bmsm+...+b0ansn+...+a0H(s) = \frac{b_ms^m + ... + b_0}{a_ns^n + ... + a_0}H(s)=an​sn+...+a0​bm​sm+...+b0​​

      Part 3: The Decision Tree - Step by Step

      STEP 1: Compare Numerator to Denominator

      Question: Is the numerator “fully contained” in the denominator?

      Example 1: H(s)=1000s+1000H(s) = \frac{1000}{s + 1000}H(s)=s+10001000​

      • Numerator: 1000
      • Denominator: s + 1000
      • YES! The 1000 appears in both → Voltage divider will work

      Example 2: H(s)=2000s+1000H(s) = \frac{2000}{s + 1000}H(s)=s+10002000​

      • Numerator: 2000
      • Denominator: s + 1000
      • NO! The 2000 is bigger than 1000 → Need gain (OpAmp)

      Example 3: H(s)=−500s+1000H(s) = \frac{-500}{s + 1000}H(s)=s+1000−500​

      • Negative sign → Need inverting OpAmp

      STEP 2: Check the DC Gain (CRITICAL FOR DESIGN CHOICE)

      DC Gain tells you if you need an OpAmp!

      Set s=0s = 0s=0 in H(s) to find DC gain:

      DC Gain=H(0)=constant in numeratorconstant in denominator\text{DC Gain} = H(0) = \frac{\text{constant in numerator}}{\text{constant in denominator}}DC Gain=H(0)=constant in denominatorconstant in numerator​

      Or more generally, compare coefficients:

      For H(s)=K⋅(numerator polynomial)(denominator polynomial)H(s) = \frac{K \cdot (\text{numerator polynomial})}{(\text{denominator polynomial})}H(s)=(denominator polynomial)K⋅(numerator polynomial)​

      Gain=K=leading coefficient of numeratorleading coefficient of denominator\text{Gain} = K = \frac{\text{leading coefficient of numerator}}{\text{leading coefficient of denominator}}Gain=K=leading coefficient of denominatorleading coefficient of numerator​

      Decision Rules:

      • If ∣K∣>1|K| > 1∣K∣>1: Need OpAmp (voltage divider alone can’t amplify)
      • If K<0K < 0K<0: Need Inverting OpAmp (passive circuits can’t invert)
      • If 0<K≤10 < K \leq 10<K≤1: Voltage divider works (passive RC circuit)

      Examples:

      1. H(s)=1000s+1000H(s) = \frac{1000}{s + 1000}H(s)=s+10001000​ → H(0)=10001000=1H(0) = \frac{1000}{1000} = 1H(0)=10001000​=1 ✓ Voltage divider OK

      2. H(s)=2000s+1000H(s) = \frac{2000}{s + 1000}H(s)=s+10002000​ → H(0)=20001000=2H(0) = \frac{2000}{1000} = 2H(0)=10002000​=2 ✗ Need gain of 2 (Non-inverting OpAmp)

      3. H(s)=−500s+1000H(s) = \frac{-500}{s + 1000}H(s)=s+1000−500​ → H(0)=−5001000=−0.5H(0) = \frac{-500}{1000} = -0.5H(0)=1000−500​=−0.5 ✗ Negative! (Inverting OpAmp)

      4. H(s)=500s+1000H(s) = \frac{500}{s + 1000}H(s)=s+1000500​ → H(0)=5001000=0.5H(0) = \frac{500}{1000} = 0.5H(0)=1000500​=0.5 ✓ Voltage divider OK

      5. H(s)=3ss+100H(s) = \frac{3s}{s + 100}H(s)=s+1003s​ → H(0)=0100=0H(0) = \frac{0}{100} = 0H(0)=1000​=0, but check high frequency: lim⁡s→∞H(s)=3\lim_{s\to\infty} H(s) = 3lims→∞​H(s)=3 ✗ Need gain of 3

      Alternative Method - Compare ANY matching terms:

      H(s)=as+bcs+dH(s) = \frac{as + b}{cs + d}H(s)=cs+das+b​

      Gain can be found by: K=acK = \frac{a}{c}K=ca​ (compare s terms) OR K=bdK = \frac{b}{d}K=db​ (compare constants)

      STEP 3A: Design with Voltage Divider (No Gain Needed)

      When: Numerator is “inside” denominator, positive, no extra gain

      Process:

      1. Rewrite denominator as sum:

        Denominator=Numerator+Other stuff\text{Denominator} = \text{Numerator} + \text{Other stuff}Denominator=Numerator+Other stuff

      2. Match to impedance form:

        H(s)=Z2Z1+Z2H(s) = \frac{Z_2}{Z_1 + Z_2}H(s)=Z1​+Z2​Z2​​

        where Z2Z_2Z2​ = numerator, Z1Z_1Z1​ = “other stuff”

      3. Convert to R and C:

        • Constant terms → Resistors
        • Terms with 1s\frac{1}{s}s1​ → Divide by s to get capacitance

      Example: H(s)=1000s+1000H(s) = \frac{1000}{s + 1000}H(s)=s+10001000​

      Step 1: Rewrite denominator

      s+1000=1000+ss + 1000 = 1000 + ss+1000=1000+s

      Step 2: Match to voltage divider

      H(s)=1000s+1000=Z2Z1+Z2H(s) = \frac{1000}{s + 1000} = \frac{Z_2}{Z_1 + Z_2}H(s)=s+10001000​=Z1​+Z2​Z2​​

      So: Z2=1000Z_2 = 1000Z2​=1000 and Z1=sZ_1 = sZ1​=s

      Step 3: Convert to components

      • Z2=1000Z_2 = 1000Z2​=1000 → R = 1000Ω
      • Z1=sZ_1 = sZ1​=s → This is tricky! We want 1sC\frac{1}{sC}sC1​, so…

      TRICK: Divide everything by s:

      H(s)=1000/s1+1000/sH(s) = \frac{1000/s}{1 + 1000/s}H(s)=1+1000/s1000/s​

      Now:

      • Z2=1000s=1sCZ_2 = \frac{1000}{s} = \frac{1}{sC}Z2​=s1000​=sC1​ → C=11000=1C = \frac{1}{1000} = 1C=10001​=1 mF
      • Z1=1Z_1 = 1Z1​=1 → R = 1Ω

      Circuit: R (1Ω) in series with C (1mF), output taken across C

      STEP 3B: Design with Non-Inverting OpAmp (Positive Gain Needed)

      When: Numerator coefficient is LARGER than denominator, positive

      Process:

      1. Factor out the gain:

        H(s)=K⋅somethings+somethingH(s) = K \cdot \frac{\text{something}}{s + \text{something}}H(s)=K⋅s+somethingsomething​

      2. Design voltage divider for the fractional part

      3. Add non-inverting amplifier with gain K

      Example: H(s)=2000s+1000H(s) = \frac{2000}{s + 1000}H(s)=s+10002000​

      Step 1: Factor out gain

      H(s)=2⋅1000s+1000H(s) = 2 \cdot \frac{1000}{s + 1000}H(s)=2⋅s+10001000​

      Step 2: Design voltage divider for 1000s+1000\frac{1000}{s + 1000}s+10001000​ (use Step 3A)

      Step 3: Non-inverting amp gain formula:

      K=1+RfRgK = 1 + \frac{R_f}{R_g}K=1+Rg​Rf​​

      For K = 2:

      2=1+RfRg2 = 1 + \frac{R_f}{R_g}2=1+Rg​Rf​​

      Rf=RgR_f = R_gRf​=Rg​

      Choose Rg=1R_g = 1Rg​=1 kΩ, Rf=1R_f = 1Rf​=1 kΩ

      STEP 3C: Design with Inverting OpAmp (Negative Gain)

      When: Negative sign in H(s)

      Process: Use impedance ratio formula:

      H(s)=−ZfZinH(s) = -\frac{Z_f}{Z_{in}}H(s)=−Zin​Zf​​

      where ZfZ_fZf​ = feedback impedance, ZinZ_{in}Zin​ = input impedance

      Example: H(s)=−1000s+1000H(s) = -\frac{1000}{s + 1000}H(s)=−s+10001000​

      Match to formula:

      −ZfZin=−1000s+1000-\frac{Z_f}{Z_{in}} = -\frac{1000}{s + 1000}−Zin​Zf​​=−s+10001000​

      So: Zf=1000Z_f = 1000Zf​=1000 and Zin=s+1000Z_{in} = s + 1000Zin​=s+1000

      Convert:

      • Zf=1000Z_f = 1000Zf​=1000 → Rf=1000R_f = 1000Rf​=1000 Ω
      • Zin=s+1000Z_{in} = s + 1000Zin​=s+1000 → Rin+1sCR_{in} + \frac{1}{sC}Rin​+sC1​

      Divide by s: Zin=1+1000sZ_{in} = 1 + \frac{1000}{s}Zin​=1+s1000​

      • Rin=1R_{in} = 1Rin​=1 Ω
      • C=11000C = \frac{1}{1000}C=10001​ F

      STEP 4: Second-Order Systems (Overdamped)

      When: Denominator is 2nd order (has s2s^2s2 term), real poles

      Strategy: Factor into two 1st-order systems and cascade them

      Example: H(s)=106(s+100)(s+1000)H(s) = \frac{10^6}{(s + 100)(s + 1000)}H(s)=(s+100)(s+1000)106​

      Step 1: Factor

      H(s)=100s+100⋅10000s+1000H(s) = \frac{100}{s + 100} \cdot \frac{10000}{s + 1000}H(s)=s+100100​⋅s+100010000​

      Wait! That gives us gains > 1. Let’s rewrite:

      H(s)=10⋅100s+100⋅1000s+1000H(s) = 10 \cdot \frac{100}{s + 100} \cdot \frac{1000}{s + 1000}H(s)=10⋅s+100100​⋅s+10001000​

      Step 2: Design each part:

      • First stage: 100s+100\frac{100}{s + 100}s+100100​ (voltage divider)
      • Buffer (voltage follower) to prevent loading
      • Second stage: 1000s+1000\frac{1000}{s + 1000}s+10001000​ (voltage divider)
      • Buffer
      • Gain stage: Non-inverting amp with gain = 10

      Part 4: The Divide-by-s Trick (CRITICAL)

      Why we do it: To make capacitor impedances obvious

      Given: H(s)=1000s+1000H(s) = \frac{1000}{s + 1000}H(s)=s+10001000​

      Divide numerator and denominator by s:

      H(s)=1000/s1+1000/sH(s) = \frac{1000/s}{1 + 1000/s}H(s)=1+1000/s1000/s​

      Now you can SEE:

      • 1000s\frac{1000}{s}s1000​ looks like 1sC\frac{1}{sC}sC1​ → so C=11000C = \frac{1}{1000}C=10001​
      • 111 looks like RRR → so R=1R = 1R=1

      Part 7: The Universal Checklist

      Before you start designing:

      1. ✓ Factor H(s) completely
      2. ✓ Check if gain > 1 or < 0
      3. ✓ Prefer capacitors over inductors
      4. ✓ Use voltage divider if possible
      5. ✓ Add OpAmp only when necessary
      6. ✓ For 2nd order: factor and cascade
      7. ✓ Always use buffers between cascaded stages
