      Complete Guide: Zero-Input Response (ZIR) and Zero-State Response (ZSR)

      Overview

      The total response of a linear circuit can be decomposed into two fundamental components using the principle of superposition:

      vtotal(t)=vZSR(t)+vZIR(t)\boxed{v_{\text{total}}(t) = v_{\text{ZSR}}(t) + v_{\text{ZIR}}(t)}vtotal​(t)=vZSR​(t)+vZIR​(t)​

      This separation is powerful because it allows us to analyze:

      • Initial condition effects (ZIR) separately from
      • Input source effects (ZSR)

      1. Conceptual Understanding

      Zero-Input Response (ZIR)

      Definition: The circuit response due only to initial conditions (stored energy), with all external inputs set to zero.

      Key Characteristics:

      • Caused by: Initial capacitor voltage vC(0)v_C(0)vC​(0) or inductor current iL(0)i_L(0)iL​(0)
      • Input sources: Set to zero (voltage sources → short circuit, current sources → open circuit)
      • Nature: Pure natural response - determined entirely by circuit components (R, L, C)
      • Physical meaning: Energy stored in the circuit “relaxing” or dissipating
      • Response form: Exponential decay based on circuit time constants

      In S-domain: Generated by initial condition sources vC(0)s\frac{v_C(0)}{s}svC​(0)​ or iL(0)s\frac{i_L(0)}{s}siL​(0)​

      Zero-State Response (ZSR)

      Definition: The circuit response due only to external input sources, assuming zero initial energy storage.

      Key Characteristics:

      • Caused by: External input vs(t)v_s(t)vs​(t) or is(t)i_s(t)is​(t)
      • Initial conditions: Set to zero (vC(0)=0v_C(0) = 0vC​(0)=0, iL(0)=0i_L(0) = 0iL​(0)=0)
      • Nature: Contains both natural and forced responses
      • Physical meaning: System response to external excitation
      • Response form: Natural component (exponentials) + forced component (matching input form)

      In S-domain: Generated by transformed input sources Vs(s)V_s(s)Vs​(s) or Is(s)I_s(s)Is​(s)

      2. S-Domain Component Models

      Resistor

      ZR=RZ_R = RZR​=R
      • No initial condition source (no energy storage)

      Capacitor

      Impedance:

      ZC=1sCZ_C = \frac{1}{sC}ZC​=sC1​

      Complete S-domain model:

      VC(s)=1sCIC(s)+vC(0)sV_C(s) = \frac{1}{sC} I_C(s) + \frac{v_C(0)}{s}VC​(s)=sC1​IC​(s)+svC​(0)​

      Equivalent circuit: Impedance 1sC\frac{1}{sC}sC1​ in series with voltage source vC(0)s\frac{v_C(0)}{s}svC​(0)​

      Inductor

      Impedance:

      ZL=sLZ_L = sLZL​=sL

      Complete S-domain model (voltage form):

      VL(s)=sL⋅IL(s)−LiL(0)V_L(s) = sL \cdot I_L(s) - Li_L(0)VL​(s)=sL⋅IL​(s)−LiL​(0)

      Complete S-domain model (current form):

      IL(s)=VL(s)sL+iL(0)sI_L(s) = \frac{V_L(s)}{sL} + \frac{i_L(0)}{s}IL​(s)=sLVL​(s)​+siL​(0)​

      Equivalent circuits:

      • Impedance sLsLsL in series with voltage source LiL(0)Li_L(0)LiL​(0) (opposite polarity), OR
      • Impedance sLsLsL in parallel with current source iL(0)s\frac{i_L(0)}{s}siL​(0)​

      3. Step-by-Step Solution Process

      Phase 0: Initial Setup

      Step 1: Draw the time-domain circuit and identify all components

      Step 2: Find initial conditions for t<0t < 0t<0 (steady-state DC analysis)

      • Capacitors: Open circuits (no current flows)
      • Inductors: Short circuits (no voltage drop)
      • Calculate vC(0−)v_C(0^-)vC​(0−) and iL(0−)i_L(0^-)iL​(0−)

      Step 3: Apply continuity conditions

      • Capacitor voltage cannot change instantly: vC(0+)=vC(0−)v_C(0^+) = v_C(0^-)vC​(0+)=vC​(0−)
      • Inductor current cannot change instantly: iL(0+)=iL(0−)i_L(0^+) = i_L(0^-)iL​(0+)=iL​(0−)

      Step 4: Transform circuit to S-domain for t≥0t \geq 0t≥0

      • Replace each component with its S-domain equivalent
      • Include initial condition sources
      • Transform input sources using Laplace tables

      Phase 1: Calculate Zero-State Response (ZSR)

      Step 1: Modify the S-domain circuit:

      • Zero all initial condition sources
        • Remove (short circuit) capacitor voltage sources vC(0)s\frac{v_C(0)}{s}svC​(0)​
        • Remove (open circuit) inductor current sources iL(0)s\frac{i_L(0)}{s}siL​(0)​
      • Keep external input sources active

      Step 2: Analyze the circuit to find VZSR(s)V_{\text{ZSR}}(s)VZSR​(s) or IZSR(s)I_{\text{ZSR}}(s)IZSR​(s)

      • Use KVL, KCL, nodal analysis, or mesh analysis
      • Apply voltage/current dividers where applicable

      Step 3: Perform Partial Fraction Expansion (PFE) on VZSR(s)V_{\text{ZSR}}(s)VZSR​(s)

      Step 4: Apply Inverse Laplace Transform to get vZSR(t)v_{\text{ZSR}}(t)vZSR​(t)

      Phase 2: Calculate Zero-Input Response (ZIR)

      Step 1: Modify the S-domain circuit:

      • Zero all external input sources
        • Short circuit voltage sources
        • Open circuit current sources
      • Keep initial condition sources active

      Step 2: Analyze the circuit to find VZIR(s)V_{\text{ZIR}}(s)VZIR​(s) or IZIR(s)I_{\text{ZIR}}(s)IZIR​(s)

      Step 3: Perform Partial Fraction Expansion (PFE) on VZIR(s)V_{\text{ZIR}}(s)VZIR​(s)

      Step 4: Apply Inverse Laplace Transform to get vZIR(t)v_{\text{ZIR}}(t)vZIR​(t)

      Phase 3: Combine Responses

      Step 1: Add responses in time domain:

      vtotal(t)=vZSR(t)+vZIR(t)for t≥0v_{\text{total}}(t) = v_{\text{ZSR}}(t) + v_{\text{ZIR}}(t) \quad \text{for } t \geq 0vtotal​(t)=vZSR​(t)+vZIR​(t)for t≥0

      Step 2: Verify (optional but recommended):

      • Calculate total response directly without separation
      • Check that it equals ZSR + ZIR

      5. Partial Fraction Expansion (PFE)

      For Distinct Real Poles

      If V(s)=N(s)(s−p1)(s−p2)⋯(s−pn)V(s) = \frac{N(s)}{(s-p_1)(s-p_2)\cdots(s-p_n)}V(s)=(s−p1​)(s−p2​)⋯(s−pn​)N(s)​ where all poles are distinct:

      V(s)=k1s−p1+k2s−p2+⋯+kns−pnV(s) = \frac{k_1}{s-p_1} + \frac{k_2}{s-p_2} + \cdots + \frac{k_n}{s-p_n}V(s)=s−p1​k1​​+s−p2​k2​​+⋯+s−pn​kn​​

      Cover-up method for residue kik_iki​:

      ki=[(s−pi)V(s)]s=pik_i = \left[(s-p_i)V(s)\right]_{s=p_i}ki​=[(s−pi​)V(s)]s=pi​​

      For Repeated Poles

      If pole p1p_1p1​ has multiplicity rrr:

      V(s)=k1,r(s−p1)r+k1,r−1(s−p1)r−1+⋯+k1,1s−p1+(other terms)V(s) = \frac{k_{1,r}}{(s-p_1)^r} + \frac{k_{1,r-1}}{(s-p_1)^{r-1}} + \cdots + \frac{k_{1,1}}{s-p_1} + \text{(other terms)}V(s)=(s−p1​)rk1,r​​+(s−p1​)r−1k1,r−1​​+⋯+s−p1​k1,1​​+(other terms)

      6. Worked Example: Series RC Circuit

      Given:

      • R=2 ΩR = 2\,\OmegaR=2Ω
      • C=0.5 FC = 0.5\,\text{F}C=0.5F
      • Input: vs(t)=10u(t) Vv_s(t) = 10u(t)\,\text{V}vs​(t)=10u(t)V
      • Initial condition: vC(0)=5 Vv_C(0) = 5\,\text{V}vC​(0)=5V

      Find: Capacitor voltage vC(t)v_C(t)vC​(t) for t≥0t \geq 0t≥0, separated into ZIR and ZSR.

      Solution: Zero-State Response (ZSR)

      Step 1: Set vC(0)=0v_C(0) = 0vC​(0)=0, keep input active

      Step 2: S-domain circuit equation (KVL):

      Vs(s)=RIC(s)+1sCIC(s)V_s(s) = RI_C(s) + \frac{1}{sC}I_C(s)Vs​(s)=RIC​(s)+sC1​IC​(s)10s=IC(s)(2+2s)=IC(s)2(s+1)s\frac{10}{s} = I_C(s)\left(2 + \frac{2}{s}\right) = I_C(s)\frac{2(s+1)}{s}s10​=IC​(s)(2+s2​)=IC​(s)s2(s+1)​IC(s)=5s+1I_C(s) = \frac{5}{s+1}IC​(s)=s+15​

      Step 3: Calculate capacitor voltage:

      VC,ZSR(s)=1sCIC(s)=2s⋅5s+1=10s(s+1)V_{C,\text{ZSR}}(s) = \frac{1}{sC} I_C(s) = \frac{2}{s} \cdot \frac{5}{s+1} = \frac{10}{s(s+1)}VC,ZSR​(s)=sC1​IC​(s)=s2​⋅s+15​=s(s+1)10​

      Step 4: Partial Fraction Expansion:

      10s(s+1)=As+Bs+1\frac{10}{s(s+1)} = \frac{A}{s} + \frac{B}{s+1}s(s+1)10​=sA​+s+1B​

      Using cover-up method:

      • A=[s⋅10s(s+1)]s=0=10A = \left[s \cdot \frac{10}{s(s+1)}\right]_{s=0} = 10A=[s⋅s(s+1)10​]s=0​=10
      • B=[(s+1)⋅10s(s+1)]s=−1=−10B = \left[(s+1) \cdot \frac{10}{s(s+1)}\right]_{s=-1} = -10B=[(s+1)⋅s(s+1)10​]s=−1​=−10
      VC,ZSR(s)=10s−10s+1V_{C,\text{ZSR}}(s) = \frac{10}{s} - \frac{10}{s+1}VC,ZSR​(s)=s10​−s+110​

      Step 5: Inverse transform:

      vC,ZSR(t)=10(1−e−t) V\boxed{v_{C,\text{ZSR}}(t) = 10(1 - e^{-t})\,\text{V}}vC,ZSR​(t)=10(1−e−t)V​

      Solution: Zero-Input Response (ZIR)

      Step 1: Set input to zero (short circuit), keep vC(0)=5 Vv_C(0) = 5\,\text{V}vC​(0)=5V

      Step 2: S-domain circuit equation (KVL):

      0=RIC(s)+1sCIC(s)+vC(0)s0 = RI_C(s) + \frac{1}{sC}I_C(s) + \frac{v_C(0)}{s}0=RIC​(s)+sC1​IC​(s)+svC​(0)​−5s=IC(s)2(s+1)s-\frac{5}{s} = I_C(s)\frac{2(s+1)}{s}−s5​=IC​(s)s2(s+1)​IC(s)=−52(s+1)I_C(s) = -\frac{5}{2(s+1)}IC​(s)=−2(s+1)5​

      Step 3: Calculate capacitor voltage:

      VC,ZIR(s)=1sCIC(s)+vC(0)sV_{C,\text{ZIR}}(s) = \frac{1}{sC}I_C(s) + \frac{v_C(0)}{s}VC,ZIR​(s)=sC1​IC​(s)+svC​(0)​VC,ZIR(s)=2s⋅(−52(s+1))+5sV_{C,\text{ZIR}}(s) = \frac{2}{s} \cdot \left(-\frac{5}{2(s+1)}\right) + \frac{5}{s}VC,ZIR​(s)=s2​⋅(−2(s+1)5​)+s5​VC,ZIR(s)=−5s(s+1)+5sV_{C,\text{ZIR}}(s) = -\frac{5}{s(s+1)} + \frac{5}{s}VC,ZIR​(s)=−s(s+1)5​+s5​

      Step 4: Simplify using PFE on first term:

      −5s(s+1)=−5s+5s+1-\frac{5}{s(s+1)} = -\frac{5}{s} + \frac{5}{s+1}−s(s+1)5​=−s5​+s+15​VC,ZIR(s)=−5s+5s+1+5s=5s+1V_{C,\text{ZIR}}(s) = -\frac{5}{s} + \frac{5}{s+1} + \frac{5}{s} = \frac{5}{s+1}VC,ZIR​(s)=−s5​+s+15​+s5​=s+15​

      Step 5: Inverse transform:

      vC,ZIR(t)=5e−t V\boxed{v_{C,\text{ZIR}}(t) = 5e^{-t}\,\text{V}}vC,ZIR​(t)=5e−tV​

      Total Response

      vC(t)=vC,ZSR(t)+vC,ZIR(t)v_C(t) = v_{C,\text{ZSR}}(t) + v_{C,\text{ZIR}}(t)vC​(t)=vC,ZSR​(t)+vC,ZIR​(t)vC(t)=10(1−e−t)+5e−t=10−5e−t V\boxed{v_C(t) = 10(1-e^{-t}) + 5e^{-t} = 10 - 5e^{-t}\,\text{V}}vC​(t)=10(1−e−t)+5e−t=10−5e−tV​

      Physical interpretation:

      • ZIR: Initial 5V decays exponentially to 0V (τ = RC = 1s)
      • ZSR: Charges from 0V to 10V exponentially (same τ)
      • Total: Transitions from 5V to 10V exponentially

      8. Common Mistakes to Avoid

      1. ❌ Forgetting initial condition sources when calculating ZIR
      2. ❌ Not zeroing initial conditions when calculating ZSR
      3. ❌ Incorrect polarity of initial condition sources
      4. ❌ Sign errors in series-connected sources
      5. ❌ Confusing natural response with ZIR (ZSR also has natural components!)
      6. ❌ Not verifying that ZIR + ZSR = Total Response
      7. ❌ Wrong impedance forms (forgetting the sss in sLsLsL or 1sC\frac{1}{sC}sC1​)

      9. Time Constants Reference

      Series RC: τ=RC\tau = RCτ=RC

      Series RL: τ=LR\tau = \frac{L}{R}τ=RL​

      Parallel RC: τ=RC\tau = RCτ=RC

      Parallel RL: τ=LR\tau = \frac{L}{R}τ=RL​

      Response form: v(t)=Vf+(Vi−Vf)e−t/τv(t) = V_f + (V_i - V_f)e^{-t/\tau}v(t)=Vf​+(Vi​−Vf​)e−t/τ

      Where:

      • ViV_iVi​ = initial value
      • VfV_fVf​ = final (steady-state) value
      • After 5τ5\tau5τ, response reaches ~99.3% of final value
