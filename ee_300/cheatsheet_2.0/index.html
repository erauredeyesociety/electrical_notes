      3. Circuit Response Components

      Total Response Breakdown

      Total Response=ZIR+ZSR\text{Total Response} = \text{ZIR} + \text{ZSR}Total Response=ZIR+ZSR

      Zero-Input Response (ZIR):

      • Caused by initial conditions only (stored energy)
      • No external sources
      • Purely natural response

      Zero-State Response (ZSR):

      • Caused by external inputs only
      • All initial conditions = 0
      • Contains natural + forced response

      Natural vs Forced Response

      Natural Response:

      • From poles of H(s)H(s)H(s) (network function)
      • Circuit’s inherent modes
      • Independent of input type
      • Determined by R, L, C values

      Forced Response:

      • From poles of input signal Vin(s)V_{in}(s)Vin​(s)
      • Same frequency as input
      • Amplitude scaled by H(s)H(s)H(s) at input frequency

      5. Poles and Zeros

      Definitions

      ZEROS: Solutions to N(s)=0N(s) = 0N(s)=0 (numerator roots)

      • Frequencies where H(s)=0H(s) = 0H(s)=0
      • Signals at these frequencies are blocked from output
      • Correspond to input side of differential equation

      POLES: Solutions to D(s)=0D(s) = 0D(s)=0 (denominator roots, characteristic equation)

      • Frequencies where H(s)→∞H(s) \to \inftyH(s)→∞
      • Determine natural response modes
      • Always present in output
      • Correspond to output side of differential equation

      7. Initial and Final Value Theorems

      Initial Value Theorem (IVT)

      f(0+)=lim⁡s→∞sF(s)f(0^+) = \lim_{s \to \infty} sF(s)f(0+)=s→∞lim​sF(s)

      Requirements: F(s)F(s)F(s) must be PRF (proper rational function)

      Final Value Theorem (FVT)

      f(∞)=lim⁡s→0sF(s)f(\infty) = \lim_{s \to 0} sF(s)f(∞)=s→0lim​sF(s)

      Requirements: ALL poles of sF(s)sF(s)sF(s) must have Re(s)<0\text{Re}(s) < 0Re(s)<0 (left half-plane)

      ⚠️ Warning: FVT is invalid if poles exist at s=0s = 0s=0 or in right half-plane

      8. Step-by-Step Problem-Solving Procedures

      A. Finding Network Function H(s) from Circuit

      1. Transform all elements to s-domain: RRR, 1sC\frac{1}{sC}sC1​, sLsLsL
      2. Zero ALL initial conditions (voltage sources → short, current sources → open)
      3. Transform input sources to s-domain
      4. Define input and output variables
      5. Apply circuit analysis:
        • Voltage/current divider
        • Series/parallel combinations
        • Nodal analysis (KCL)
        • Mesh analysis (KVL)
      6. Form H(s)=Output(s)Input(s)H(s) = \frac{\text{Output}(s)}{\text{Input}(s)}H(s)=Input(s)Output(s)​
      7. Simplify to rational function form

      B. Finding Poles and Zeros

      1. Express H(s)=N(s)D(s)H(s) = \frac{N(s)}{D(s)}H(s)=D(s)N(s)​ in polynomial form
      2. Find Zeros: Set N(s)=0N(s) = 0N(s)=0, solve for sss
        • 1st order: s=−bas = -\frac{b}{a}s=−ab​
        • 2nd order: s=−b±b2−4ac2as = \frac{-b \pm \sqrt{b^2-4ac}}{2a}s=2a−b±b2−4ac​​
        • Higher: Factor or numerical methods
      3. Find Poles: Set D(s)=0D(s) = 0D(s)=0, solve for sss (same methods)
      4. Check discriminant (Δ=b2−4ac\Delta = b^2 - 4acΔ=b2−4ac) for 2nd order:
        • Δ>0\Delta > 0Δ>0: Two real poles
        • Δ=0\Delta = 0Δ=0: Repeated real pole
        • Δ<0\Delta < 0Δ<0: Complex conjugate pair

      C. Finding Time Response via Inverse Laplace

      1. Calculate Vout(s)=H(s)⋅Vin(s)V_{out}(s) = H(s) \cdot V_{in}(s)Vout​(s)=H(s)⋅Vin​(s)
      2. Check if PRF or IRF
        • If IRF: Perform long division first
      3. Factor denominator completely to find ALL poles
      4. Setup PFE based on pole types (distinct, complex, repeated)
      5. Find residues:
        • Simple poles: Cover-up method
        • Complex poles: Cover-up, then convert to magnitude/angle
        • Repeated poles: Derivative method
      6. Inverse transform each term
      7. Add u(t)u(t)u(t) to ensure causality
      8. Identify components:
        • Terms from H(s)H(s)H(s) poles = Natural response
        • Terms from Vin(s)V_{in}(s)Vin​(s) poles = Forced response

      D. Converting H(s) to Differential Equation

      1. Start with H(s)=Vout(s)Vin(s)=N(s)D(s)H(s) = \frac{V_{out}(s)}{V_{in}(s)} = \frac{N(s)}{D(s)}H(s)=Vin​(s)Vout​(s)​=D(s)N(s)​
      2. Cross-multiply: Vout(s)⋅D(s)=Vin(s)⋅N(s)V_{out}(s) \cdot D(s) = V_{in}(s) \cdot N(s)Vout​(s)⋅D(s)=Vin​(s)⋅N(s)
      3. Replace sss operators:
        • s→ddts \to \frac{d}{dt}s→dtd​
        • s2→d2dt2s^2 \to \frac{d^2}{dt^2}s2→dt2d2​
        • sn→dndtns^n \to \frac{d^n}{dt^n}sn→dtndn​
      4. Write ODE: D(ddt)vout(t)=N(ddt)vin(t)D\left(\frac{d}{dt}\right) v_{out}(t) = N\left(\frac{d}{dt}\right) v_{in}(t)D(dtd​)vout​(t)=N(dtd​)vin​(t)

      Key: Denominator (poles) → output side (left), Numerator (zeros) → input side (right)

      E. Solving Differential Equations with Laplace

      1. Take Laplace transform of entire equation
      2. Apply differentiation property: L{f(n)}=snF(s)−sn−1f(0−)−⋯−f(n−1)(0−)\mathcal{L}\{f^{(n)}\} = s^n F(s) - s^{n-1}f(0^-) - \cdots - f^{(n-1)}(0^-)L{f(n)}=snF(s)−sn−1f(0−)−⋯−f(n−1)(0−)
      3. Substitute known initial conditions
      4. Solve algebraically for F(s)F(s)F(s)
      5. Apply inverse Laplace (PFE + residues)
      6. Verify using IVT/FVT if applicable

      F. Complete Circuit Analysis with Initial Conditions

      1. Analyze t<0t < 0t<0 circuit: Find vC(0−)v_C(0^-)vC​(0−) and iL(0−)i_L(0^-)iL​(0−)
      2. Draw s-domain circuit for t≥0t \geq 0t≥0:
        • Replace elements with impedances
        • Add initial condition sources:
          • Capacitor: vc(0−)s\frac{v_c(0^-)}{s}svc​(0−)​ in series with 1sC\frac{1}{sC}sC1​
          • Inductor: LiL(0−)Li_L(0^-)LiL​(0−) in series with sLsLsL
        • Transform input sources
      3. Apply circuit analysis (KVL, KCL, nodal, mesh) in s-domain
      4. Solve for desired V(s)V(s)V(s) or I(s)I(s)I(s)
      5. Apply inverse Laplace transform
      6. (Optional) Verify with IVT: v(0+)v(0^+)v(0+) or FVT: v(∞)v(\infty)v(∞)

      9. Problem-Solving Decision Tree

      When given a CIRCUIT:

      1. Need H(s)H(s)H(s)? → Procedure A (zero ICs!)
      2. Need poles/zeros? → First find H(s)H(s)H(s), then Procedure B
      3. Need time response? → Find H(s)H(s)H(s), then Procedure C
      4. Have initial conditions? → Procedure F

      When given H(s):

      1. Need poles/zeros? → Procedure B
      2. Need differential equation? → Procedure D
      3. Need time response? → Procedure C (need input too)

      When given DIFFERENTIAL EQUATION:

      1. Need H(s)H(s)H(s)? → Laplace transform both sides, solve for Output/Input
      2. Need solution? → Procedure E
      3. Need poles/zeros? → First get H(s)H(s)H(s), then Procedure B

      10. Critical Reminders & Common Mistakes

      Must Remember

      ✓ Always zero initial conditions when finding H(s)H(s)H(s) ✓ H(s)H(s)H(s) describes ZSR only, not ZIR ✓ Include u(t)u(t)u(t) in all time-domain responses ✓ Complex poles always come in conjugate pairs ✓ Check FVT applicability before using (poles must be in left half-plane) ✓ Factor denominator completely before PFE

      Common Errors

      ✗ Wrong impedances: CCC is 1sC\frac{1}{sC}sC1​ NOT sCsCsC; LLL is sLsLsL NOT 1sL\frac{1}{sL}sL1​ ✗ Confusing poles (denominator) with zeros (numerator) ✗ Using FVT when poles at origin or in right half-plane ✗ Forgetting initial condition sources in s-domain circuit ✗ Not checking if PRF before applying IVT ✗ Mixing up natural (from H(s)H(s)H(s) poles) vs forced (from input poles) response ✗ Incorrect signs on initial condition sources
