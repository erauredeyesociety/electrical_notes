      PATH B: Complete Process with Examples

      Overview

      This document presents the complete PATH B process from start to finish. PATH B derives the differential equation first in time domain, then applies Laplace transform.

        graph TD
    A[Time Domain Circuit] --> B[Step 1: Apply KVL/KCL<br/>in Time Domain]
    B --> C[Step 2: Substitute<br/>Component Equations]
    C --> D[Step 3: Derive<br/>Differential Equation]
    D --> E[Step 4: Apply<br/>Laplace Transform]
    E --> F[Step 5: Substitute<br/>Initial Conditions]
    F --> G[Step 6: Solve for<br/>Vs or Is]
    G --> H[Step 7: Ready for<br/>Inverse Laplace]

      Complete Example 1: Series RC Circuit

      Problem Statement

      Find the voltage across the capacitor v_C(t) for t ≥ 0.

      Given:

      • v_s(t) = 10u(t) V
      • R = 100Ω
      • C = 10µF
      • v_C(0) = 0

      Time Domain Circuit 

      v_s(t) = 10u(t) V ----[R=100Ω]----[C=10µF]---- ground

      STEP 1: Apply KVL in Time Domain

      Going around the loop:

      vs(t)=vR(t)+vC(t)v_s(t) = v_R(t) + v_C(t)vs​(t)=vR​(t)+vC​(t)

      STEP 2: Substitute Component Equations

      For resistor:

      vR(t)=R⋅i(t)v_R(t) = R \cdot i(t)vR​(t)=R⋅i(t)

      For capacitor:

      i(t)=CdvC(t)dti(t) = C\frac{dv_C(t)}{dt}i(t)=CdtdvC​(t)​

      Therefore:

      vR(t)=RCdvC(t)dtv_R(t) = RC\frac{dv_C(t)}{dt}vR​(t)=RCdtdvC​(t)​

      Substitute values:

      vR(t)=100×10×10−6dvC(t)dt=10−3dvC(t)dtv_R(t) = 100 \times 10 \times 10^{-6} \frac{dv_C(t)}{dt} = 10^{-3}\frac{dv_C(t)}{dt}vR​(t)=100×10×10−6dtdvC​(t)​=10−3dtdvC​(t)​

      STEP 3: Derive Differential Equation

      Substitute into KVL:

      vs(t)=10−3dvC(t)dt+vC(t)v_s(t) = 10^{-3}\frac{dv_C(t)}{dt} + v_C(t)vs​(t)=10−3dtdvC​(t)​+vC​(t)

      Rearrange to standard form:

      10−3dvC(t)dt+vC(t)=vs(t)10^{-3}\frac{dv_C(t)}{dt} + v_C(t) = v_s(t)10−3dtdvC​(t)​+vC​(t)=vs​(t)

      Divide by 10⁻³:

      dvC(t)dt+1000vC(t)=1000vs(t)\frac{dv_C(t)}{dt} + 1000v_C(t) = 1000v_s(t)dtdvC​(t)​+1000vC​(t)=1000vs​(t)

      For v_s(t) = 10u(t):

      dvC(t)dt+1000vC(t)=10000u(t)\frac{dv_C(t)}{dt} + 1000v_C(t) = 10000u(t)dtdvC​(t)​+1000vC​(t)=10000u(t)

      This is the differential equation.

      STEP 4: Apply Laplace Transform

      Transform both sides:

      L{dvC(t)dt}+1000L{vC(t)}=10000L{u(t)}\mathcal{L}\left\{\frac{dv_C(t)}{dt}\right\} + 1000\mathcal{L}\{v_C(t)\} = 10000\mathcal{L}\{u(t)\}L{dtdvC​(t)​}+1000L{vC​(t)}=10000L{u(t)}

      Use differentiation property:

      sVC(s)−vC(0)+1000VC(s)=10000⋅1ssV_C(s) - v_C(0) + 1000V_C(s) = 10000 \cdot \frac{1}{s}sVC​(s)−vC​(0)+1000VC​(s)=10000⋅s1​

      STEP 5: Substitute Initial Conditions

      Given v_C(0) = 0:

      sVC(s)−0+1000VC(s)=10000ssV_C(s) - 0 + 1000V_C(s) = \frac{10000}{s}sVC​(s)−0+1000VC​(s)=s10000​VC(s)[s+1000]=10000sV_C(s)[s + 1000] = \frac{10000}{s}VC​(s)[s+1000]=s10000​

      STEP 6: Solve for V_C(s)

      VC(s)=10000s(s+1000)V_C(s) = \frac{10000}{s(s+1000)}VC​(s)=s(s+1000)10000​

      STEP 7: Ready for Inverse Laplace Transform

      Final result:

      VC(s)=10000s(s+1000)V_C(s) = \frac{10000}{s(s+1000)}VC​(s)=s(s+1000)10000​

      Note: This exactly matches PATH A Example 1!

      Complete Example 2: Series RL Circuit

      Problem Statement

      Find the current i(t) for t ≥ 0.

      Given:

      • v_s(t) = 20u(t) V
      • R = 10Ω
      • L = 0.5H
      • i_L(0) = 2A

      Time Domain Circuit 

      v_s(t) = 20u(t) V ----[R=10Ω]----[L=0.5H]---- ground

      STEP 1: Apply KVL in Time Domain

      vs(t)=vR(t)+vL(t)v_s(t) = v_R(t) + v_L(t)vs​(t)=vR​(t)+vL​(t)

      STEP 2: Substitute Component Equations

      For resistor:

      vR(t)=Ri(t)=10i(t)v_R(t) = Ri(t) = 10i(t)vR​(t)=Ri(t)=10i(t)

      For inductor:

      vL(t)=Ldi(t)dt=0.5di(t)dtv_L(t) = L\frac{di(t)}{dt} = 0.5\frac{di(t)}{dt}vL​(t)=Ldtdi(t)​=0.5dtdi(t)​

      STEP 3: Derive Differential Equation

      Substitute into KVL:

      vs(t)=10i(t)+0.5di(t)dtv_s(t) = 10i(t) + 0.5\frac{di(t)}{dt}vs​(t)=10i(t)+0.5dtdi(t)​

      Rearrange:

      0.5di(t)dt+10i(t)=vs(t)0.5\frac{di(t)}{dt} + 10i(t) = v_s(t)0.5dtdi(t)​+10i(t)=vs​(t)

      Divide by 0.5:

      di(t)dt+20i(t)=2vs(t)\frac{di(t)}{dt} + 20i(t) = 2v_s(t)dtdi(t)​+20i(t)=2vs​(t)

      For v_s(t) = 20u(t):

      di(t)dt+20i(t)=40u(t)\frac{di(t)}{dt} + 20i(t) = 40u(t)dtdi(t)​+20i(t)=40u(t)

      This is the differential equation.

      STEP 4: Apply Laplace Transform

      L{di(t)dt}+20L{i(t)}=40L{u(t)}\mathcal{L}\left\{\frac{di(t)}{dt}\right\} + 20\mathcal{L}\{i(t)\} = 40\mathcal{L}\{u(t)\}L{dtdi(t)​}+20L{i(t)}=40L{u(t)}sI(s)−i(0)+20I(s)=40ssI(s) - i(0) + 20I(s) = \frac{40}{s}sI(s)−i(0)+20I(s)=s40​

      STEP 5: Substitute Initial Conditions

      Given i(0) = 2A:

      sI(s)−2+20I(s)=40ssI(s) - 2 + 20I(s) = \frac{40}{s}sI(s)−2+20I(s)=s40​I(s)[s+20]=40s+2I(s)[s + 20] = \frac{40}{s} + 2I(s)[s+20]=s40​+2

      Common denominator:

      I(s)[s+20]=40+2ssI(s)[s + 20] = \frac{40 + 2s}{s}I(s)[s+20]=s40+2s​

      STEP 6: Solve for I(s)

      I(s)=40+2ss(s+20)=2s+40s(s+20)I(s) = \frac{40 + 2s}{s(s+20)} = \frac{2s + 40}{s(s+20)}I(s)=s(s+20)40+2s​=s(s+20)2s+40​

      Factor:

      I(s)=2(s+20)s(s+20)=2sI(s) = \frac{2(s + 20)}{s(s+20)} = \frac{2}{s}I(s)=s(s+20)2(s+20)​=s2​

      STEP 7: Ready for Inverse Laplace Transform

      Final result:

      I(s)=2sI(s) = \frac{2}{s}I(s)=s2​

      This is simple: i(t)=2u(t)i(t) = 2u(t)i(t)=2u(t) A

      Physical interpretation: With 20V applied across 10Ω total resistance, steady-state current is 2A. The initial condition i(0)=2A means we start at steady state!

      Complete Example 3: Parallel RC Circuit

      Problem Statement

      Find the voltage v(t) across the parallel combination for t ≥ 0.

      Given:

      • i_s(t) = 2u(t) A
      • R = 4Ω
      • C = 0.25F
      • v(0) = 0

      Time Domain Circuit 

               i_s(t) = 2u(t) A
              |
         +----v(t)----+
         |            |
        R=4Ω        C=0.25F
         |            |
       ground       ground

      STEP 1: Apply KCL at Top Node

      is(t)=iR(t)+iC(t)i_s(t) = i_R(t) + i_C(t)is​(t)=iR​(t)+iC​(t)

      STEP 2: Substitute Component Equations

      For resistor:

      iR(t)=v(t)R=v(t)4i_R(t) = \frac{v(t)}{R} = \frac{v(t)}{4}iR​(t)=Rv(t)​=4v(t)​

      For capacitor:

      iC(t)=Cdv(t)dt=0.25dv(t)dti_C(t) = C\frac{dv(t)}{dt} = 0.25\frac{dv(t)}{dt}iC​(t)=Cdtdv(t)​=0.25dtdv(t)​

      STEP 3: Derive Differential Equation

      Substitute into KCL:

      is(t)=v(t)4+0.25dv(t)dti_s(t) = \frac{v(t)}{4} + 0.25\frac{dv(t)}{dt}is​(t)=4v(t)​+0.25dtdv(t)​

      Rearrange:

      0.25dv(t)dt+v(t)4=is(t)0.25\frac{dv(t)}{dt} + \frac{v(t)}{4} = i_s(t)0.25dtdv(t)​+4v(t)​=is​(t)

      Multiply by 4:

      dv(t)dt+v(t)=4is(t)\frac{dv(t)}{dt} + v(t) = 4i_s(t)dtdv(t)​+v(t)=4is​(t)

      For i_s(t) = 2u(t):

      dv(t)dt+v(t)=8u(t)\frac{dv(t)}{dt} + v(t) = 8u(t)dtdv(t)​+v(t)=8u(t)

      This is the differential equation.

      STEP 4: Apply Laplace Transform

      sV(s)−v(0)+V(s)=8ssV(s) - v(0) + V(s) = \frac{8}{s}sV(s)−v(0)+V(s)=s8​

      STEP 5: Substitute Initial Conditions

      Given v(0) = 0:

      sV(s)+V(s)=8ssV(s) + V(s) = \frac{8}{s}sV(s)+V(s)=s8​V(s)[s+1]=8sV(s)[s + 1] = \frac{8}{s}V(s)[s+1]=s8​

      STEP 6: Solve for V(s)

      V(s)=8s(s+1)V(s) = \frac{8}{s(s+1)}V(s)=s(s+1)8​

      STEP 7: Ready for Inverse Laplace Transform

      Final result:

      V(s)=8s(s+1)V(s) = \frac{8}{s(s+1)}V(s)=s(s+1)8​

      Complete Example 4: Series RLC Circuit

      Problem Statement

      Find the current i(t) for t ≥ 0.

      Given:

      • v_s(t) = 10u(t) V
      • R = 2Ω
      • L = 1H
      • C = 0.1F
      • i(0) = 0, i’(0) = 0

      Time Domain Circuit 

      v_s(t) = 10u(t) V ----[R=2Ω]----[L=1H]----[C=0.1F]---- ground

      STEP 1: Apply KVL in Time Domain

      vs(t)=vR(t)+vL(t)+vC(t)v_s(t) = v_R(t) + v_L(t) + v_C(t)vs​(t)=vR​(t)+vL​(t)+vC​(t)

      STEP 2: Substitute Component Equations

      For resistor:

      vR(t)=Ri(t)=2i(t)v_R(t) = Ri(t) = 2i(t)vR​(t)=Ri(t)=2i(t)

      For inductor:

      vL(t)=Ldi(t)dt=di(t)dtv_L(t) = L\frac{di(t)}{dt} = \frac{di(t)}{dt}vL​(t)=Ldtdi(t)​=dtdi(t)​

      For capacitor:

      vC(t)=1C∫0ti(τ)dτ+vC(0)=10.1∫0ti(τ)dτ+vC(0)v_C(t) = \frac{1}{C}\int_0^t i(\tau)d\tau + v_C(0) = \frac{1}{0.1}\int_0^t i(\tau)d\tau + v_C(0)vC​(t)=C1​∫0t​i(τ)dτ+vC​(0)=0.11​∫0t​i(τ)dτ+vC​(0)vC(t)=10∫0ti(τ)dτ+vC(0)v_C(t) = 10\int_0^t i(\tau)d\tau + v_C(0)vC​(t)=10∫0t​i(τ)dτ+vC​(0)

      STEP 3: Derive Differential Equation

      Substitute into KVL:

      vs(t)=2i(t)+di(t)dt+10∫0ti(τ)dτ+vC(0)v_s(t) = 2i(t) + \frac{di(t)}{dt} + 10\int_0^t i(\tau)d\tau + v_C(0)vs​(t)=2i(t)+dtdi(t)​+10∫0t​i(τ)dτ+vC​(0)

      Differentiate to eliminate integral:

      dvs(t)dt=2di(t)dt+d2i(t)dt2+10i(t)\frac{dv_s(t)}{dt} = 2\frac{di(t)}{dt} + \frac{d^2i(t)}{dt^2} + 10i(t)dtdvs​(t)​=2dtdi(t)​+dt2d2i(t)​+10i(t)

      Rearrange:

      d2i(t)dt2+2di(t)dt+10i(t)=dvs(t)dt\frac{d^2i(t)}{dt^2} + 2\frac{di(t)}{dt} + 10i(t) = \frac{dv_s(t)}{dt}dt2d2i(t)​+2dtdi(t)​+10i(t)=dtdvs​(t)​

      For v_s(t) = 10u(t), the derivative is 10δ(t):

      d2i(t)dt2+2di(t)dt+10i(t)=10δ(t)\frac{d^2i(t)}{dt^2} + 2\frac{di(t)}{dt} + 10i(t) = 10\delta(t)dt2d2i(t)​+2dtdi(t)​+10i(t)=10δ(t)

      This is the second-order differential equation.

      STEP 4: Apply Laplace Transform

      s2I(s)−si(0)−i′(0)+2[sI(s)−i(0)]+10I(s)=10s^2I(s) - si(0) - i'(0) + 2[sI(s) - i(0)] + 10I(s) = 10s2I(s)−si(0)−i′(0)+2[sI(s)−i(0)]+10I(s)=10

      STEP 5: Substitute Initial Conditions

      Given i(0) = 0, i’(0) = 0:

      s2I(s)−0−0+2sI(s)−0+10I(s)=10s^2I(s) - 0 - 0 + 2sI(s) - 0 + 10I(s) = 10s2I(s)−0−0+2sI(s)−0+10I(s)=10I(s)[s2+2s+10]=10I(s)[s^2 + 2s + 10] = 10I(s)[s2+2s+10]=10

      STEP 6: Solve for I(s)

      I(s)=10s2+2s+10I(s) = \frac{10}{s^2 + 2s + 10}I(s)=s2+2s+1010​

      Complete the square:

      s2+2s+10=(s+1)2+9s^2 + 2s + 10 = (s+1)^2 + 9s2+2s+10=(s+1)2+9I(s)=10(s+1)2+9I(s) = \frac{10}{(s+1)^2 + 9}I(s)=(s+1)2+910​

      STEP 7: Ready for Inverse Laplace Transform

      Final result:

      I(s)=10(s+1)2+9=10(s+1)2+32I(s) = \frac{10}{(s+1)^2 + 9} = \frac{10}{(s+1)^2 + 3^2}I(s)=(s+1)2+910​=(s+1)2+3210​

      This has the form of a damped sinusoid: ω(s+α)2+ω2\frac{\omega}{(s+\alpha)^2 + \omega^2}(s+α)2+ω2ω​ where ω=3, α=1

      The inverse transform will be: i(t)=103e−tsin⁡(3t)u(t)i(t) = \frac{10}{3}e^{-t}\sin(3t)u(t)i(t)=310​e−tsin(3t)u(t) A

      Advantages of PATH B Demonstrated

      Advantage 1: Explicit Differential Equation

      From Example 4, we can immediately see:

      d2i(t)dt2+2di(t)dt+10i(t)=forcing\frac{d^2i(t)}{dt^2} + 2\frac{di(t)}{dt} + 10i(t) = \text{forcing}dt2d2i(t)​+2dtdi(t)​+10i(t)=forcing

      Standard form: d2idt2+2ζωndidt+ωn2i=forcing\frac{d^2i}{dt^2} + 2\zeta\omega_n\frac{di}{dt} + \omega_n^2 i = \text{forcing}dt2d2i​+2ζωn​dtdi​+ωn2​i=forcing

      We can identify:

      • ωn2=10\omega_n^2 = 10ωn2​=10 → ωn=10≈3.16\omega_n = \sqrt{10} \approx 3.16ωn​=10​≈3.16 rad/s
      • 2ζωn=22\zeta\omega_n = 22ζωn​=2 → ζ=110≈0.316\zeta = \frac{1}{\sqrt{10}} \approx 0.316ζ=10​1​≈0.316
      • System is underdamped (ζ < 1)
      • Damped frequency: ωd=ωn1−ζ2=3\omega_d = \omega_n\sqrt{1-\zeta^2} = 3ωd​=ωn​1−ζ2​=3 rad/s

      This physical insight is immediately clear in PATH B!

      Advantage 2: Understanding Order

      The order of the differential equation tells you:

      • First order (one derivative) → one energy storage element
      • Second order (two derivatives) → two energy storage elements

      Advantage 3: Initial Condition Clarity

      In the differential equation, you can see exactly where initial conditions enter:

      • i(0) affects the zero-order term
      • i’(0) affects the first-derivative term

      Disadvantages of PATH B Demonstrated

      Disadvantage 1: More Steps

      PATH B requires:

      1. KVL/KCL in time domain
      2. Component substitution
      3. Algebraic manipulation
      4. Differentiation (if needed)
      5. THEN Laplace transform

      PATH A skips straight to s-domain (steps 1-2 become one transformation step).

      Disadvantage 2: More Algebra in Time Domain

      Manipulating time-domain equations with integrals and derivatives is often messier than working with impedances.

      Disadvantage 3: Must Handle Integrals

      For circuits with capacitors, you often get integrals that must be eliminated by differentiation, adding complexity.

      Comparison: PATH A vs PATH B on Same Problem

      Problem: Series RL Circuit

      • v_s(t) = 20u(t) V
      • R = 10Ω, L = 0.5H
      • i(0) = 2A

      PATH A Steps:

      1. Transform: R=10Ω, Z_L=0.5s, V_s=20/s
      2. KVL: 20/s = I(s)[10 + 0.5s]
      3. Add IC: 20/s = I(s)[10 + 0.5s] + Li(0) = I(s)[10 + 0.5s] + 1
      4. Solve: I(s) = (20/s - 1)/(10 + 0.5s) = (2s + 40)/(s(s+20))

      Total: ~4 steps

      PATH B Steps:

      1. KVL time: v_s = Ri + Ldi/dt
      2. Substitute: 20u(t) = 10i + 0.5di/dt
      3. Standard form: di/dt + 20i = 40u(t)
      4. Laplace: sI(s) - i(0) + 20I(s) = 40/s
      5. Substitute IC: sI(s) - 2 + 20I(s) = 40/s
      6. Solve: I(s) = (40/s + 2)/(s+20) = (2s + 40)/(s(s+20))

      Total: ~6 steps

      Conclusion: PATH A is faster for this problem.

      When PATH B is Actually Better

      Scenario 1: Problem Asks for Differential Equation

      Problem statement: “Derive the differential equation governing the circuit, then solve using Laplace transform.”

      PATH B is required here - you must show the differential equation explicitly.

      Scenario 2: System Analysis

      Problem statement: “Find the natural frequency and damping ratio of the circuit.”

      PATH B makes this obvious from the differential equation standard form.

      Scenario 3: Educational Context

      When learning circuit theory, PATH B helps you understand:

      • How differential equations arise from physical laws
      • The connection between circuit topology and system order
      • Why certain responses occur (overdamped, underdamped, etc.)

      Key Takeaways

      1. Both paths give identical s-domain results
      2. PATH B shows the physics more clearly
      3. PATH A is faster for most problems
      4. Choose based on problem requirements
      5. Understanding both deepens your knowledge
