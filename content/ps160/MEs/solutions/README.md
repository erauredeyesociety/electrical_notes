## Study tips

1. Start with the **module summary** in `m##/M##_summary.md` (or the lecture outline) to refresh the equation set.
2. Re-derive the verified ME questions yourself — these match the answer key, so you know when you've made an error.
3. Then attempt the MQ practice quizzes; my values are independent computations, treat them as a second opinion.
4. The midterms exercise question types likely to appear on the final; midterm 3 is essentially a final-prep dry run.

## Notes / caveats

- Several PV-diagram items (ME19 Q3, MQ19a Q2, MQ19a Q3, MQ20a Q2, MT3 Q10) depend on figures I couldn't see; the most likely path interpretation is given with a note.
- Two-mirror geometry items (ME33 Q1, MT3 Q14) likewise need the figure.
- Q10 phase shifts in SHM: pay close attention to the **quadrant** — the $\arctan$ formula gives a value mod $\pi$; the signs of $x_0$ and $v_0$ pick which quadrant the actual phase lives in.
