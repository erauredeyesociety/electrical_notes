# MQ35

## Q1 — Wavelength from double-slit

$y = m\lambda L/d \Rightarrow \lambda = yd/(mL) = (0.0025)(2.7\times 10^{-4})/(3\cdot 2.2) = \boxed{102.3\ \text{nm}}$

## Q2 — Antireflection coating, $n_g=1.75$, $n_c=1.53$

Both reflections have $\pi$ flips ⇒ net 0 flips ⇒ destructive needs $2nt = (m+\tfrac12)\lambda$.
$$t_{\min} = \dfrac{\lambda}{4n_c} = \dfrac{543}{4(1.53)} = \boxed{88.7\ \text{nm}}$$

# MQ36

## Q1 — Single-slit wavelength from $m=5$ minimum

$a\sin\theta = m\lambda$, small-angle: $y = m\lambda L/a$.
$$\lambda = ya/(mL) = (0.0073)(2.6\times 10^{-4})/(5\cdot 1.8) = 2.109\times 10^{-7}\ \text{m} = \boxed{210.9\ \text{nm}}$$

## Q2 — Grating to resolve sodium-like doublet

$\Delta\lambda = 0.78$ nm, $\bar\lambda = 588.91$ nm.
$R = \bar\lambda/\Delta\lambda = 754.9$. First order: $N = R = 754.9$.
$$\sigma = 754.9/1.667 = \boxed{452.9\ \text{lines/cm}}$$

## Q3 — Bragg interatomic spacing

$d = m\lambda/(2\sin\theta) = 0.22/(2\sin 22.7°) = 0.22/0.7718 = \boxed{0.285\ \text{nm}}$

## Q4 — Resolution of the eye

$\theta = 1.22\lambda/D = 1.22(551\times 10^{-9})/(5\times 10^{-3}) = 1.344\times 10^{-4}\ \text{rad}$
$\theta\cdot 180/\pi = 7.70\times 10^{-3°} = \boxed{7.70\ \text{millidegrees}}$
