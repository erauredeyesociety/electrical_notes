# MQ33a

## Q1 — Two perpendicular mirrors (corner reflector)

Ray hits vertical mirror at 58° from its normal ⇒ after reflection it makes 58° on the other side of that normal. The horizontal mirror's normal is *perpendicular* to the vertical mirror's normal, so the angle the same ray makes with the horizontal-mirror normal is the complement:
$$\phi = 90° - 58° = \boxed{32°}$$

## Q2 — Wavelength in medium ($n=2.8$)

$\lambda_{\text{med}} = \lambda_{\text{vac}}/n = 536/2.8 = \boxed{191.43\ \text{nm}}$

## Q3 — Wavelength from speed in medium

$n = c/v = 3\times 10^8/2.01\times 10^8 = 1.4925$.
$\lambda_{\text{med}} = 502/1.4925 = \boxed{336.3\ \text{nm}}$

## Q4 — Refraction across a boundary

$\sin\theta_2 = (n_1/n_2)\sin\theta_1 = (1.4/2.3)\sin(39°) = 0.3831$ ⇒ $\theta_2 = \boxed{22.53°}$

## Q5 — Lateral shift through a slab

In glass: $\sin\theta' = \sin(33°)/7 = 0.0778 \Rightarrow \theta' = 4.46°$.
$$d = h\dfrac{\sin(\theta-\theta')}{\cos\theta'} = 8\cdot\dfrac{\sin(28.54°)}{\cos(4.46°)} = 8\cdot 0.4791 = \boxed{3.83\ \text{cm}}$$

(Note: $n=7$ is unphysical — using as given.)

# MQ33b

## Q1 — BK7 dispersion through 0.3-m slab at 49°

$\sin(49°) = 0.7547$.
$\theta_2 = \arcsin(0.7547/1.523) = 29.71°$, $\tan = 0.5703$.
$\theta_4 = \arcsin(0.7547/1.515) = 29.88°$, $\tan = 0.5749$.
$\Delta x = 0.3(0.5749 - 0.5703) = 1.38\times 10^{-3}\ \text{m} = \boxed{1.38\ \text{mm}}$

## Q2 — Critical angle

$\sin\theta_c = n_2/n_1 = 1.4/2 = 0.7 \Rightarrow \theta_c = \boxed{44.43°}$

## Q3 — Refraction angle from a known critical angle

Liquid index: $n = 1/\sin(44.1°) = 1/0.6959 = 1.4370$.
Air→liquid at 28°: $\sin\theta_r = \sin(28°)/1.4370 = 0.3268 \Rightarrow \theta_r = \boxed{19.07°}$

## Q4 — Two polarizers (16°, then +47°)

$I_2/I_0 = \cos^2(16°)\cos^2(47°) = 0.9239(0.4651) = \boxed{0.4297}$

## Q5 — Brewster's angle, find $n$

$\tan\theta_B = n_2/n_1 = n_{\text{fluid}}/1$ ⇒ $n = \tan(47.2°) = \boxed{1.081}$
