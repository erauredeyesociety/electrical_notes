# CEC 315 — Clear Explanations

A series of deep-dive, intuition-first explanations of concepts that routinely trip students up in Signals and Systems. Each topic gets both a Markdown version (for reading) and a LaTeX version (for PDF / Overleaf).

**Philosophy:** these are *not* cheatsheets or formula dumps. They're written to answer "why" and "what does this actually mean" questions that the textbook and lectures gloss over. If you can already recite the rule but don't know why it's the rule, these are for you.
