\section*{The Big Idea}

Any periodic signal can be decomposed into a sum of harmonically related complex exponentials. Complex exponentials are \textbf{eigenfunctions} of LTI systems -- they pass through unchanged in frequency, only scaled in amplitude and shifted in phase.

\section*{Periodic Signals}

A signal is periodic if $x(t) = x(t + T)$ for all $t$, where $T$ is the fundamental period and $\omega_0 = 2\pi/T$ is the fundamental frequency.

\section*{The Eigenfunction Property}

For an LTI system with impulse response $h(t)$:

\[e^{st} \longrightarrow H(s) \cdot e^{st}\]

where the \textbf{system function} is:

\[H(s) = \int_{-\infty}^{\infty} h(\tau) e^{-s\tau} d\tau\]

The output is the same exponential scaled by $H(s)$. The frequency does \textbf{not} change.

For frequency analysis, set $s = j\omega$:

\[H(j\omega) = \int_{-\infty}^{\infty} h(\tau) e^{-j\omega\tau} d\tau\]

\section*{The CTFS Pair}

\subsection*{Synthesis Equation (frequency -> time)}

\[x(t) = \sum_{k=-\infty}^{\infty} a_k \, e^{jk\omega_0 t}\]

\subsection*{Analysis Equation (time -> frequency)}

\[a_k = \frac{1}{T} \int_T x(t) \, e^{-jk\omega_0 t} \, dt\]

\subsection*{DC Component}

\[a_0 = \frac{1}{T} \int_T x(t) \, dt = \text{average value of } x(t)\]

\section*{How to Compute FS Coefficients}

\begin{enumerate}
  \item Identify $T$ and $\omega_0 = 2\pi/T$
  \item Compute $a_0$ (average value)
  \item Set up the analysis integral for general $a_k$
  \item Integrate only over intervals where $x(t) \neq 0$
  \item Evaluate and simplify
\end{enumerate}

\section*{Conjugate Symmetry (Real Signals)}

If $x(t)$ is real-valued:
\begin{itemize}
  \item $a_{-k} = a_k^*$ (conjugate symmetry)
  \item $|a_{-k}| = |a_k|$ (even magnitude spectrum)
  \item $\angle a_{-k} = -\angle a_k$ (odd phase spectrum)
\end{itemize}
