•

•

•

ATCRBS – Air Traffic Control Radar Beacon System
Developed in 1956, ATCRBS was the first air route surveillance radar system 
developed and purchased for the purposes of air traffic control. The 
technology was based closely on that of the military's IFF (Identification 
Friend or Foe) system.
Interrogation from ground– 1030 MHz
Reply from airplane – 1090 MHz

When Reply to Interrogation
ATCRBS Transponder replies to an interrogation so long as:
1.
Transmission of P1 to P2 is 2𝜇𝑠 apart
2.
P1 > P2 signal
3.
Delay of 8 𝜇𝑠 if Mode A or 21𝜇𝑠 if Mode C from P1 rising edge to rising 
edge of P3
Once a valid interrogation is received, the turnaround time for reply is 3𝜇𝑠.

Transponder Sensitivity and Power Output
Transponder Minimum Triggering Level is Set to -73 dBm +/- 4 dB at the 
Transponder Input/Output Connector
Power Output for Class 1A or 2A ATCRBS Transponders is 125 Watts
Power Output for Class 1B and 2B ATCRBS Transponders is 70 Watts
No Class of Transponder may Exceed 500 Watts

Encoding Altimeter
•
An encoding altimeter is used to transmit the barometric altitude of an 
aircraft for a Mode C and Mode S transponders.
•
Mode C – altitudes are encoded in 100 ft increments.
•
Mode S – altitudes are encoded in 25 ft increments
•
Gilham Codes – a type of “gray code” is used to encode the altitude into 
the appropriate 11-bit sequence


--- Page 16 ---
The Gillham Code
A form of Gray code
Range: -1200’ to 126,700’
100’ increments
Uses 11 of the available 12 bits
D1 is not used
It is used to transmit uncorrected barometric altitude between an 
encoding altimeter or analog air data computer and a digital 
transponder.

Mode S Technology attempts to mitigate ATCRBS
•
All Call  to request all aircraft to reply with a specified probability that they 
should respond, e.g. ½, ¼, 1/8, 1/16 chance to reply
•
After several attempts, the majority of traffic should be detected at the 
ground station.
•
Stochastic acquisition
•
Selective Interrogations (aka “roll call” – once local aircraft are known, 
interrogate those aircraft.  
•
Lock out aircraft that have responded with updates to avoid unnecessary 
replies to all-call.

All-Call Signal Timing
•
Signal transmitted with P1, P2, and P3 (like ATCRBS)
•
Additional P4 signal alternates between 0.8 𝜇𝑠 to 1.6 𝜇𝑠 pulse width  (all other pulses in the 
interrogation are 0.8 𝜇𝑠
•
0.8 𝜇𝑠 ignored by Mode S, but will trigger Mode A and C transponders
•
1.6 𝜇𝑠 triggers Mode S transponders to respond
•
Enables backward compatibility 
•
Reply includes ICAO address and interrogators identifier.
•
Aircraft replies that are received without garble are considered “locked” or “acquired”.
(Image source: 5 Primary and Secondary Radar | Aircraft Systems )

Locking Out
•
After All-Call, Mode S radar will only interrogate the known aircraft with the 
SELECT(Roll Call) interrogation.
•
Lockout message sent with the first SELECT Message.
•
Aircraft transponders lock out replies to the station’s ALL Call interrogations 
until SELECT Message indicating lock out has ended.


--- Page 30 ---
Stochastic Acquisition
•
Mitigate garbling by not having each aircraft respond to every all call.
•
With less aircraft responding, there is less likelihood of garble
•
Set by the ground-station’s interrogation, the aircraft are told to respond 
with one of the following probabilities:  100%, 50%, 25%, 6.25%, 0%.
•
After several all-calls the neighboring aircraft should be acquired.

--- Page 34 ---
Reducing Errors
•
Parity Check:
•
Cyclic Redundant Check (CRC) used to provide a sequency of parity bits, 
which are appended to the transmitted message. 
•
Polynomial, iterative XOR process
•
Receiver can perform same CRC then compare the results.
•
In response, the system can re-interrogate the aircraft that produced an 
error message.
•
ATC can be notified of aircraft with recurring errors (might be faulty or 
dealing with local interference impacting ground or aircraft).


--- Page 35 ---
Air-to-Air Mode S
An advantage of Mode S is that it can support not only ground-to-air SSR.
It also supports Air-to-Air!
•
Aircraft can send ALL CALL and suitably equipped aircraft will respond 
(Mode A, Mode C, and other Mode S).
•
With aircraft acquired, it can then use the SELECT message to request 
updates from the detected aircraft.
•
Having antenna on top and bottom of aircraft enable trans and receipt to 
aircraft above it.
