• Webster-Miriam: 
• “A computational method that is a 
subfield of artificial intelligence and 
that enables a computer to learn to 
perform tasks by analyzing a large 
dataset without being explicitly 
programmed.
• Using historically collected data or 
past states of a system, an ML 
model is trained to solve a problem.

• Supervised Learning.
• Unsupervised Learning
• Reinforcement Learning

Supervised Learning
• Training using labeled data (inputs and 
expected outputs) to 
• Make a prediction
• Determine the measure of error or loss
• Make corrections based upon the errors
• Classification – determines the category of a 
given input (e.g. classify a radar target by 
aircraft type)
• Regression – fits the model to the data to 
produce a continuous value (e.g. 
predicting house prices)
CEC 300
College of Engineering, Daytona Beach, FL

--- Page 8 ---
Unsupervised Learning
• Unsupervised Learning – learning is performed on unlabeled data in which the underlying structure is not 
previously known.
• Clustering is an example of unsupervised learning where data is grouped based on its internal 
structure/features
• Anomaly Detection analyzes some stream of input data to identify outliners in the data set
• Dimension reduction used to narrow/clean data sets for tasks such as visualization or to improve the 
performance of other algorithms.

• Reinforcement Learning – learns to respond to its state and stimuli through the 
reinforcement of which actions provided favorable results vs. which options provided

Reinforcement Learning
• Typically explained using agent paradigm
• Agent must learn a policy, which dictates actions given state
• Learning encourages maximization of rewards and/or 
minimization of penalties
• Both supervised and unsupervised RL techniques exist
• Reinforcement Learning Algorithms:
• Monte Carlo
• Q-Learning
• Markov Decision Modeling (MDM)

--- Page 18 ---
Instance-based Algorithms
• Learn examples by heart
• Generalizes new data based upon some similarity metric
• Example
• k-Nearest Neighbors utilizes the data and labels from past examples to identify the k neighbors of 
“smallest distance away” from a new input and use their labels to facilitate classification and/or 
prediction.
CS 455/CS 595a: Introduction to Artificial 
Intelligence

--- Page 19 ---
Model-based Learning
• From a set of examples, a model is constructed to make predictions based 
upon new input cases
• Model Elements:
• Model selection
• Model parameters
• Utility or fitness function (positive evaluation)
• Or, cost function (negative evaluation)
• Training is performed by feeding it data and adjusting its parameters to 
improve the model’s fit to the data.

Pilot Program: ATCT AI @ Heathrow
• AI System "Amy": Tested at Heathrow Airport to assist air 
traffic controllers.
• Purpose: Manage UK's most congested airspace efficiently.
• System Features
•
Data Integration: Combines radar and video data for holistic 
aircraft visualization.
•
Digital Control Tower: Replaces traditional tower view with 
digital processing.
•
Real-Time Data: Provides flight number, aircraft type, and 
status (inbound/outbound).
• Testing: Over 40,000 flights tested.
• Future Plans: Fully operational digital contingency tower 
by 2027.
• Limitations: Lack of contextual judgment and data 
limitations.
• Human Collaboration: AI works alongside human 
controllers for better results.
