Fuzzy Theory in Decision Analysis
Fuzziness, Chaos, and Uncertainty
 Think of the most of real-world things handled
  daily.
  ◼ They are fuzzy and highly uncertain in nature.
  ◼ We need our knowledge and experience to handle them.
  ◼ The methods and/or solutions to handle them are usually
    highly dynamic (or sometimes chaotic) in nature.
Fuzzy
 Definition:
   ◼ Something blurred or indistinct, in shape or outline
     (Oxford English Dictionary).
   ◼ Lacking in clarity or definition (Webster).



 Means of specifying how well an object or property
  satisfies a vague description.
   ◼   “Lack of crispness” considered to be the state of a real-
       world object or property.
Fuzzy
 Example:


     “John is tall”

     Is this true if John is 5’10”?
     Most people would hesitate to answer “true” or “false”
     preferring to say, “sort of”
     This is not a question of uncertainty about the external

     The term “tall” does not refer to a sharp demarcation of

     We consider that there are degrees of tallness.
Fuzzy Set Theory
 L. A. Zadeh, Fuzzy Sets, Information and Control,
  1965.
  ◼   Describes the degree of belonging of an individual
      member, which can exist as any real number between 0
      and 1, instead of just either 0 or 1.
 Fuzzy Set vs. Crisp Set.
  ◼ “Lack of crispness” is in most cases the truth of a real-
    world property.
  ◼ Cannot determine exactly (100%) a certain characteristic.
  ◼ All the measured qualities appearing in our world of
    perception are fuzzy in the sense that it is not the quality
    itself that is fuzzy, but rather that we cannot always
    measure these physical qualities with 100% certainty.
Fuzzy Set Theory
 Extension of classic set theory.
   ◼   Classic set theory: elements of fuzzy set theory either are
       or are not members of the set.
 Example:
   ◼ Set of even natural positive numbers less than 10 is a
     precise collection of four elements {2, 4, 6, 8}.
   ◼ The set could be described as a membership function mA(x)
     defined over some “universe of discourse,” where the
     function takes on the value 1 when the argument x is an
     element of set A, and 0 otherwise.
   ◼ A fuzzy set offers the possibility for mA(x) to take on values
     other than {0, 1}, say in the range (0, α), where the value
     assumed indicates the degree of membership of x in A.
   ◼ For consistency, the possible range of values is often
     scaled to the real numbers between 0 and 1, inclusive.
Fuzzy Set Theory
 Example:
  ◼ A membership mA(x) = 1 indicates that the element x is
    definitely a member of A.
  ◼ A membership mA(x) = 0 indicates that the element x is not
    a member of A.
  ◼ Intermediate values indicate intermediate degrees of
    membership.
  ◼ This intermediate range represents the extension of fuzzy
    sets beyond the classic “crisp” sets.
Fuzzy Set Theory
 Back to the example:
  ◼ Treat Tall as a fuzzy predicate.
  ◼ The truth value of Tall(John) is a number between 0 and 1,
    rather than being just true or false.
 The name “fuzzy set” derives from the
  interpretation of the predicate as implicitly
  defining a set of its members.
Fuzzy Set Operations
 A fuzzy set is said to be empty if and only if its
  membership function is identically 0 on the entire
  universe of discourse X (all possible arguments to
  the function).
 Two fuzzy sets A and B are equal, A = B, if and only if
  mA(x) = mB(x), for all x  X. This can be written mA = mB
 The complement of a fuzzy set A is denoted A’ and
  defined as mA’ = 1 – mA
 The fuzzy set A is contained in or is a subset of B if
  and only if mA ≤ mB
Fuzzy Set Operations
 The union of two fuzzy sets A and B is a fuzzy set
  C = A  B with a membership function
  mC(x) = max{mA(x), mB(x)}, x  X
 The intersection of two fuzzy sets A and B is a
  fuzzy set C = A  B with a membership function
  mC(x) = min{mA(x), mB(x)}, x  X
Fuzzy Set Operations
 Example: flower that is yellow and fragrant.
  ◼ mYellow(x) = 0.8
  ◼ mFragrant(x) = 0.9
  ◼ Flower membership in the fuzzy set of yellow and
    fragrant flowers is min{0.8, 0.9} = 0.8
  ◼ Flower membership in the fuzzy set of yellow or fragrant
    flowers set is max{0.8, 0.9} = 0.9
  ◼ Flower membership in the fuzzy set of flowers that are
    not yellow is 1 – 0.8 = 0.2

Fuzzy Inference Process
 Fuzzy inference refers to the fuzzy reasoning
  operations based on the knowledge in the fuzzy
  system.
   ◼ Given two groups of fuzzy sets A, A' X and B, B' Y
  ◼   Direct Reasoning:
        For any two variables x and y:

        Implication: If x is A, then y is B

        Premise: x is A’

        Conclusion: y is B’

        This can be denoted as: B’ = A’        R (where R is the relation)
  ◼   Indirect Reasoning:
        For any two variables x and y:

        Implication: If x is A, then y is B

        Premise: y is B’

        Conclusion: x is A’

        This can be denoted as: A’ = R         B’ (where R is the relation)
Fuzzy Inference Process
 Fuzzy compositional rule for fuzzy inference.
  ◼   When there is a complex knowledge base containing a
      list of fuzzy rules (Rk) with the corresponding input
      signals A’k for each fuzzy set Ak.

       X = (A’1, A’2, …, A’k, …, A’n)
       Y = B’j, j = [1 … n]

  ◼   Using direct reasoning, the fuzzy variable B’ can be
      inferred as the composition operation between fuzzy
      set Ak and the fuzzy relation R, as follows:

       B’ = (A’1, A’2, …, A’k, ..., A’n)  R
Fuzzy Inference Process
 Fuzzy compositional rule for fuzzy inference.
  ◼   Two composition schemes are commonly used.
       Sup-min operation.
       Sup-product operation.

  ◼   Composite fuzzy reasoning consists of the following steps:
       1. Identify each fuzzy rule and determine the
       “connectives” for each condition in the rule base, in
       which an “OR” clause represents a union operation and
       an “AND” clause represents an intersection operation.
       2. For each fuzzy rule (Rk), locate the αk.
       3. Using either the sup-min operation or sup-product
       operation method, infer the overall defuzzification
       control action.

Using Fuzzy Logic
 When the inputs or model parameters are
  uncertain but not random.
 Fuzzy numbers have a range; the larger the range,
  the more uncertainty.
 Fuzzy numbers have a degree of belief
  (membership function) between 0 and 1 associated
  with each point in the range.
  ◼ The degree of belief is not a probability of occurrence.
  ◼ The degree of belief curve does not integrate to 1, and
    integrating it does not produce a CDF.
Using Fuzzy Logic
 Triangular fuzzy number <l, m, h>


 Degree of belief is 0 from -∞ to 1 and from h to ∞.
 Degree of belief at m is 1, from which the degree of
  belief varies linearly to 0 at both l and h.
 Number “5” can be represented as: <3, 5, 9>, <4, 5, 6>,
  <5, 5, 5.5>, or <5, 5, 5> depending on the potential
  values the number might have other than exactly 5.
Using Fuzzy Logic
 Triangular fuzzy number <l, m, h>


       a  <l, m, h> + b = <al + b, am + b, ah + b>
       <l1, m1, h1> + <l2, m2, h2> = <l1 + l2, m1 + m2, h1 + h2>
       <l1, m1, h1>  <l2, m2, h2> = <l1  l2, m1  m2, h1  h2>

 When a deterministic number is needed to implement the
  result of an analysis, the fuzzy output must be defuzzified.
 An appropriate deterministic representation is considered the
  centroid of the fuzzy range.
 Defuzzified value for <l, m, h> is (l + m + h)/3
